# 서류 심사 진행 및 결과 처리

> **해당 단계:** STEP 9 (서류심사 진행) → STEP 10 (심사점수 입력) → STEP 11 (결과 처리)
> **구분:** 신규임용

---

## 📍 개요

서류심사(1차)는 심사위원 구성부터 점수 입력, 최종 결과 확정까지 세 단계로 진행됩니다.

- **산출물:** 서류심사 평가표(서명본) / 합격자·예비·불합격자 명단
- **관련 시스템 메뉴:**
  - STEP 10: `일반행정 > 채용관리 > 직원 > 전형별 심사 입력` (메뉴 1.20)
  - STEP 11: `일반행정 > 채용관리 > 직원 > 전형별 심사 결과` (메뉴 1.21)

---

## STEP 9 — 서류심사 진행

### 진행 순서

1. 심사위원(내부/외부) 확정 및 서약서 수령
2. 지원서·자기소개서·증빙자료 배부 (또는 열람 안내)
3. 심사위원별 개별 평가 (평가표 작성 및 날인)
4. 점수 취합 및 순위 산정
5. 결과 확정 (내부결재 또는 합의체 확정 방식)

### 사전 준비 체크리스트

- [ ] 심사위원 명단 확정 (3명 이상 권장)
- [ ] 서약서·회피신고서 등 필수 서류 확보
- [ ] 평가 기준(항목·배점·가중치) 사전 공유
- [ ] 점수 취합 시 오류·누락·서명 여부 검증
- [ ] 합격·예비·불합격 기준 및 배수(예: 3배수) 확정

### 유의사항

- 개인정보(주민등록번호 등)가 포함된 자료는 **최소 열람·보관 원칙** 준수
- **평가표 원본(서명본)은 분쟁·감사 대비를 위해 반드시 보관**

---

## STEP 10 — 심사점수 입력

### 화면 진입 경로

`일반행정 > 채용관리 > 직원 > 전형별 심사 입력`

### 조회 전 필수 선택 항목

| 항목 | 필수 여부 |
|------|-----------|
| 모집연도 | ✅ 필수 |
| 모집명 | ✅ 필수 |
| 모집부서 | ✅ 필수 |
| 모집분야 | ✅ 필수 |
| 심사구분 | ✅ 필수 |
| 심사위원 | 🔍 돋보기로 검색·선택 (**심사위원이 없을 경우 먼저 등록**) |

### 방법 1 — 화면에서 직접 입력 (소수 지원자)

1. 조회 조건 선택 후 **[조회]** 클릭
2. 왼쪽 목록에서 지원자 클릭
3. 오른쪽 심사점수 입력 영역에 항목별 점수 입력
4. **[저장]** 클릭

### 방법 2 — 엑셀 업로드 (다수 지원자, 권장)

1. **[엑셀 다운로드]** 클릭 → 서식 파일 저장
2. 엑셀 파일에 심사위원·심사항목·점수·등급 입력
3. **[엑셀 업로드]** 클릭 → 파일 선택
4. 오류내역 없음 확인 후 **[저장]** 클릭

> **⚠️ 반드시 시스템의 [엑셀 다운로드]로 받은 서식 파일을 사용해야 합니다. 자체 제작 파일은 업로드 불가.**

> 📌 등급은 필수값이 아니며, 점수와 별개로 필요한 경우에만 입력합니다.

---

## STEP 11 — 심사 결과 처리

### 화면 진입 경로

`일반행정 > 채용관리 > 직원 > 전형별 심사 결과`

### 처리 순서

| 순서 | 버튼/동작 | 설명 |
|------|-----------|------|
| 1 | 조회 조건 선택 후 **[조회]** | 지원자 목록 + 심사위원별 점수 확인 |
| 2 | 사정일자 설정 후 **[심사사정]** 클릭 | 총점 기준 자동 순위 산정 |
| 3 | 가산점 입력 (해당자) | 총점·평균점수 자동 반영 |
| 4 | 결과 선택 | `합격` / `불합격` / `예비순위` |
| 5 | **[저장]** 클릭 | 임시 저장 (아직 확정 아님) |
| 6 | **[확정]** 클릭 | ⚠️ 확정일자 등록, 이후 합격여부 수정 불가 |

### 결과값 기준

| 결과값 | 의미 | 다음 단계 처리 |
|--------|------|----------------|
| `합격` | 다음 심사단계 진행 대상 | 다음 전형에서 조회 가능 |
| `불합격` | 탈락 처리 | 다음 전형에서 미표시 |
| `예비순위` | 추후 합격 전환 가능 | 담당자 별도 관리 |

### ⚠️ 핵심 주의사항

- **[확정] 클릭 후에는 합격여부 수정이 불가합니다.** 가산점·비고 항목만 수정 가능합니다.
- **오확정 시 처리 방법:** `[확정취소]` 클릭 → 확정일자 공란 변경 → 결과 수정 → 재확정
- **심사사정은 점수 기준 등수만 자동 산정**하며, 합격·불합격·예비순위는 **담당자가 직접 선택 후 확정**해야 합니다.
- **합격 처리된 지원자만** 다음 심사구분에서 조회 가능합니다.

## 📋 전체 흐름 요약

```
[STEP 9] 심사위원 구성 및 서면 평가 실시
        ↓
[STEP 10] 시스템에 심사점수 입력 (직접 입력 또는 엑셀 업로드)
        ↓
[STEP 11] 심사사정 → 결과 선택(합격/불합격/예비) → 저장 → 확정
```
