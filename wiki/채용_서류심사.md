# 지원자 접수 및 서류심사

> **구분:** 신규임용 | **관련 STEP:** STEP 8 → STEP 9 → STEP 10 → STEP 11
> **목적:** 지원자 접수 관리부터 서류심사 진행, 점수 입력, 결과 확정까지 서류전형 전 과정 수행

---

## 전체 흐름 요약

```
지원자 접수 관리 (STEP 8)
    → 서류심사 진행 (STEP 9)
        → 심사점수 입력 (STEP 10)
            → 심사 결과 처리 (STEP 11)
```

---

## STEP 8. 지원자 접수 관리

### 화면 진입 경로

`일반행정 > 채용관리 > 직원 > 채용지원자 관리`

---

### 1. 지원자 현황 조회 및 관리 (메뉴 1.13)

조회 조건: 모집연도·모집명·모집부서·분야·지원자번호/성명

**주요 기능 버튼**

| 버튼 | 기능 |
|------|------|
| **[조회]** | 조건별 지원자 목록 검색 |
| **[추가]** | 관리자가 지원자 직접 등록 (1.14 팝업) |
| **[삭제]** | 선택 지원자 삭제 (심사점수 입력 전만 가능) |
| **[지원서 출력]** | 지원자 정보 리포트 출력 |
| **[자기소개서 출력]** | 자기소개서 리포트 출력 |
| **[공고별 제출서류 일괄다운로드]** | 전체 지원자 첨부파일 압축 다운로드 |

**탭별 관리 항목**

| 탭 | 관리 항목 |
|----|-----------|
| 기본 | 성명·생년월일·성별·연락처·이메일·국적·학가구분 등 |
| 학력사항 | 학력구분·학교명·학과명·학위·입학일·졸업일·취득국가 |
| 경력사항 | 기관명·직위명·재직기간·근무년수·담당직무·임상여부 |
| 자격,면허,상,벌 | 자격증·면허·수상·징계 이력 |
| 자기소개서 | 항목별 자소서 내용 (설정된 항목만 표시) |
| 제출서류 | 첨부파일 업로드·다운로드·일괄다운로드 |

> ⚠️ **심사점수가 이미 입력된 지원자는 삭제 불가**

---

### 2. 지원자 직접 등록 (메뉴 1.14 Popup)

오프라인 접수 지원자 등록 또는 시스템 오류 발생 시 관리자가 직접 등록합니다.

| 입력항목 | 내용 |
|----------|------|
| 모집명 | 드롭다운 선택 |
| 모집부서 / 모집분야 | 드롭다운 선택 |
| 한글 성명 / 영문 성명 / 한자 성명 | 직접 입력 |
| 생년월일 | 드롭다운 선택 |
| 성별 | 남 / 여 선택 |

> 📌 지원번호는 저장 시 시스템이 **자동 생성**됩니다.

---

### 3. 지원불가자 예외 처리 (메뉴 1.19)

- 지원 희망자가 지원이 안 된다고 연락이 오면 **총무인사팀으로 문의**하여 예외자 처리를 진행합니다.

---

### FAQ

**Q. 입사지원서 제출 후 지원자가 수정을 희망할 때는 어떻게 하나요?**

진행상태 정보를 `임시저장`으로 변경 후 저장하면 재작성 가능한 상태가 됩니다. **변경 후 반드시 대상자에게 안내**하시기 바랍니다.

> ⚠️ **진행상태 변경(임시저장→제출)은 부득이한 경우에만 사용**합니다.

---

## STEP 9. 서류심사 진행

### 개요

- **목적:** 서류심사(1차) 수행 및 결과 확정
- **산출물:** 서류심사 평가표(서명본) / 합격자·예비·불합격자 명단

---

### 진행 순서

1. 심사위원(내부/외부) 확정 및 서약서 수령
2. 지원서·자기소개서·증빙자료 배부(또는 열람 안내)
3. 개별 평가 진행 (평가표 작성 및 날인)
4. 점수 취합 및 순위 산정
5. 결과 확정 (내부결재 또는 합의체 확정 방식)

---

### 체크리스트

- [ ] 심사위원 명단 확정 (3명 이상 권장)
- [ ] 서약서·회피신고서 등 필수 서류 확보
- [ ] 평가 기준(항목·배점·가중치) 사전 공유
- [ ] 점수 취합 시 오류·누락·서명 여부 검증
- [ ] 합격·예비·불합격 기준 및 배수(예: 3배수) 확정

---

### 유의사항

- **개인정보(주민등록번호 등) 포함 자료 취급 시 최소 열람·보관 원칙 준수**
- **평가표 원본(서명본)은 차후 분쟁·감사 대비를 위해 반드시 보관**

---

## STEP 10. 전형별 심사점수 입력

### 화면 진입 경로

`일반행정 > 채용관리 > 직원 > 전형별 심사 입력`

---

### 조회 전 필수 선택 항목

| 항목 | 필수 여부 |
|------|-----------|
| 모집연도 | ✅ 필수 |
| 모집명 | ✅ 필수 |
| 모집부서 | ✅ 필수 |
| 모집분야 | ✅ 필수 |
| 심사구분 | ✅ 필수 |
| 심사위원 | 🔍 돋보기로 검색·선택 (미등록 시 위원 먼저 등록) |

> ⚠️ **심사위원이 등록되어 있지 않으면 먼저 위원 등록 후 진행**

---

### 방법 1 — 화면 직접 입력 (소수 지원자)

1. 조회 조건 선택 후 **[조회]** 클릭
2. 왼쪽 목록에서 지원자 클릭
3. 오른쪽 심사점수 입력 영역에 항목별 점수 입력
4. **[저장]** 클릭

---

### 방법 2 — 엑셀 업로드 (다수 지원자, 권장)

1. **[엑셀 다운로드]** 클릭 → 서식 파일 저장
2. 엑셀 파일에 심사위원·심사항목·점수·등급 입력
3. **[엑셀 업로드]** 클릭 → 파일 선택
4. 오류내역 없음 확인 후 **[저장]** 클릭

> ⚠️ **반드시 시스템에서 [엑셀 다운로드]로 받은 서식 파일만 사용** — 자체 제작 파일 업로드 불가
>
> 📌 등급은 필수값이 아니며 점수와 별개로 필요 시에만 사용합니다.

---

## STEP 11. 심사 결과 처리

### 화면 진입 경로

`일반행정 > 채용관리 > 직원 > 전형별 심사 결과`

---

### 처리 순서

| 순서 | 버튼/동작 | 설명 |
|------|-----------|------|
| 1 | 조회 조건 선택 후 **[조회]** | 지원자 목록 + 심사위원별 점수 확인 |
| 2 | 사정일자 설정 후 **[심사사정]** 클릭 | 총점 기준 자동 순위 산정 |
| 3 | 가산점 입력 (해당자) | 총점·평균점수 자동 반영 |
| 4 | 결과 선택 | `합격` / `불합격` / `예비순위` |
| 5 | **[저장]** 클릭 | 임시 저장 (아직 확정 아님) |
| 6 | **[확정]** 클릭 | ⚠️ 확정일자 등록, 이후 합격여부 수정 불가 |

---

### 합격/불합격/예비 처리 기준

| 결과값 | 의미 | 다음 단계 처리 |
|--------|------|----------------|
| `합격` | 다음 심사단계 진행 대상 | 다음 전형에서 조회 가능 |
| `불합격` | 탈락 처리 | 다음 전형에서 미표시 |
| `예비순위` | 추후 합격 전환 가능 | 담당자 관리 |

---

### ⚠️ 핵심 주의사항

- **확정 후 합격여부 수정 불가** — 가산점·비고만 수정 가능
- 오확정 시: **[확정취소]** 클릭 → 확정일자 공란으로 변경 → 결과 수정 → 재확정
- **합격 결과 지원자만** 다음 심사구분에서 조회 가능
- 심사사정은 점수 기준 등수만 자동 산정 → **합격/불합격/예비순위는 담당자가 직접 선택 후 확정**
