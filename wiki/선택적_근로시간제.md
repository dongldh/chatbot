# 선택적 근로시간제 사용 안내

> 업무의 효율적 수행과 일·생활 균형을 위해 선택적 근로시간제를 운영하고 있습니다. 팀원·팀장·부서장 역할별 절차를 숙지하여 연장근로가 적절히 보상받을 수 있도록 활용하시기 바랍니다.

---

## 1. 선택적 근로시간제 개요

### 제도 정의

선택적 근로시간제는 유연근무제의 일종으로, **1개월 정산 기간 내 1주 평균 52시간(기본 40시간 + 연장근로 12시간)을 초과하지 않는 범위**에서 근로자가 근무시간을 자유롭게 조정하는 제도입니다.

- 의무근로시간대: **11:00 ~ 16:00**
- 일반 근무시간: 9:00 ~ 18:00 (일부 부서 8:00 ~ 17:00)

### 제도 이점

- 월요일에 10시간 근무 시, 화요일에 6시간 근무가 가능한 유연한 체계
- 한 달 근로시간 합산 후 기준근로시간(근무일수 × 8시간)을 초과한 경우, **다음 달에 보상휴가 부여**
- **부여된 보상휴가는 해당 월 내 사용하는 것이 원칙**

### 대체휴무 vs. 보상휴가 차이

| 구분 | 설명 | 사용 배율 | 사용 시기 |
|------|------|-----------|-----------|
| **대체휴무** | 연장근무시간 내에서 해당 월에 사용하는 휴식. 해당 월 소정근로시간을 초과하여 근무한(예상되는) 경우에만 사용 가능 | 연장시간 × 1.0 | 연장 발생 당월 |
| **보상휴가** | 해당 월에 휴식하지 못한 연장근무시간을 다음 달 이후 휴가로 사용 | 연장시간 × 1.5 | 다음 달 이후 |

## 2. 팀원 업무 절차

### 1단계. 연장근로 사전 보고 및 UWIN 입력

**연장근로는 반드시 사전 보고 및 승인을 받아야 인정됩니다.**

1. 팀장에게 연장근로 필요 사유를 보고하고 승인을 받습니다.
2. 팀장 승인 후 UWIN에 연장근로 정보를 입력합니다.
   - 메뉴 경로: `원스탑 근로관리 > 근로관리 > 내근로현황` → 해당 일자 클릭 → 팝업 입력
   1. 연장근로 예정 시간을 입력합니다. (10분 단위)
   2. 휴게시간 등을 반영하여 자동 산정된 근로시간을 확인합니다.
   3. 연장근로 사유를 **구체적으로** 입력합니다.
   4. [저장]을 누르고 저장된 내용을 확인합니다.
3. 팀장은 일과 종료 전 해당일 연장근로를 **부서장 전결로 전자결재**하여 총무인사팀에 송부합니다.

### 2단계. 퇴근 기록 남기기

1. 연장근로 종료 후 **반드시 KT 단말기에 퇴근 기록을 남깁니다.**
2. **퇴근 기록이 있는 경우에만 연장근로로 인정됩니다.**
   - **실수로 기록하지 않은 경우에도 인정되지 않으므로 반드시 기록하시기 바랍니다.**

### 3단계. 대체휴무 또는 보상휴가 신청

메뉴 경로 (공통): `원스탑 근로관리 > 근로관리 > 내근로현황` → 해당 일자 클릭 → 팝업 입력

**① 대체휴무 신청**

1. [대체휴무]를 선택하고 사용할 시간을 입력합니다. (10분 단위)
2. 휴무 시간을 확인합니다.
3. 사유란에 `대체휴가 사용`이라고 입력합니다.
4. [저장]을 누르고 저장된 내용을 확인합니다.

**② 보상휴가 신청**

1. [보상휴가]를 선택하고 사용할 시간을 입력합니다. (10분 단위)
2. 보상휴가 시간을 확인합니다.
3. 사유란에 `보상휴가 사용`이라고 입력합니다.
4. [저장]을 누르고 저장된 내용을 확인합니다.

## 3. 팀장 업무 절차

### 연장근로 당일 — 사전 승인 및 전자결재

1. 팀원으로부터 연장근로 보고를 받고 승인 여부를 결정합니다.
2. 팀원이 UWIN에 연장근로 정보를 입력하였는지 확인합니다.
3. **일과 종료 전**, 당일 예정된 연장근로를 **부서장 전결로 전자결재**하여 총무인사팀에 송부합니다.

### 연장근로 익일 — 출퇴근 기록 확인

1. 팀원이 KT 단말기에 남긴 퇴근 기록을 확인합니다.
2. **퇴근 기록이 전날 UWIN에 입력한 시간과 다를 경우** 총무인사팀에 조정을 요청합니다.
   - 예시: 22:00 종료 예정 입력 → 실제 퇴근 기록 23:00인 경우 조정 요청

### 연장근로에 따른 휴게시간 보장

- 연장근로 발생 시 팀원의 휴게시간이 적절히 보장되는지 확인합니다.

### 월별 근로시간 확정 및 보상휴가 부여 신청

1. 해당 월 근로시간이 최종 확정되면 UWIN에서 보상휴가 부여 전자결재를 진행합니다.
   - **소속 단위에 [초과시간]이 존재하는 경우에만 전자결재 버튼이 활성화됩니다.**
2. 결재라인은 **부서장(처장)까지** 지정하고, 총무인사팀을 수신 부서로 지정합니다.
3. **⚠️ 전자결재 버튼을 누르는 순간 해당 연월의 근로 데이터가 확정(잠금)됩니다. 확정 이후에는 수정이 불가하므로, 반드시 최종 근로시간을 검토한 뒤 전자결재를 올리시기 바랍니다.**

---

## 4. 연장근로·대체휴무 사후결재 절차 (팀장·부서장)

사후결재가 필요한 경우 아래 절차에 따라 처리합니다.

1. 그룹웨어에 첨부된 엑셀 파일을 열어 내용을 확인합니다.
2. UWIN에서 결재를 진행합니다.
   - 메뉴 경로: `일반행정 > 직원인사 > 근로관리 > 부서근태관리(일간)`
3. 조회 조건(부서 정보 등)을 입력·선택한 후 [조회] 버튼을 누릅니다.
4. 조회된 정보를 확인하고 전자결재를 완료합니다.
