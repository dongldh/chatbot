# 심사위원 및 점수 관리

## 개요

채용 프로세스에서 심사위원을 등록하고 전형별 심사점수를 입력하는 일련의 과정을 정리한 페이지입니다. 이 두 업무는 순차적으로 진행되며, **심사위원 등록 없이는 점수 입력이 불가능**합니다.

---

## 심사위원 등록

### 진입 경로

`1.3 모집부서 관리 → 심사단계 [선택] → 심사과정 관리 팝업(1.7) → 심사단계별 [관리] 버튼 클릭`

또는

`1.12 심사사정 관리`

### 등록 전 필수 확인사항

**심사위원 관리 버튼 활성화 조건:**
- **[관리] 버튼은 심사단계가 저장된 후에만 표시됩니다**
- 심사단계 저장 시 다음 조건을 충족해야 합니다:
  - 심사항목의 총 배점이 **100점**이어야 함
  - 심사항목이 등록되어 있어야 함

### 내부 vs 외부 심사위원 등록

| 구분 | 등록 방법 | 입력 항목 |
| --- | --- | --- |
| **내부** | 🔍 돋보기 → 교직원 검색 팝업 → 선택 | 자동 입력 |
| **외부** | [추가] → 직접 입력 | 성명 / 부서(소속) / 직급 |

### 등록 절차

1. 심사단계 목록에서 심사위원을 등록할 단계 선택
2. **[관리]** 버튼 클릭 (심사위원 관리 팝업 열림)
3. **[추가]** 클릭하여 신규 행 생성
4. **구분 선택**: `내부` 또는 `외부`
5. 심사위원 정보 입력
   - **내부**: 🔍 아이콘 클릭 → 교직원 검색 팝업에서 선택 → 성명·부서·직급 자동 입력
   - **외부**: 성명, 부서(소속), 직급을 직접 입력
6. **[저장]** 클릭

### ⚠️ 심사위원 관리 주의사항

- **이미 심사평가가 진행 중인 심사위원은 삭제할 수 없습니다**
- 심사단계는 다음 경우 삭제 불가:
  - 심사위원이 등록된 경우
  - 심사항목이 등록된 경우

## 심사항목별 점수 입력

### 진입 경로

`일반행정 > 채용관리 > 직원 > 1.20 전형별 심사 입력`

### 입력 전 필수 확인사항

**조회에 필수적인 선택 항목:**

| 항목 | 필수 여부 | 비고 |
| --- | --- | --- |
| 모집연도 | ✅ 필수 | — |
| 모집명 | ✅ 필수 | — |
| 모집부서 | ✅ 필수 | — |
| 모집분야 | ✅ 필수 | — |
| 심사구분 | ✅ 필수 | 필기, 면접 등 |
| 심사위원 | ✅ 필수 | 🔍 돋보기로 검색·선택 |

**사전 준비:**
- **심사위원이 없으면 먼저 등록해야 합니다**
- 심사단계와 심사항목이 완료되어 있어야 합니다

### 점수 입력 방법

#### 방법 1: 화면 직접 입력 (소수 지원자)

1. 조회 조건 (모집연도, 모집명, 모집부서, 모집분야, 심사구분, 심사위원) 선택
2. **[조회]** 클릭
3. 왼쪽 지원자 목록에서 심사 대상자 클릭
4. 오른쪽 심사점수 입력 영역에 항목별 점수 입력
5. **[저장]** 클릭

#### 방법 2: 엑셀 업로드 (다수 지원자, 권장)

1. **[엑셀 다운로드]** 클릭 → 시스템 서식 파일 저장
2. 엑셀 파일에 다음 항목 입력:
   - 심사위원
   - 심사항목
   - 점수
   - 등급 (선택사항)
3. **[엑셀 업로드]** 클릭 → 저장한 파일 선택
4. 오류내역 없음 확인 후 **[저장]** 클릭

### ⚠️ 점수 입력 주의사항

- **반드시 시스템의 [엑셀 다운로드]로 받은 서식 파일을 사용하세요**
  - 자체 제작 파일은 업로드할 수 없습니다
- **등급은 선택사항**이며, 점수와 별개로 필요한 경우만 입력합니다
- 엑셀 업로드 시 오류 메시지를 확인하고 수정 후 재업로드합니다

### 전형별 점수 입력

면접, 필기, 기타 전형 등 **여러 전형에 대해 동일한 방식으로 입력**합니다:

- 심사구분을 해당 전형으로 변경하여 동일한 절차 반복
- 각 전형마다 [조회] → 점수 입력 → [저장] 과정 수행

## 통합 관리 (심사사정 관리)

### 진입 경로

`1.12 심사사정 관리`

### 주요 기능

| 기능 | 설명 |
| --- | --- |
| [추가] | 심사단계 또는 심사항목 추가 |
| [저장] | 추가·수정한 내용 저장 |
| [삭제] | 선택한 단계 또는 항목 삭제 |
| [심사위원] | 단계별 심사위원 관리 팝업 열기 |
| [심사항목설정] | 해당 심사단계의 항목 조회 |

**용도:**
- 모집연도/모집명/모집부서/모집분야별로 심사단계와 심사항목을 통합 확인
- 1.7 팝업과 동일한 기능 제공
- 한눈에 전체 심사 구조 파악 가능

## 주요 제약사항 요약

| 상황 | 제약 | 대응 방안 |
| --- | --- | --- |
| 심사위원 등록 버튼 미표시 | [관리] 버튼이 보이지 않음 | 먼저 심사단계를 저장해야 함 |
| 심사단계 삭제 불가 | 이미 심사위원 또는 심사항목이 등록됨 | 관련 데이터 삭제 후 시도 |
| 심사항목 삭제 불가 | 지원자 평가가 이미 진행됨 | 해당 평가 데이터 삭제 필요 |
| 점수 입력 불가 | 심사위원 미등록 | 먼저 심사위원 등록 필수 |
| 엑셀 업로드 실패 | 시스템 서식이 아닌 다른 파일 사용 | [엑셀 다운로드]로 받은 정본 파일 사용 |
| 심사항목 저장 불가 | 총 배점이 100점이 아님 | 배점을 조정하여 합계 100점 구성 |

## 업무 흐름도

```
1. 모집 및 심사단계 생성
   ↓
2. 심사항목 등록 (총 배점 100점)
   ↓
3. 심사단계 저장
   ↓
4. ★ 심사위원 등록 ★
   ↓
5. ★ 전형별 심사점수 입력 ★ (필기, 면접 등)
   ↓
6. 평가 완료 및 결과 집계
```
