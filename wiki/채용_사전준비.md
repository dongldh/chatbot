# 채용 사전 준비 (TO 확인 ~ 공고 등록)

> **적용 구분:** 신규임용
> **관련 단계:** STEP 1 → STEP 2 → STEP 3 → STEP 4

---

## 개요

채용 공고를 게시하기 전까지의 사전 준비 절차입니다. TO 확인부터 내부결재, 모집부서 설정, 채용공고 등록까지 순서대로 진행하며, **각 단계의 내용이 이후 단계에 직접 영향을 미치므로 순서를 반드시 준수**해야 합니다.

| STEP | 단계명 | 게시 전 완료 필수 |
|------|--------|:-----------------:|
| 1 | TO 확인 | — |
| 2 | 내부결재 | ✅ |
| 3 | 모집부서 설정 | ✅ |
| 4 | 채용공고 등록 | — |

---

## STEP 1. TO 확인

### 목적

채용 필요 인원(TO) 및 충원 사유를 확정하고 채용계획 초안을 수립합니다.

> **이 단계에서 확정된 TO·직무·일정은 이후 공고 게시 이후 변경이 어려우므로, 기본 골격을 이 단계에서 명확히 확정해야 합니다.**

### 진행 절차

1. 결원 또는 증원 사유를 정리합니다. (퇴직, 신규사업, 정원조정 등)
2. 해당 부서의 TO 승인 여부를 확인합니다.
3. 직무 및 담당업무 범위를 구체적으로 정리합니다.
4. 채용 형태를 검토합니다. (정규직 / 무기계약직 / 계약직 등)
5. 예산 및 인건비 재원을 확인합니다. (필요 시)
6. 채용 전체 일정의 큰 틀을 수립합니다. (접수 → 심사 → 임용)

### 체크리스트

- [ ] 결원/증원 사유 정리 완료
- [ ] 직무/담당업무 범위 정리 완료
- [ ] 채용 형태 및 기간 검토 완료
- [ ] 예산/인건비 재원 확인 완료
- [ ] 채용 일정 초안 수립 완료

---

## STEP 2. 내부결재

### 목적

채용 진행에 대한 내부 승인(결재)을 확보합니다.

> **결재 문서 확정 전에 공고 내용을 미리 작성하지 마십시오. 결재 후 내용이 변경될 경우 공고 수정 리스크가 발생합니다.**

### 결재 전 준비사항

- [ ] 채용 사유/배경 및 필요성 정리
- [ ] 직무/담당업무 및 자격요건(필수/우대) 정리
- [ ] 전형 단계/방법(서류, 면접 등) 및 합격자 선발 기준 초안 작성
- [ ] 채용 일정 초안 작성 (공고 → 접수 → 심사 → 발표 → 임용)
- [ ] 예산/보수/계약기간 검토 (해당 시)

### 내부결재 작성 예시

```
제 목: 총무처 총무인사팀 발간실 계약직원 채용(안)

발간실 운영을 위하여 다음과 같이 계약직원을 채용하고자 합니다.

1. 퇴직 예정 인원
   가. 소속 : 총무처 총무인사팀(발간실)
   나. 성명 : OOO
   다. 퇴직예정일 : 2026.02.28.(수)

2. 신규채용 기본사항
   직종: 단기계약직 / 근무부서: 총무인사팀(발간실)
   채용인원: 1명 / 채용방법: 공개채용 / 임용예정일: YYYY.MM.DD

3. 담당 업무 : 발간실 업무 보조 등
4. 채용 절차 : 서류접수 → 서류심사 → 면접심사 → 최종합격
5. 지원 자격 및 급여 기준 : 본교 2년차 조교 수준
6. 급여 재원 : 교비

붙임  발간실 계약직원 채용(안) 1부. 끝.
```

> 내부결재 첨부자료에는 채용 사유, 해당 직무의 필요성 등을 간략히 기술한 자료를 함께 첨부합니다.

### 결재 후 확인사항

- [ ] 결재 완료(승인) 여부 확인
- [ ] **확정된 결재 내용(정원, 직무, 일정, 자격요건)을 기준으로 이후 공고 등록 진행**

---

## STEP 3. 모집부서 설정

### 화면 진입 경로

`일반행정 > 채용관리 > 직원 > 모집부서 관리`

> 🚨 **이 단계는 공고 게시 전 반드시 완료해야 합니다. 공고 게시 후에는 모집부서·자기소개서·첨부파일 항목의 추가·수정·삭제가 불가합니다.**

---

### 3-1. 모집부서 등록

**관련 메뉴:** `1.3 모집부서 관리` → `1.11 모집부서 관리(Popup)`

1. 모집년도·모집명을 선택한 후 **[조회]** 클릭
2. **[추가]** 버튼 클릭 또는 기존 모집분야명 클릭 → 모집부서 관리 팝업(1.11) 열림
3. 아래 항목 입력 후 **[저장]** 클릭

| 항목 | 입력 방법 | 예시 |
|------|-----------|------|
| 모집부서 | 🔍 돋보기로 부서 검색·선택 | `정보화개발팀` |
| 모집분야 | 텍스트 직접 입력 | `전산직` |
| 모집인원 | 숫자 입력 | `2` |
| 지원자격 | 텍스트 직접 입력 | `학사학위 이상 소지자` |
| 우대사항 | 텍스트 직접 입력 | `정보처리기사 자격증 소지자` |

> 📌 이 팝업에 입력한 내용은 **공고문에 자동 반영되지 않으며**, 출력물 및 내부 관리용으로만 사용됩니다.

**⚠️ 주의:** 게시 중인 공고의 모집부서는 삭제 불가. 지원자가 있는 경우에도 삭제 불가.

---

### 3-2. 심사과정(전형 단계) 설정

**관련 메뉴:** `1.7 심사과정 관리(Popup)`

1. 심사단계 **[추가]** 클릭 → 단계 행 생성
2. 심사명·심사비율·최종점 입력 (예: `서류심사 / 50% / 100점`)
3. 심사단계 **[저장]** 클릭 → [심사위원] 버튼 활성화
4. 심사항목설정 **[선택]** 클릭 → 오른쪽 심사항목 영역에 표시
5. **[기본 설정 가져오기]** 클릭 또는 직접 항목 추가 (공통코드 E93 기반)
6. 심사항목별 배점 입력 — **⚠️ 배점 합계는 반드시 100점**
7. 심사항목 **[저장]** 클릭

**심사단계 구성 예시**

| 단계 | 심사명 | 비율(%) |
|------|--------|:-------:|
| 1단계 | 서류심사 | 50 |
| 2단계 | 전공심사 | 20 |
| 3단계 | 면접심사 | 30 |

> **[심사위원] 버튼은 심사단계 저장 후에만 표시됩니다. 배점 합계가 100점이 아니면 저장이 불가합니다.**

---

### 3-3. 자기소개서 항목 설정

**관련 메뉴:** `1.9 자기소개서 항목 관리(Popup)`

1. **[기본설정 가져오기]** 클릭 → 공통코드(E95) 표준 항목 자동 세팅
2. 항목 추가 필요 시 **[추가]** 클릭 → 신규 항목명 입력
3. 항목별 글자수 제한 설정 (기본값: 1,000자)
4. **[저장]** 클릭

**기본 항목 목록:** 지원동기 / 성장과정 / 생활신조 / 연구역량 및 연구계획 / 강의계획

> **⚠️ 이미 게시된 공고에서는 항목 추가·수정·삭제가 불가합니다.**

---

### 3-4. 필수 첨부파일 항목 설정

**관련 메뉴:** `1.10 첨부파일항목 관리(Popup)`

1. **[기본설정 가져오기]** 클릭 → 공통코드(E88) 표준 서류 자동 세팅
2. 항목별 필수여부 체크박스 설정 (✅ 필수 / □ 선택)
3. 불필요한 항목은 체크박스 선택 후 **[삭제]**
4. **[저장]** 클릭

**기본 제출서류 목록**

| # | 제출서류 | 필수 여부 |
|---|----------|:---------:|
| 1 | 연구실적목록 | ✅ |
| 2 | 연구실적 증빙자료 | ✅ |
| 3 | 학위별 학력증명서 및 성적증명서 | ✅ |
| 4 | 석·박사 학위 논문 | □ |
| 5 | 경력 및 재직증명서 | ✅ |
| 6 | 의사면허증/전문의자격증/간호사면허증 | □ |
| 7 | 교육관련자료 | ✅ |

> **⚠️ 이미 게시된 공고에서는 항목 추가·수정·삭제가 불가합니다.**

---

## STEP 4. 채용공고 등록

### 화면 진입 경로

`일반행정 > 채용관리 > 직원 > 채용공고 관리`

> 🚨 **접수기간 내에는 게시된 공고 내용을 수정할 수 없습니다. 잘못 게시한 경우 진행상태를 N으로 변경한 후 신규 공고를 등록하십시오.**

---

### 4-1. 공고문 템플릿 관리 (사전 준비)

**화면 위치:** `일반행정 > 채용관리 > 직원 > 채용공고문 템플릿 관리`

공고 등록 전, 반복 사용할 공고문 양식을 미리 템플릿으로 저장해 둘 수 있습니다.

1. **[추가]** 클릭 → 템플릿명 입력 칸 및 에디터 초기화
2. 템플릿명 입력 (예: `2025년 직원채용 기본양식`)
3. 에디터에 공고문 내용 작성 (지원자격, 제출서류, 전형일정 등)
4. **[저장]** 클릭

**수정/삭제:** 목록에서 템플릿명 클릭 → 수정 후 [저장] / 체크박스 선택 후 [삭제]

> **⚠️ 템플릿을 나중에 수정해도 이미 등록된 채용공고문에는 반영되지 않습니다.**

---

### 4-2. 채용공고 등록

**관련 메뉴:** `1.2 채용공고 관리` → `1.6 채용공고 관리(Popup)`

1. **[추가]** 클릭 → 채용공고 관리 팝업(1.6) 열림
2. 아래 항목 입력
3. **[저장]** 클릭 → 공고 목록에 추가
4. 채용공고문 **[선택]** 버튼 클릭 → 템플릿 조회 팝업(1.4) 열림
5. 템플릿 선택 후 **[공고문 템플릿 적용]** → 공고문 관리 팝업(1.5)으로 이동
6. 공고문 내용 최종 편집 후 **[저장]** 클릭

**입력항목 상세**

| 항목 | 필수 | 입력 방법 |
