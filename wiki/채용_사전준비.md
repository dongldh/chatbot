# 채용 사전준비 (TO 확인 · 내부결재 · 모집부서 설정)

> 채용 프로세스를 시작하기 전, 아래 3단계 사전준비를 순서대로 완료해야 합니다.
> **각 단계는 이후 단계의 선행 조건이므로 순서를 반드시 준수하세요.**

---

## 전체 흐름 요약

| 단계 | 작업 | 게시 전 필수 여부 |
|------|------|:-----------------:|
| STEP 1 | 채용 TO 확인 | — |
| STEP 2 | 채용 내부결재 | ✅ 필수 |
| STEP 3 | 모집부서 설정 | ✅ 필수 |

---

## STEP 1. 채용 TO 확인

### 개요

소속부서의 채용 필요 인원(TO)과 충원 사유를 확정하는 단계입니다.
**이후 단계(공고 게시 등)에서는 변경이 어려우므로, TO·직무·일정의 기본 골격을 이 단계에서 반드시 확정합니다.**

### 절차

1. 결원·증원 사유를 정리합니다. (퇴직, 신규사업, 정원조정 등)
2. 해당 직무 및 담당업무 범위를 구체적으로 정리합니다.
3. 채용 형태(정규직 / 무기계약직 / 계약직 등) 및 계약 기간을 검토합니다.
4. 예산·인건비 재원을 확인합니다. (필요 시)
5. 채용 일정(접수 → 심사 → 임용)의 큰 틀을 수립합니다.

### 확인 체크리스트

- [ ] 결원/증원 사유 정리 완료
- [ ] 직무/담당업무 범위 정리 완료
- [ ] 채용 형태 및 기간 검토 완료
- [ ] 예산/인건비 재원 확인 완료
- [ ] 채용 일정(접수~임용) 수립 완료

---

## STEP 2. 채용 내부결재

### 개요

채용 진행에 대한 내부 승인(결재)을 확보하는 단계입니다.
**내부결재가 완료된 후에 공고 등록을 진행하며, 결재 문서 확정 전에 공고 내용을 미리 작성하지 않습니다.** (내용 변경 리스크 방지)

### 결재 전 준비 절차

1. 채용 사유·배경 및 필요성을 정리합니다.
2. 직무·담당업무 및 자격요건(필수/우대)을 정리합니다.
3. 전형 단계·방법(서류, 면접 등) 및 합격자 선발 기준 초안을 작성합니다.
4. 채용 일정(공고 → 접수 → 심사 → 발표 → 임용) 초안을 작성합니다.
5. 예산·보수·계약기간(해당 시)을 검토합니다.

### 내부결재 문서 작성 예시

> 아래는 참고용 예시입니다. 실제 작성 시 부서·직종·일정에 맞게 수정하세요.

```
제 목: 총무처 총무인사팀 발간실 계약직원 채용(안)

발간실 운영을 위하여 다음과 같이 계약직원을 채용하고자 합니다.

1. 퇴직 예정 인원
   가. 소속: 총무처 총무인사팀(발간실)
   나. 성명: OOO
   다. 퇴직예정일: 2026.02.28.(수)

2. 신규채용 기본사항
   - 직종: 단기계약직
   - 근무부서: 총무인사팀(발간실)
   - 채용인원: 1명
   - 채용방법: 공개채용
   - 임용예정일: YYYY.MM.DD

3. 담당 업무: 발간실 업무 보조 등
4. 채용 절차: 서류접수 → 서류심사 → 면접심사 → 최종합격
5. 지원 자격 및 급여 기준: 본교 2년차 조교 수준
6. 급여 재원: 교비

붙임  발간실 계약직원 채용(안) 1부. 끝.
```

> 📌 **핵심:** 채용 전 임용 계획 전반(급여 수준, 근무 시작일, 담당 업무 등)을 내부결재로 확정합니다.
> 내부결재 첨부자료에는 채용 사유 및 해당 직무의 필요성을 간략히 기입한 자료를 첨부합니다.

### 결재 후 확인 체크리스트

- [ ] 결재 완료(승인) 여부 확인
- [ ] 확정된 결재 내용(정원, 직무, 일정, 자격요건 등) 기준으로 공고 등록 준비 완료

## STEP 3. 모집부서 설정

### 개요

시스템에 모집부서·전형 단계·자기소개서 항목·첨부파일 항목을 등록하는 단계입니다.

> 🚨 **이 단계는 공고 게시 전 반드시 완료해야 합니다.
> 공고 게시 후에는 모집부서·자기소개서·첨부파일 항목의 추가·수정·삭제가 불가합니다.**

### 화면 진입 경로

`일반행정 > 채용관리 > 직원 > 모집부서 관리`

---

### 3-1. 모집부서 등록

관련 메뉴: `1.3 모집부서 관리` → `1.11 팝업`

1. 모집년도·모집명을 선택한 후 **[조회]** 버튼을 클릭합니다.
2. **[추가]** 버튼을 클릭하거나 기존 모집분야명을 클릭하여 모집부서 관리 팝업(1.11)을 엽니다.
3. 팝업에서 아래 항목을 입력한 후 **[저장]** 버튼을 클릭합니다.

| 입력 항목 | 설명 | 예시 |
|-----------|------|------|
| 모집부서 | 🔍 돋보기로 부서 검색·선택 | `정보화개발팀` |
| 모집분야 | 채용 직무명 | `전산직` |
| 모집인원 | 숫자 입력 | `2` |
| 지원자격 | 텍스트 직접 입력 | `학사학위 이상 소지자` |
| 우대사항 | 텍스트 직접 입력 | `정보처리기사 자격증 소지자` |

> 📌 팝업에 입력한 내용은 **공고문에 자동 반영되지 않으며**, 출력물·내부 관리용으로 사용됩니다.

> ⚠️ 게시 중인 공고의 모집부서는 삭제 불가 / 지원자가 있는 경우에도 삭제 불가

---

### 3-2. 심사과정(전형 단계) 설정

관련 메뉴: `1.7 심사과정 관리 (Popup)`

1. 심사단계 **[추가]** 버튼을 클릭하여 단계 행을 생성합니다.
2. 심사명·심사비율·최종점을 입력합니다. (예: `서류심사 / 50% / 100점`)
3. 심사단계 **[저장]** 버튼을 클릭합니다. (저장 후 [심사위원] 버튼이 활성화됩니다.)
4. 심사항목설정 영역에서 **[선택]** 버튼을 클릭하여 심사항목을 불러옵니다.
5. **[기본 설정 가져오기]** 버튼으로 공통코드(E93) 기본 항목을 불러오거나, 항목을 직접 추가합니다.
6. 심사항목별 배점을 입력합니다. (**합계가 반드시 100점이어야 합니다.**)
7. 심사항목 **[저장]** 버튼을 클릭하여 완료합니다.

**심사단계 구성 예시**

| 단계 | 심사명 | 비율(%) |
|------|--------|:-------:|
| 1단계 | 서류심사 | 50 |
| 2단계 | 전공심사 | 20 |
| 3단계 | 면접심사 | 30 |

> ⚠️ [심사위원] 버튼은 심사단계 저장 후에만 표시됩니다.
> **배점 합계가 100점이 아니면 저장이 불가합니다.**

---

### 3-3. 자기소개서 항목 설정

관련 메뉴: `1.9 자기소개서 항목 관리 (Popup)`

1. **[기본설정 가져오기]** 버튼을 클릭하여 공통코드(E95) 표준 항목을 자동으로 세팅합니다.
2. 추가 항목이 필요한 경우 **[추가]** 버튼을 클릭하여 신규 항목명을 입력합니다.
3. 항목별 글자수 제한을 설정합니다. (기본값: 1,000자)
4. **[저장]** 버튼을 클릭하여 완료합니다.

**기본 항목 목록**

- 지원동기
- 성장과정
- 생활신조
- 연구역량 및 연구계획
- 강의계획

> ⚠️ **이미 게시된 공고에서는 항목 추가·수정·삭제가 불가합니다.**

---

### 3-4. 필수 첨부파일 항목 설정

관련 메뉴: `1.10 첨부파일항목 관리 (Popup)`

1. **[기본설정 가져오기]** 버튼을 클릭하여 공통코드(E88) 표준 서류를 자동으로 세팅합니다.
2. 항목별 필수 여부 체크박스를 설정합니다. (✅ 필수 / □ 선택)
3. 불필요한 항목은 체크박스 선택 후 **[삭제]** 버튼으로 제거합니다.
4. **[저장]** 버튼을 클릭하여 완료합니다.

**기본 제출서류 목록**

| # | 제출서류 | 필수 여부 |
|---|----------|:---------:|
| 1 | 연구실적목록 | ✅ |
| 2 | 연구실적 증빙자료 | ✅ |
| 3 | 학위별 학력증명서 및 성적증명서 | ✅ |
| 4 | 석·박사 학위 논문 | □ |
| 5 | 경력 및 재직증명서 | ✅ |
| 6 | 의사면허증/전문의자격증/간호사면허증 | □ |
| 7 | 교육관련자료 | ✅ |

> ⚠️ **이미 게시된 공고에서는 항목 추가·수정·삭제가 불가합니다.**
