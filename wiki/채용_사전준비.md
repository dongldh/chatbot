# 채용 사전 준비 (TO 확인 · 내부결재 · 모집부서 설정)

> 이 페이지는 채용 프로세스의 초기 단계인 **STEP 1 ~ STEP 3**을 다룹니다.
> 공고 게시 전 완료해야 할 필수 절차를 단계별로 정리합니다.

---

## 전체 흐름 요약

```
[STEP 1] TO 확인 및 채용계획 초안 수립
        ↓
[STEP 2] 내부결재 (공고 게시 전 필수 완료)
        ↓
[STEP 3] 모집부서 설정 (공고 게시 전 필수 완료)
        ↓
[STEP 4] 공고 등록 및 게시
```

---

## STEP 1. 채용 TO 확인

### 개요

채용 필요 인원(TO) 및 충원 사유를 확정하는 단계입니다.
**이후 단계(공고 게시 등)에서 변경이 어려우므로, TO·직무·일정의 기본 골격을 이 단계에서 확정해야 합니다.**

### 진행 절차

1. 소속 부서의 TO 승인 여부 확인
2. 결원 또는 증원 사유 정리 (퇴직, 신규사업, 정원조정 등)
3. 직무 및 담당업무 범위 정리
4. 채용 형태 및 기간 검토 (정규직 / 무기계약직 / 계약직 등)
5. 예산 및 인건비 재원 확인 (필요 시)
6. 채용 일정 큰 틀 수립 (접수 → 심사 → 임용)

### 체크리스트

- [ ] 결원/증원 사유 정리 (퇴직, 신규사업, 정원조정 등)
- [ ] 직무/담당업무 범위 정리
- [ ] 채용 형태(정규직/무기계약직/계약직 등) 및 기간 검토
- [ ] 예산/인건비 재원 확인
- [ ] 채용 일정(접수~임용) 큰 틀 수립

---

## STEP 2. 채용 내부결재

### 개요

채용 진행에 대한 내부 승인(결재)을 확보하는 단계입니다.
**내부결재 완료 후 공고 등록을 진행하며, 결재 문서 확정 전 공고 내용 작성을 금지합니다 (변경 리스크).**

### 결재 전 준비사항

1. 채용 사유/배경 및 필요성 정리
2. 직무/담당업무 및 자격요건 (필수/우대) 정리
3. 전형 단계/방법 초안 작성 (서류, 면접 등 및 합격자 선발 기준)
4. 채용 일정 초안 작성 (공고 → 접수 → 심사 → 발표 → 임용)
5. 예산/보수/계약기간 검토 (해당 시)

### 내부결재 문서 작성 예시

> 아래는 참고용 예시이며, 실제 작성 시 해당 부서 및 직무에 맞게 수정합니다.

---

**제목:** 총무처 총무인사팀 발간실 계약직원 채용(안)

발간실 운영을 위하여 다음과 같이 계약직원을 채용하고자 합니다.

**1. 퇴직 예정 인원**
- 소속: 총무처 총무인사팀(발간실)
- 성명: OOO
- 퇴직예정일: 2026.02.28.(수)

**2. 신규채용 기본사항**

| 직종 | 근무부서 | 채용인원 | 채용방법 | 임용예정일 |
|------|----------|----------|----------|------------|
| 단기계약직 | 총무인사팀(발간실) | 1명 | 공개채용 | YYYY.MM.DD |

**3. 담당 업무:** 발간실 업무 보조 등

**4. 채용 절차:** 서류접수 → 서류심사 → 면접심사 → 최종합격

**5. 지원 자격 및 급여 기준:** 본교 2년차 조교 수준

**6. 급여 재원:** 교비

붙임: 발간실 계약직원 채용(안) 1부. 끝.

---

> 📌 내부결재 첨부자료에는 채용 사유, 해당 직무의 필요성 등을 간략히 기재한 자료를 첨부합니다.

### 결재 후 확인사항

- [ ] 결재 완료(승인) 여부 확인
- [ ] 결재 확정 내용(정원·직무·일정·자격요건 등)을 기준으로 공고 등록 진행

## STEP 3. 모집부서 설정

### 개요

시스템 내에서 공고에 연결할 모집부서, 심사과정, 자기소개서 항목, 첨부파일 항목을 설정하는 단계입니다.

> 🚨 **공고 게시 전 반드시 완료해야 합니다. 게시 후에는 모집부서·자기소개서·첨부파일 항목의 추가·수정·삭제가 불가능합니다.**

### 화면 진입 경로

`일반행정 > 채용관리 > 직원 > 모집부서 관리`

---

### 3-1. 모집부서 등록

관련 메뉴: `1.3 모집부서 관리` → `1.11 Popup`

1. 모집년도·모집명 선택 후 **[조회]** 클릭 → 해당 공고의 모집부서 목록 조회
2. **[추가]** 버튼 클릭 또는 모집분야명 클릭 → 모집부서 관리 팝업(1.11) 열림
3. 팝업에서 아래 항목 입력 후 **[저장]** 클릭

**모집부서 입력항목**

| 항목 | 설명 | 예시 |
|------|------|------|
| 모집부서 | 🔍 돋보기로 부서 검색·선택 | `정보화개발팀` |
| 모집분야 | 채용 직무명 | `전산직` |
| 모집인원 | 숫자 입력 | `2` |
| 지원자격 | 텍스트 직접 입력 | `학사학위 이상 소지자` |
| 우대사항 | 텍스트 직접 입력 | `정보처리기사 자격증 소지자` |

> 📌 팝업 내 입력 내용은 **공고문에 자동 반영되지 않으며**, 출력물·내부 관리 용도로 사용됩니다.

**⚠️ 주의:** 게시 중인 공고의 모집부서는 삭제 불가 / 지원자가 있는 경우에도 삭제 불가

---

### 3-2. 심사과정(전형 단계) 설정

관련 메뉴: `1.7 심사과정 관리 (Popup)`

1. 심사단계 **[추가]** 클릭 → 단계 행 생성
2. 심사명·심사비율·최종점 입력 (예: `서류심사 / 50% / 100점`)
3. 심사단계 **[저장]** → 저장 후 [심사위원] 버튼 활성화됨
4. 심사항목설정 **[선택]** → 오른쪽 심사항목 영역에 표시
5. **[기본 설정 가져오기]** 클릭 또는 직접 추가 (공통코드 E93 기본 항목 불러오기)
6. 심사항목별 배점 입력 (**⚠️ 합계 반드시 100점**)
7. 심사항목 **[저장]** → 완료

**심사단계 구성 예시**

| 단계 | 심사명 | 비율(%) |
|------|--------|---------|
| 1단계 | 서류심사 | 50 |
| 2단계 | 전공심사 | 20 |
| 3단계 | 면접심사 | 30 |

**⚠️ 주의:** [심사위원] 버튼은 심사단계 저장 후에만 표시됨 / **배점 합계가 100점이 아니면 저장 불가**

---

### 3-3. 자기소개서 항목 설정

관련 메뉴: `1.9 자기소개서 항목 관리 (Popup)`

1. **[기본설정 가져오기]** 클릭 → 공통코드(E95) 표준 항목 자동 세팅
2. 항목 추가가 필요한 경우 **[추가]** 클릭 후 신규 항목명 입력
3. 글자수 제한 설정 (기본값: 1,000자)
4. **[저장]** 클릭 → 완료

**기본 항목 목록**

- 지원동기 / 성장과정 / 생활신조 / 연구역량 및 연구계획 / 강의계획

**⚠️ 주의:** 이미 게시된 공고에서는 항목 추가·수정·삭제 불가

---

### 3-4. 필수 첨부파일 항목 설정

관련 메뉴: `1.10 첨부파일항목 관리 (Popup)`

1. **[기본설정 가져오기]** 클릭 → 공통코드(E88) 표준 서류 자동 세팅
2. 필수여부 체크박스 설정 (✅ = 필수, □ = 선택)
3. 불필요한 항목 선택 후 **[삭제]**
4. **[저장]** 클릭 → 완료

**기본 제출서류 목록**

| # | 제출서류 | 필수 여부 |
|---|----------|-----------|
| 1 | 연구실적목록 | ✅ 필수 |
| 2 | 연구실적 증빙자료 | ✅ 필수 |
| 3 | 학위별 학력증명서 및 성적증명서 | ✅ 필수 |
| 4 | 석·박사 학위 논문 | □ 선택 |
| 5 | 경력 및 재직증명서 | ✅ 필수 |
| 6 | 의사면허증/전문의자격증/간호사면허증 | □ 선택 |
| 7 | 교육관련자료 | ✅ 필수 |

**⚠️ 주의:** 이미 게시된 공고에서는 항목 추가·수정·삭제 불가

---

## 단계별 완료 기준 요약

| STEP | 단계명 | 게시 전 필수 완료 | 완료 기준 |
|------|--------|:-----------------:|-----------|
| STEP 1 | TO 확인 | - | 채용 인원·직무·일정 초안 확정 |
| STEP 2 | 내부결재 | **Yes** | 결재 승인 완료 및 확정 내용 확인 |
| STEP 3 | 모집부서 설정 | **Yes** | 모집부서·심사과정·자소서·첨부파일 항목 저장 완료 |

> 🚨 **STEP 2(내부결재)와 STEP 3(모집부서 설정)은 공고 게시 전 반드시 완료해야 하며, 게시 후에는 주요 항목의 수정이 제한됩니다.**
