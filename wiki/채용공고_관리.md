# 채용공고 관리

## 개요

울산대학교 직원 채용 공고 등록부터 관리까지의 전 과정을 안내합니다. 각 단계별로 필수 완료 항목과 주의사항을 확인한 후 진행하세요.

---

## STEP 1: 채용 TO 확인

**목적:** 채용 필요 인원(TO) 및 충원 사유를 확정하고 기본 일정을 수립합니다.

### 사전 확인사항

- [ ] 결원/증원 사유 정리 (퇴직, 신규사업, 정원조정 등)
- [ ] 직무/담당업무 범위 정리
- [ ] 채용 형태 (정규직/무기계약직/계약직 등) 및 기간 검토
- [ ] 예산/인건비 재원 확인 (필요 시)
- [ ] 채용 일정 (접수~임용) 큰 틀 수립

**⚠️ 주의:** 이후 단계에서는 변경이 어려우므로, TO·직무·일정의 기본 골격을 이 단계에서 확정해야 합니다.

---

## STEP 2: 채용공고 등록

**목적:** 채용 공고를 등록하고 공고문을 작성합니다.

**메뉴 경로:** `일반행정 > 채용관리 > 직원 > 채용공고 관리`

### 2-1. 템플릿 관리 (선택사항)

공고문 양식을 미리 만들어 두는 단계입니다.

**메뉴:** `일반행정 > 채용관리 > 직원 > 채용공고문 템플릿 관리`

1. **[추가]** 클릭
2. 템플릿명 입력 (예: `2025년 직원채용 기본양식`)
3. 에디터에 공고문 작성 (지원자격·제출서류·전형일정 등)
4. **[저장]** 클릭

**수정/삭제:** 목록에서 템플릿명 클릭 → 수정 후 [저장] / 체크박스 선택 후 [삭제]

⚠️ **중요:** 템플릿을 나중에 수정해도 **이미 등록된 채용공고문은 변경되지 않습니다.**

### 2-2. 채용공고 등록

1. **[추가]** 클릭 → 채용공고 관리 팝업 열림
2. 기본 정보 입력
3. **[저장]** 클릭
4. 채용공고문 **[선택]** 버튼 클릭 → 템플릿 조회 팝업 열림
5. 템플릿 선택 → **[공고문 템플릿 적용]** → 공고문 관리 팝업으로 이동
6. 공고문 내용 편집 후 **[저장]**

**필수 입력항목**

| 항목 | 입력 방법 | 예시/설명 |
| --- | --- | --- |
| 모집명 | 직접 입력 | `2025학년도 울산대학교 직원채용 공고` |
| 구분 | 드롭다운 | `직원` / `계약직원` |
| 담당부서 | 직접 입력 | `정보화개발팀` |
| 학가구분 | 드롭다운 | 미선택 시 전체 대상 |
| 모집시작일/시간 | 날짜+시간 | `2025.03.05 / 09:00` |
| 모집종료일/시간 | 날짜+시간 | `2025.05.05 / 17:00` |
| 진행상태 | Y/N 선택 | **Y = 사이트 게시됨** |

**선택 항목**

| 항목 | 설명 |
| --- | --- |
| 중복지원 가능여부 | 체크 시 같은 공고 내 다른 부서 복수지원 가능 |
| 타부서열람여부 | 체크 시 타부서 담당자 열람 가능 |

**진행상태별 사이트 표시**

| 진행상태 | 날짜 조건 | 사이트 표시 |
| --- | --- | --- |
| Y | 시작 전 | `오픈예정` |
| Y | 접수 중 | `접수 중` |
| Y | 종료 후 | `마감` |
| N | 무관 | 미게시 |

### 2-3. 공고문 작성

공고문 관리 팝업(메뉴 1.5)에서:

- **에디터:** 서식 있는 텍스트 편집 (HTML 직접 편집 탭도 제공)
- **첨부파일:** 지원서 양식 등 파일 업로드 가능
- **[저장]:** 작성한 공고문 저장

**💡 팁:** 아래 채용공고문 생성기를 이용하여 보기 좋은 공고문을 생성 후, HTML을 복사하여 UWIN 공고문 편집기 **HTML 모드**에 붙여넣으세요.

[채용공고문 생성기 바로가기](https://recruitment-generator.vercel.app/)

**⚠️ 중요:** 잘못 게시된 공고는 접수기간 내 수정 불가. 진행상태를 **N**으로 변경 후 신규 공고 등록해야 합니다.

## STEP 3: 모집부서 설정

**목적:** 공고별 모집 부서, 전형 단계, 지원 시 제출 서류를 설정합니다.

**메뉴 경로:** `일반행정 > 채용관리 > 직원 > 모집부서 관리`

**🚨 필수:** 공고 게시 전 반드시 완료해야 합니다. 게시 후에는 모집부서, 자기소개서, 첨부파일 항목 추가·수정·삭제 불가.

### 3-1. 모집부서 등록

1. 모집년도·모집명 선택 후 **[조회]**
2. **[추가]** 또는 기존 모집분야명 클릭
3. 팝업에서 항목 입력 후 **[저장]**

**입력항목**

| 항목 | 설명 | 예시 |
| --- | --- | --- |
| 모집부서 | 🔍 돋보기로 부서 검색·선택 | `정보화개발팀` |
| 모집분야 | 채용 직무명 | `전산직` |
| 모집인원 | 숫자 입력 | `2` |
| 지원자격 | 텍스트 직접 입력 | `학사학위 이상 소지자` |
| 우대사항 | 텍스트 직접 입력 | `정보처리기사 자격증 소지자` |

📌 **참고:** 팝업 내용은 **공고문에 자동 반영되지 않으며** 출력물·내부 관리용입니다.

⚠️ 게시 중인 공고의 모집부서는 삭제 불가 / 지원자가 있으면 삭제 불가

### 3-2. 심사과정(전형 단계) 설정

심사 단계별 배점을 설정합니다.

**메뉴:** 모집부서 관리 화면의 **[심사과정 관리]** 팝업

1. **[추가]** 클릭 → 단계 행 생성
2. 심사명·심사비율·최종점 입력 (예: `서류심사 / 50% / 100점`)
3. **[저장]** 클릭 → [심사위원] 버튼 활성화됨
4. **[선택]** 클릭 → 오른쪽 심사항목 영역에 표시
5. **[기본 설정 가져오기]** 또는 직접 추가 (공통코드 E93)
6. 심사항목별 배점 입력 → **합계 반드시 100점**
7. **[저장]** 클릭

**심사단계 구성 예시**

| 단계 | 심사명 | 비율(%) |
| --- | --- | --- |
| 1단계 | 서류심사 | 50 |
| 2단계 | 전공심사 | 20 |
| 3단계 | 면접심사 | 30 |

⚠️ **주의:**
- [심사위원] 버튼은 심사단계 저장 후에만 표시
- 배점 합계 ≠ 100점이면 저장 불가

### 3-3. 자기소개서 항목 설정

지원 시 제출 받을 자기소개서 항목을 설정합니다.

**메뉴:** 모집부서 관리 화면의 **[자기소개서 항목 관리]** 팝업

1. **[기본설정 가져오기]** 클릭 → 공통코드(E95) 표준 항목 자동 세팅
2. 항목 추가 필요 시 **[추가]** (신규 항목명 입력)
3. 글자수 제한 설정 (기본값 1000자)
4. **[저장]** 클릭

**기본 항목**
- 지원동기
- 성장과정
- 생활신조
- 연구역량 및 연구계획
- 강의계획

⚠️ **중요:** 이미 게시된 공고에서는 추가·수정·삭제 불가

### 3-4. 필수 첨부파일 항목 설정

지원 시 필수로 제출해야 할 서류를 설정합니다.

**메뉴:** 모집부서 관리 화면의 **[첨부파일항목 관리]** 팝업

1. **[기본설정 가져오기]** 클릭 → 공통코드(E88) 표준 서류 자동 세팅
2. 필수여부 체크박스 설정 (✅ = 필수, □ = 선택)
3. 불필요한 항목 삭제 (체크박스 → [삭제])
4. **[저장]** 클릭

**기본 제출서류 목록**

| # | 제출서류 | 필수 |
| --- | --- | --- |
| 1 | 연구실적목록 | ✅ |
| 2 | 연구실적 증빙자료 | ✅ |
| 3 | 학위별 학력증명서 및 성적증명서 | ✅ |
| 4 | 석·박사 학위 논문 | □ |
| 5 | 경력 및 재직증명서 | ✅ |
| 6 | 의사면허증/전문의자격증/간호사면허증 | □ |
| 7 | 교육관련자료 | ✅ |

⚠️ **중요:** 이미 게시된 공고에서는 추가·수정·삭제 불가

## 공고 게시 이후 단계

공고 게시 후 진행되는 심사, 합격자 관리 등의 단계는 별도 가이드를 참고하세요.

## 최종합격자 계약 정보 입력

**목적:** 최종합격자의 업무 및 급여 정보를 입력합니다. (계약서 작성 시 활용)

**메뉴 경로:** `일반행정 > 직원인사 > 임용관리 > 계약직원신규계약`

### 입력 절차

1. 메뉴 접근 후 **[조회]**
2. 작성 상태 확인하고 임시 사번(6자리 숫자) 클릭

### 입력항목

| 항목 | 입력 방법 | 예시/설명 |
| --- | --- | --- |
| 계약기간 | 날짜 입력 | `2026.04.01.~2027.03.31.` |
| 근무시간 | 시간 입력 | `09:00~18:00` |
| 담당업무 | 텍스트 입력 | `학사코디네이터` 등 기입 |
| 담당부서 | 복수 입력 가능 | 1. XXXXXXX / 2. AAAAAAA / 3. BBBBBBBB |
| 급여 정보 | 숫자 입력 | 급여 정보 유의하여 입력 |
| 급여 계정 정보 | 계정 선택 | 어떤 계정에서 지급하는지 명시 |

### 저장 및 신청 절차

1. **[저장]** 클릭 → 임시 저장
2. 저장 후 **[신청]** 버튼 생성 → 클릭
3. 인사부서에서 승인 대기 상태가 됨
4. **[인쇄]** 클릭 → 자체직 근무(변동) 신고서 PDF 다운로드
5. PDF를 `<채용결과 통보 및 계약의뢰 공문>`에 첨부

⚠️ **주의:**
- 소속 부서의 기존 입력 사례를 참고
