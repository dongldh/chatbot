## 1. 업무 개요

### 근태란
출근과 퇴근에 관련된 모든 사항을 관리하는 업무로, 기본적으로 **UWIN에서 신청 후 부서장 결재**를 통해 처리된다.

### 전결 기준 원칙
| 대상 | 전결권자 |
|---|---|
| 팀(실)원 근태 | 팀(실)장 전결 → 총무인사팀 제출 |
| 팀(실)장 근태 | 부서장(처장) 전결 → 총무인사팀 제출 |
| 총무처 소속 팀장 근태 | 총무인사팀장 전결 (중복 방지) |

### 증빙 자료 기준
- **증빙 불필요**: 연가, 하계휴가 등 자연 발생적 휴가
- **증빙 필요**: 병가, 출산휴가 등 사유가 필요한 휴가
- **특수 근태 보고**: 병가·청가 등 특수한 경우 총무처장 결재 필요

> **참고**: 연가는 위임전결규정에 따라 소속 부서 결재로 종료되며, 총무인사팀으로 결재 문서가 오지 않는다.

## 2. 정기 처리 업무

### 매일 처리 — 근태 접수

**경로**: `UWIN 교직원포탈 > 전자결재 > 부서수신함`

1. 수신된 휴가원 더블 클릭
2. (휴가원 팝업) 접수자 전결 클릭 / 결재자 지정
3. (기록물철 선택 팝업) **근태관계철** 더블클릭 후 확인 클릭
4. UWIN 비밀번호 입력 후 확인
5. 완료

---

### 매월 1일 처리

1. **인사발령 처리**: 1일자 발령 건 UWIN 등록 여부 확인
2. **호봉 승급 처리**: `일반행정 > 직원인사 > 임용관리 > 호봉및근속년수처리[CAD05U]`
3. **시점별 현황 생성**: `일반행정 > 직원인사 > 통계관리 > 시점별현황생성[CAH20U]`
   - 해당 월 1일자로 생성 버튼 클릭
4. **기본근로시간 생성**: `일반행정 > 직원인사 > 근로관리 > 기본근로시간관리(부서별)[CAK05U]`
   - **의과대학 3개 팀은 선택 해제** (출퇴근 시각이 달라 직접 입력)

**기본근로시간 설정값**

| 구분 | 요일 | 근무시간 | 휴게1 | 단축시간 | 휴게2 |
|---|---|---|---|---|---|
| 학기중 (월~목) | 월/화/수/목 | 09:00~18:00 | 12:00~13:00 | 00:00~00:00 | 18:00~18:30 |
| 학기중 (금) | 금 | 09:00~18:00 | 12:00~13:00 | 17:00~18:00 | 18:00~18:30 |
| 방학중 (월~금) | 월~금 | 09:00~18:00 | 12:00~13:00 | 15:00~18:00 | 18:00~18:30 |

---

## 3. 선택적 근로시간제

### 개요
연장근로에 대한 보상 방식으로 노사 합의를 통해 도입된 제도로, 대학 근로시간 제도의 핵심이다.

### 선근제 부서 등록 절차
1. 부서 요청 접수
2. 공문 접수
3. 협약서 조정
4. 전산 등록

**전산 등록 방법**: `일반행정 > 직원인사 > 기준정보관리 > 선택적근무부서등록[CAB13U]`
- 부서 조회 → 시작일 입력 → **종료일 입력(99991231)**
- **종료일을 입력하지 않으면 현재 부서로 확인되지 않으므로 반드시 입력**

---

### 매일 처리 — 연장근로 문서 접수

접수 경로: `그룹웨어 > 전자결재 > 부서수신함`
부서 활용 메뉴: `일반행정 > 직원인사 > 근로관리 > 부서근태관리(일간)[CAK07U]`

**연장근로 확인 필수 항목**

| 항목 | 내용 |
|---|---|
| 시작·종료시간 | 연장근로로 인정받을 시간 |
| KT 출퇴근시간 | 출퇴근 단말기 기록 시간 |
| 내용(사유) | 사유 불명확 또는 미기록 시 반송 |
| 부서장 결재 여부 | **팀장 전결 불가 — 부서장 결재 필수** |

---

### 방학 중 대체휴무 사용 가능 여부 확인

**확인 메뉴**: `일반행정 > 직원인사 > 근로관리 > 개인별근태관리[CAK02U]`

- 연장근로(A) 시간 확인
- 단축근무(B) 시간 확인
- **A − B = 사용 가능한 대체휴무 시간**

> **예시**: 연장근로 10시간, 단축근무 5시간 → 5시간 대체휴무 사용 가능

---

### 방학 중 근태 신청 관련 자주 묻는 질문

| 문의 내용 | 답변 |
|---|---|
| 반일연가 + 외출 + 단축근무 조합으로 연가 효과 가능한지 | **불가** — 외출은 근무지 복귀 전제, 근무지 미출근 상태에서 신청 불가 |
| 지각 + 조퇴 + 단축근무 조합 가능한지 | **불가** — 지각은 출근을 전제로 사용 |
| 조퇴(13~15시) + 단축근무로 반일연가 효과 가능한지 | **불가** — 조퇴 종료 시각은 근무 종료시각(18:00)으로 신청해야 함 |
| 외출(13~15시) + 단축근무 조합 | **일부 가능** — 15:00 복귀 후 단축근무 적용 가능 |

## 4. 연장근로 및 보상휴가

### 보상휴가 기본 원칙
- 발생한 보상휴가는 **모두 사용하거나 수당으로 보상**해야 한다.
- 보상휴가 유효기간: 발생 후 최소 6~11개월
- 수당으로 보상한 경우 **반드시 기록을 남겨야 한다.**

### 보상휴가 정산 처리 절차

1. **보상휴가 잔여시간 확인**
   - 전체 발생일수: `일반행정 > 직원인사 > 근로관리 > 보상휴가현황[CAK10U]`
   - 전체 사용일수: `일반행정 > 직원인사 > 복무관리 > 휴가신청조회[CAC10U]`
2. **미사용자 보상금액 산정**: `일반행정 > 직원인사 > 근로관리 > 근태현황[CAK11P]` → 6. 연장근로수당발생현황 → 시작월 선택 후 [엑셀] 클릭
3. **연장근로수당 신청서 접수**
4. **노동조합 사전 통보**
5. **보상 처리 후 기록 남기기**

### 수당 지급 시 보상휴가 차감 방법

**위치**: `일반행정 > 직원인사 > 복무관리 > 휴가신청[CAC09U]`

- 휴가구분: '보상휴가 수당지급건' 선택
- 휴가기간 비활성화
- 차감 필요 시간 입력 (예: 73.25시간)
- 비고에 '연장근로수당 지급 완료' 입력 → 저장 → **총무처장 결재**

> **주의**: 수당으로 지급한 시간을 차감하지 않으면 미사용 보상휴가로 남아 추후 이중 지급 문제가 발생할 수 있다.

---

### 연장근로수당 지급 처리 순서

1. 일자별 원데이터 수집 및 확인
2. 회계팀 사전 통보 (예상 금액 안내)
3. **휴일근로 가산시간** 정상 산정 여부 확인
4. **야간근로 가산시간** 정상 산정 여부 확인
5. 휴직자 통상임금 재산정 (휴직 전 12개월 기준)
6. 일반근로시간제 부서 보상시간 별도 확인: `근태현황[CAK11P]` → 7. 연장근로 발생현황(일반근로제)
7. 연장근로수당 신청서 수령 및 서명
8. 총장 결재 상신 (처장 전자결재 진행)
9. 지출품의서 및 지출결의서 작성
10. 정상 지출 여부 확인

---

### 52시간 초과자 관리

**월 허용 근로시간 계산식**: `해당 월 총 일수 / 7 × 52`
- 예: 31일 기준 → 31/7 × 52 = 약 230시간

**52시간 초과 시 처리 원칙**
- 초과 시간은 다른 달로 이전하여 처리 (근거 확보 필수)
- **차량 출입 기록 등과 대조 시 문제가 없도록 개연성 있게 처리**
- 관리자(부서장)가 52시간 기준을 인지하고 업무 조정해야 함

3. 휴가시작일 클릭 → 이름 또는 사번 조회
