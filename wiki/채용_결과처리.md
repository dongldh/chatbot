# 채용 결과 처리 (성범죄 조회 · 결과처리 · 합격자 발표 · UWIN 이관)

> **구분**: 신규임용 | **관련 단계**: STEP 11 → 14 → 15 → 16 → 17
> **관련 메뉴**: `1.21 전형별 심사 결과` / `1.22 최종합격자 관리` / `1.23 합격자발표 관리`

---

## 전체 처리 흐름

```
① 심사 결과 처리 (서류·필기)
    → ② 심사 결과 처리 (면접)
        → ③ 성범죄 사실 조회
            → ④ 최종합격자 확정 및 UWIN 이관
                → ⑤ 합격자 발표 (메일 통보)
```

## ① 심사 결과 처리 (STEP 11 · 14)

### 화면 진입 경로

`일반행정 > 채용관리 > 직원 > 전형별 심사 결과`

### 처리 절차

1. 조회 조건(모집연도·모집명·심사구분) 선택 후 **[조회]** 클릭
   - 지원자 목록과 심사위원별 점수 확인
2. 사정일자 설정 후 **[심사사정]** 클릭
   - 총점 기준 자동 순위 산정
3. 가산점 입력 (해당자에 한함)
   - 총점·평균점수 자동 반영
4. 합격 여부 선택 (`합격` / `불합격` / `예비순위`)
   - **담당자가 직접 선택** (심사사정은 등수만 산정, 합격 여부는 자동 처리되지 않음)
5. **[저장]** 클릭 → 임시 저장 상태 (아직 확정 아님)
6. **[확정]** 클릭 → 확정일자 등록 완료

> **면접 전형(STEP 14)**도 동일한 화면·방법으로 처리하며, 심사구분을 `면접`으로 선택하여 진행합니다.

### 합격 결과 구분

| 결과값 | 의미 | 다음 단계 처리 |
|--------|------|----------------|
| `합격` | 다음 심사단계 진행 대상 | 다음 전형에서 조회 가능 |
| `불합격` | 탈락 처리 | 다음 전형에서 미표시 |
| `예비순위` | 추후 합격 전환 가능 | 담당자 별도 관리 |

### ⚠️ 주의사항

- **확정 후에는 합격여부 수정 불가** (가산점·비고 항목만 수정 가능)
- 오확정 시: **[확정취소]** 클릭 → 확정일자 공란 처리 → 결과 수정 → 재확정
- **합격 처리된 지원자만** 다음 심사구분에서 조회 가능

## ② 성범죄 사실 조회 (STEP 15)

> 🚨 **최종합격자를 대상으로 임용일 이전에 반드시 완료해야 합니다.**

### 의뢰기관

- **울산남부경찰서**

### 처리 절차

1. 최종합격 대상자에게 조회 목적 안내 및 **동의서 수령 (원본 보관)**
2. 대상자 명단을 첨부하여 울산남부경찰서 앞 **외부발신 공문 발송**
3. 회신 결과 수신 (통상 **2~3일 이내**)
4. 결격 여부를 총무인사팀에 통보

### 산출물

| 항목 | 비고 |
|------|------|
| 조회 요청 공문 및 대상자 명단 | 외부 발송 |
| 동의서 | **원본 보관 필수** |
| 회신 결과 문서 | 결격 여부 확인 후 팀 내 보고 |

## ③ 최종합격자 확정 및 UWIN 이관 (STEP 16)

### 화면 진입 경로

`일반행정 > 채용관리 > 직원 > 최종합격자 관리`

### 처리 절차

1. **[조회]** 클릭 → `1.21 전형별 심사 결과`에서 합격 처리된 지원자 목록 표시
2. 최종 결과 선택 (`합격` / `불합격` / `예비순위`)
3. **[저장]** 클릭 → 임시 저장
4. **[확정]** 클릭 → 확정일자 등록 (이후 합격여부 수정 불가)
5. 합격 처리된 지원자의 체크박스 선택 (합격자만 체크박스 활성화됨)
6. **[합격자이관]** 클릭 → **교직원정보 시스템(UWIN)으로 데이터 이관**
7. **[모집현황 출력]** → 모집현황 리포트 출력 및 보관

### ⚠️ 주의사항

- **이관 전 교직원정보 데이터 정확성을 반드시 재확인**하세요.
- 확정 취소 시: **[확정취소]** 클릭 → 확정일자 공란 처리
- **합격통보 메일은 확정일자 등록 후에만 발송 가능** (다음 단계 진행 전 반드시 확정 완료)

## ④ 합격자 발표 (메일 통보) (STEP 17)

### 화면 진입 경로

`일반행정 > 채용관리 > 직원 > 합격자발표 관리`

### 발송 전 필수 체크리스트

| 항목 | 확인 |
|------|------|
| 모집연도·모집명 조회 완료 | ☐ |
| `1.22`에서 확정일자 등록 완료 | ☐ |
| 합격·불합격 메일 문구 입력 완료 | ☐ |
| 테스트 발송으로 문구 최종 확인 | ☐ |
| **[실제발송]** 클릭 | ☐ |

### 처리 절차

1. 모집연도·모집명 선택 후 **[조회]** (필수 입력 항목)
2. 확정일자·합격자발표 기간(시작~종료 일시) 입력 후 **[저장]**
3. 합격 메시지 **[메일]** 버튼 클릭 → 에디터에서 합격자 메일 문구 작성
4. 불합격 메시지 **[메일]** 버튼 클릭 → 불합격자 메일 문구 작성
5. **[테스트 — 메일]** 클릭 → 로그인 사용자 본인에게 테스트 발송 후 **문구 최종 확인**
6. **[실제발송 — 메일]** 클릭 → 합격자·불합격자에게 실제 발송

> 📌 합격·불합격 메시지 문구 입력 완료 시 해당 체크박스가 자동 체크 후 비활성화됩니다.

### 메일 문구 예시

**합격 안내 메일**

> **제목**: \[울산대학교\] 직원 채용 최종합격 안내(임용 예정)
>
> 안녕하세요. 울산대학교 XXXXX팀입니다.
> \[채용명\] 최종합격을 안내드립니다.
>
> 임용(출근) 관련 주요 안내사항은 신규임용 직원 안내자료를 확인해 주세요.
> 세부 안내(계약서 작성 일정, 제출서류 등)는 확정되는 대로 별도 안내드리겠습니다.
> 감사합니다.

**불합격 안내 메일**

> **제목**: \[울산대학교\] 직원 채용 결과 안내
>
> 안녕하세요. 울산대학교 XXXXX팀입니다.
> \[채용명\] 전형 결과를 안내드립니다.
>
> 이번 전형에서는 안타깝게도 선발되지 않으셨습니다.
> 지원해 주셔서 감사합니다.

### ⚠️ 주의사항

- 🚫 **SMS 발송은 현재 발신번호 제약으로 실사용 불가 — 메일로만 운용**
- **확정일자가 등록되지 않은 경우 메일 발송 불가** → `1.22 최종합격자 관리`에서 먼저 확정 처리

## ❓ FAQ

| 질문 | 답변 |
|------|------|
| 실수로 불합격 확정했어요 | **[확정취소]** 클릭 → 결과 수정 → 재확정 |
| 합격 통보 메일이 발송되지 않아요 | `1.22`에서 **확정일자 등록** 후 재시도 |
| SMS 합격 통보가 안 돼요 | 현재 발신번호 제약으로 SMS 불가. **메일로만 발송**하세요 |
| 엑셀 심사점수 업로드 오류가 납니다 | 반드시 시스템에서 **[엑셀 다운로드]로 받은 서식 파일** 사용. 자체 제작 파일 업로드 불가 |
| 지원자가 임시저장 상태예요 | `1.13 채용지원자 관리`에서 진행상태를 `제출`로 변경 (부득이한 경우에만 사용) |
| 계약직 교직원의 지원 예외 허용이 필요해요 | `1.19 지원불가자예외 관리`에서 해당 교직원 등록 |
