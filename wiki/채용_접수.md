## 📍 화면 진입 경로

`일반행정 > 채용관리 > 직원 > 채용지원자 관리`

## 1. 지원자 현황 조회 및 관리 (메뉴 1.13)

### 조회 및 주요 기능 버튼

1. `일반행정 > 채용관리 > 직원 > 채용지원자 관리` 메뉴 진입
2. 아래 조건으로 지원자 검색 후 **[조회]** 클릭
   - 모집연도, 모집명, 모집부서, 분야, 지원자번호/성명

| 버튼 | 기능 |
|------|------|
| **[조회]** | 모집연도·모집명·모집부서·분야·지원자번호/명으로 검색 |
| **[추가]** | 관리자가 지원자 직접 등록 (1.14 팝업 실행) |
| **[삭제]** | 체크박스 선택 지원자 삭제 (심사점수 입력 전만 가능) |
| **[지원서 출력]** | 지원자 정보 리포트 출력 |
| **[자기소개서 출력]** | 자기소개서 리포트 출력 |
| **[공고별 제출서류 일괄다운로드]** | 전체 지원자 첨부파일 압축 다운로드 |

### 탭별 서류 검토 항목

| 탭 | 관리 항목 |
|----|-----------|
| **기본** | 성명·생년월일·성별·연락처·이메일·국적·학가구분 등 |
| **학력사항** | 학력구분·학교명·학과명·학위·입학일·졸업일·취득국가 |
| **경력사항** | 기관명·직위명·재직기간·근무년수·담당직무·임상여부 |
| **자격,면허,상,벌** | 자격증·면허·수상·징계 이력 |
| **자기소개서** | 항목별 자소서 내용 (설정된 항목만 표시) |
| **제출서류** | 첨부파일 업로드·다운로드·일괄다운로드 |

## 2. 지원자 직접 등록 (메뉴 1.14 Popup)

오프라인 접수 지원자 등록 또는 **시스템 오류 발생 시** 관리자가 직접 등록합니다.

1. **[추가]** 버튼 클릭 → `1.14 지원자 등록` 팝업 실행
2. 아래 항목 입력 후 저장

| 입력 항목 | 내용 |
|-----------|------|
| 모집명 | 드롭다운 선택 |
| 모집부서 / 모집분야 | 드롭다운 선택 |
| 한글 성명 / 영문 성명 / 한자 성명 | 직접 입력 |
| 생년월일 | 드롭다운 선택 |
| 성별 | 남 / 여 선택 |

> 📌 **지원번호는 저장 시 시스템이 자동 생성**됩니다. 별도 입력 불필요.

---

## 3. 진행 상태 변경 (임시저장 → 제출)

지원자가 제출한 입사지원서의 수정이 필요한 경우, 상태를 변경하여 재작성 가능한 상태로 전환합니다.

1. `1.13 채용지원자 관리` 에서 해당 지원자 선택
2. 진행 상태를 **[임시저장]** 으로 변경 후 저장
3. **변경 후 반드시 해당 지원자에게 재작성 가능 상태임을 안내**

> ⚠️ **진행 상태 변경(임시저장 → 제출 포함)은 부득이한 경우에만 사용**하며, 무분별한 변경을 지양합니다.

---

## 4. 지원불가자 예외 처리 (메뉴 1.19)

지원 희망자가 시스템상 지원이 불가하다고 연락이 오는 경우 아래 절차를 따릅니다.

1. 지원 희망자로부터 지원 불가 관련 문의 접수
2. **인사부서(총무인사팀)에 연락하여 예외자 처리 요청**
3. 총무인사팀에서 `1.19 지원불가자예외 관리` 메뉴를 통해 예외 처리 진행

---

## ⚠️ 핵심 주의사항

- **심사점수가 입력된 지원자는 삭제 불가**합니다. 삭제가 필요한 경우 심사점수 입력 전에 처리해야 합니다.
- **진행 상태 변경(임시저장 → 제출)은 부득이한 경우에만 사용**합니다.
- 지원불가자 예외 처리는 반드시 **총무인사팀을 통해** 진행합니다.
- 지원자 상태 변경 후에는 **반드시 해당 지원자에게 변경 사실을 안내**합니다.

## ❓ FAQ

**Q. 입사지원서를 제출한 지원자가 수정을 희망할 때 어떻게 하나요?**

> 지원자의 진행 상태 정보를 **[임시저장]** 으로 변경 후 저장하면 재작성이 가능한 상태가 됩니다. 변경 후 반드시 대상자에게 재작성 가능함을 안내하시기 바랍니다.
