## STEP 15 · 성범죄 사실 조회

### 개요

- **목적**: 최종합격자 대상 성범죄 경력 조회 (임용 전 완료 필수)
- **의뢰기관**: 울산남부경찰서
- **산출물**: 조회 요청 공문/명단, 동의서(원본 보관), 회신 결과

> 🚨 **최종합격자 대상으로 임용일 이전에 조회 완료가 필요합니다.**

### 처리 절차

1. 대상자에게 조회 안내 및 **동의서 수신 (원본 보관)**
2. 대상자 명단을 첨부하여 울산남부경찰서로 외부발신 공문 발송
3. 결과 회신 수신 (통상 **2~3일 내**)
4. 결격 여부를 총무인사팀에 통보

## STEP 16 · 최종합격자 확정 및 UWIN 이관

### 개요

- **목적**: 최종합격 결과 확정 및 교직원정보 시스템으로 데이터 이관
- **UWIN 경로**: `일반행정 > 채용관리 > 직원 > 최종합격자 관리` (메뉴 1.22)

### 처리 절차

| 순서 | 버튼/동작 | 설명 |
|------|-----------|------|
| 1 | **[조회]** | 이전 단계에서 합격 처리된 지원자 표시 |
| 2 | 결과 선택 | `합격` / `불합격` / `예비순위` 최종 확정 |
| 3 | **[저장]** | 임시 저장 |
| 4 | **[확정]** | 확정일자 등록 — **이후 합격여부 수정 불가** |
| 5 | 합격자 체크박스 선택 | 합격 처리된 지원자만 체크박스 활성화 |
| 6 | **[합격자이관]** | 교직원정보 시스템으로 데이터 이관 |
| 7 | **[모집현황 출력]** | 모집현황 리포트 출력 |

### 주의사항

- **이관 전 교직원정보 데이터 정확성을 반드시 재확인**
- 확정 취소가 필요한 경우: **[확정취소]** 클릭 → 확정일자 공란 처리 → 결과 수정 후 재확정
- **합격 통보 메일은 확정일자 등록 후에만 발송 가능** (확정 전 발송 불가)

## STEP 17 · 합격자 발표 (메일 통보)

### 개요

- **목적**: 합격자·불합격자에게 전형 결과 메일 발송
- **UWIN 경로**: `일반행정 > 채용관리 > 직원 > 합격자발표 관리` (메뉴 1.23)

### 발송 전 체크리스트

- [ ] 모집연도·모집명 조회 완료
- [ ] 확정일자가 등록되어 있음 (STEP 16 완료 확인)
- [ ] 합격/불합격 메시지 문구 입력 완료
- [ ] **[테스트 — 메일]** 로 문구 최종 확인 (로그인 사용자 본인에게 발송)
- [ ] **[실제발송 — 메일]** 클릭

### 처리 절차

| 순서 | 버튼/동작 | 설명 |
|------|-----------|------|
| 1 | 모집연도·모집명 선택 후 **[조회]** | 필수 입력 항목 |
| 2 | 확정일자·합격자발표 기간 입력 후 **[저장]** | 발표 시작~종료일시 설정 |
| 3 | 합격 메시지 **[메일]** 버튼 | 에디터로 합격자 메일 문구 작성 |
| 4 | 불합격 메시지 **[메일]** 버튼 | 에디터로 불합격자 메일 문구 작성 |
| 5 | **[테스트 — 메일]** | 본인에게 테스트 발송 → 문구 확인 필수 |
| 6 | **[실제발송 — 메일]** | 합격자·불합격자에게 실제 발송 |

### 메일 예시 문구

**합격 안내 메일**

> **제목**: \[울산대학교\] 직원 채용 최종합격 안내(임용 예정)
>
> 안녕하세요. 울산대학교 XXXXX팀입니다.
> \[채용명\] 최종합격을 안내드립니다.
>
> 임용(출근) 관련 주요 안내사항은 아래 안내자료에서 확인해 주세요.
> 세부 안내(계약서 작성 일정, 제출서류 등)는 확정되는 대로 별도 안내드리겠습니다.
> 감사합니다.

**불합격 안내 메일**

> **제목**: \[울산대학교\] 직원 채용 결과 안내
>
> 안녕하세요. 울산대학교 XXXXX팀입니다.
> \[채용명\] 전형 결과를 안내드립니다.
>
> 이번 전형에서는 안타깝게도 선발되지 않으셨습니다.
> 지원해 주셔서 감사합니다.

### 주의사항

- 🚫 **SMS 발송은 현재 발신번호 제약으로 실제 사용 불가 — 메일로만 발송**
- 합격/불합격 문구 입력 완료 시 체크박스 자동 체크 후 비활성화 처리됨

## STEP 18 · 직무 및 급여정보 세부정보 입력

### 개요

- **목적**: 최종합격자의 업무 및 급여(보수) 정보 입력
- ⭐ **계약서 작성 시 활용되는 핵심 영역 — 정확히 입력할 것**
- **UWIN 경로**: `일반행정 > 직원인사 > 임용관리 > 계약직원신규계약`

### 입력 절차

1. 메뉴 접근 후 **[조회]**
2. 작성 상태 확인 후 **임시 사번(숫자 6자리)** 클릭
3. 아래 항목 순서대로 입력

| 순서 | 항목 | 예시/설명 | 비고 |
|------|------|-----------|------|
| 1 | 계약기간 | `2026.04.01. ~ 2027.03.31.` | |
| 2 | 근무시간 | `09:00 ~ 18:00` | |
| 3 | 담당업무 | 업무 목록 기재 | |
| 4 | 담당직무 | 학사코디네이터 등 직무명 기입 | |
| 5 | 급여 정보 | 급여 정보 유의하여 입력 | |
| 6 | 급여 계정 정보 | 급여 지급 계정 입력 | **계정 오입력 방지를 위해 철저 확인 필요** |
| 7 | **[저장]** | 임시 저장 | |
| 8 | **[신청]** | 저장 완료 후 신청 버튼 클릭 → 인사부서 승인 대기 상태 | |
| 9 | **[인쇄]** | 자체직 근무(변동) 신고서 PDF 다운로드 → 채용결과 통보 공문에 첨부 | 별도 수기 결재 불필요, PDF 파일만 첨부 |

### 유의사항

- 소속 부서의 **기존 입력 사례를 참고**하여 일관되게 입력
- **인건비 재원이 사업비 등으로 별도 편성된 경우 반드시 확인 후 입력**

## STEP 19 · 채용결과 통보 및 계약의뢰 (→ 총무인사팀)

### 개요

- **목적**: 채용 결과(최종합격자)를 총무인사팀에 통보하고 계약서 작성 의뢰
- **처리 기한**: **임용(채용)일 5일 전까지** 공문 송부 (채용부서 → 총무인사팀)
- **산출물**: 채용결과 통보 + 계약요청 공문, 첨부서류 일체 (**자체직 근무(변동) 신고서 필수 첨부**)

### 처리 절차

1. (채용부서) 최종합격자 확정 자료 정리 후 공문 작성
   - 포함 내용: 명단, 심사자료, 성범죄 조회 결과, **자체직 근무(변동) 신고서 (STEP 18에서 PDF 출력)**
2. (채용부서 → 총무인사팀) 공문 발송
3. (총무인사팀) 접수 후 계약서 작성 일시를 공문 발신자에게 안내
4. (채용부서) 안내받은 일시에 **총무인사팀 방문**

### 공문 작성 양식

```
제목: 총무인사팀 계약직원 채용결과 통보 및 계약 요청

본문
1. 관련: (최종 내부결재 문서번호 및 제목 기입)
2. 관련 문서에 의거하여 계약직원 채용 결과를 통보드립니다.
3. 해당 인원의 계약서 작성을 의뢰하오니 협조하여 주시기 바랍니다.

   가. 성  명:
   나. 담당업무:
       -
       -
   다. 생년월일:
   라. 임용일자:
   마. 급    여:
   바. 급여재원:

붙임 1. 관련 공문 각 1부.
     2. 자체직 근무(변동) 신고서 1부.  끝.
```
