# 합격자 처리 및 발표

## 개요

심사 결과 확정부터 최종합격자 발표, UWIN 시스템 이관, 인사부서 계약의뢰까지의 전체 프로세스를 다룹니다.

**주요 단계:**
1. 심사 결과 확정 (1.21 전형별 심사 결과)
2. 최종합격자 확정 및 이관 (1.22 최종합격자 관리)
3. 합격자 발표 및 통보 (1.23 합격자발표 관리)
4. 인사부서 계약의뢰 (STEP 19)

---

## 1단계: 심사 결과 확정 처리

### 화면 진입 경로

`일반행정 > 채용관리 > 직원 > 전형별 심사 결과`

### 처리 순서

1. 조회 조건 선택 후 **[조회]** 버튼 클릭
   - 지원자 목록 및 심사위원별 점수 확인

2. 사정일자 설정 후 **[심사사정]** 클릭
   - 총점 기준으로 자동 순위 산정

3. 가산점 입력 (해당자만)
   - 총점·평균점수 자동 반영

4. 결과 선택
   - `합격`: 다음 심사단계 진행 대상
   - `불합격`: 탈락 처리
   - `예비순위`: 추후 합격 전환 가능

5. **[저장]** 클릭
   - 임시 저장 상태

6. **[확정]** 클릭
   - ⚠️ **확정일자 등록 후, 합격여부는 수정 불가** (가산점·비고만 수정 가능)
   - **합격 결과 지원자만** 다음 심사구분에서 조회 가능

### 주요 주의사항

- **확정 후 수정 제약**: 확정일자가 등록되면 합격여부(합격/불합격/예비순위) 변경 불가
- **확정 취소 절차**: 오확정 시 **[확정취소]** → 확정일자 공란 처리 → 결과 수정 → 재확정
- **심사사정의 역할**: 점수 기준 등수만 산정하며, 합격/불합격/예비순위는 **담당자가 직접 선택 후 확정**

## 2단계: 최종합격자 확정 및 이관

### 화면 진입 경로

`일반행정 > 채용관리 > 직원 > 최종합격자 관리`

### 처리 순서

1. **[조회]** 클릭
   - 1.21에서 합격 처리된 지원자 표시

2. 결과 선택
   - 최종 합격/불합격/예비순위 재확인

3. **[저장]** 클릭
   - 임시 저장

4. **[확정]** 클릭
   - 확정일자 등록
   - 이후 합격여부 수정 불가

5. 합격자 체크박스 선택
   - 합격 처리된 지원자만 체크박스 활성화

6. **[합격자이관]** 클릭
   - **교직원정보 시스템으로 데이터 이관**

7. **[모집현황 출력]**
   - 모집현황 리포트 출력

### 주요 주의사항

- ⚠️ **이관 전 교직원정보 정확성 재확인 필수**
- 확정 취소: **[확정취소]** → 확정일자 공란 처리

## 3단계: 합격자 발표 및 통보

### 화면 진입 경로

`일반행정 > 채용관리 > 직원 > 합격자발표 관리`

### 사전 확인 체크리스트

| 항목 | 확인 |
|------|------|
| 모집연도·모집명 조회 완료 | ☐ |
| **확정일자가 등록되어 있음** | ☐ |
| 합격/불합격 메시지 문구 입력 완료 | ☐ |
| 테스트 발송으로 문구 최종 확인 | ☐ |

### 처리 순서

1. 모집연도·모집명 선택 후 **[조회]** 클릭 (필수)

2. 확정일자·합격자발표 기간 입력 후 **[저장]**
   - 발표 시작일시 ~ 종료일시 설정

3. 합격 메시지 **[메일]** 버튼 클릭
   - 에디터로 합격자 메일 문구 작성

4. 불합격 메시지 **[메일]** 버튼 클릭
   - 불합격자 메일 문구 작성

5. **[테스트 — 메일]** 클릭
   - ✅ 로그인 사용자 본인에게 테스트 발송
   - **문구 확인 필수**

6. **[실제발송 — 메일]** 클릭
   - 합격자·불합격자에게 실제 발송

### 메일 작성 가이드

#### 합격 안내 메일 예시

- **제목**: [울산대학교] 직원 채용 최종합격 안내(임용 예정)
- **본문 구성**:
  - 인사말
  - 채용명 + 최종합격 안내
  - 임용(출근) 관련 주요 안내사항
    - 기본안내자료: 울산대학교 신규임용 직원 안내자료 링크
    - 추가안내사항: 부서별 특정 내용
  - 세부 안내 향후 별도 공지 안내

#### 불합격 안내 메일 예시

- **제목**: [울산대학교] 직원 채용 결과 안내
- **본문 구성**:
  - 인사말
  - 전형명 + 불합격 안내
  - 감사의 말씀

### 메일 발송 필수 조건

- ⚠️ **합격통보 메일은 확정일자 등록 후에만 발송 가능**
- 합격/불합격 메시지 문구 입력 완료
- 테스트 메일로 문구 사전 확인
- 🚫 **SMS 발송은 현재 발신번호 제약으로 실제 사용 불가 (메일만 운용)**

## 4단계: 인사부서 계약의뢰

### 개요

최종합격자 명단을 인사부서에 통보하고 계약서 작성 의뢰합니다.

### 처리 기한

**원칙: 임용(채용일) 5일 전까지** 공문 송부 (채용부서 → 총무인사팀)

### 처리 절차

1. (채용부서) 최종합격자 확정 자료 정리
   - 합격자 명단
   - 심사자료
   - 성범죄 조회 결과 등
   - **자체직 근무(변동) 신고서 필수 첨부** (STEP 18 참고)

2. (채용부서) 결과 통보 + 계약요청 공문 작성
   - 총무인사팀에 송부

3. (인사부서) 접수 시 계약서 작성 진행

4. (인사부서) 공문 발신자에게 계약서 작성 일시 안내

5. (채용부서) 해당 일시에 총무인사팀 방문하여 계약서 작성

### 공문 작성 템플릿

**제목**: 총무인사팀 계약직원 채용결과 통보 및 계약 요청

**본문**:
```
1. 관련 : (최종내부결재 문서 번호 및 제목)

2. 관련 문서에 의거하여 계약직원 채용 결과를 통보드립니다.

3. 해당 인원의 계약서 작성을 의뢰하오니 협조하여 주시기 바랍니다.

   가. 성 명 :
   나. 담당업무
       - 
       - 
   다. 생년월일 :
   라. 임용일자 :
   마. 급 여 :
   바. 급여재원 :

붙임 1. 관련 공문 각 1부.
     2. 자체직 근무(변동) 신고서 1부.

                                              끝.
```

### 주요 주의사항

- ⚠️ **자체직 근무(변동) 신고서는 필수 첨부 서류**
- 인사부서 계약서 작성 일정을 미리 확인하고 준비
- 임용일 5일 전까지 공문 송부 필수

## 자주 묻는 질문 (FAQ)

### Q1. 심사 결과를 실수로 불합격 처리하고 확정했어요. 어떻게 하나요?

**[확정취소]** 버튼을 클릭하면 확정이 해제됩니다. 이후 결과를 수정하고 다시 확정하세요.

### Q2. 합격 통보 메일이 발송되지 않습니다.

**확정일자가 등록되어 있는지 확인**하세요. 1.22 메뉴에서 최종합격자를 먼저 확정하고, 확정일자를 등록한 후에만 메일 발송이 가능합니다.

### Q3. SMS로 합격 통보를 보낼 수 없나요?

🚫 **현재 발신번호 제약으로 SMS 실제 사용이 불가합니다.** 메일로만 발송하세요.

### Q4. 합격자 이관 전에 확인해야 할 사항이 있나요?

**교직원정보 시스템의 데이터 정확성을 재확인**하세요. 이관 후 수정이 어려울 수 있으므로 성명, 생년월일, 연락처 등의 정보가 올바른지 확인해야 합니다.

### Q5. 테스트 메일은 어디로 발송되나요?

**로그인한 사용자 본인의 이메일 주소**로 테스트 메일이 발송됩니다. 실제 발송 전에 이를 통해 메일 문구를 최종 확인하세요.

### Q6. 메일 본문에 개행이 제대로 반영되지 않습니다.

에디터에서 작성할 때 줄바꿈을 정확히 입력하고, 테스트 메일로 반드시 확인하세요. 에디터의 기능을 활용하여 문서 형식을 조정할 수 있습니다.

### Q7. 합격자발표 기간을 설정하는 이유가 뭔가요?

합격자발표 기간(시작 ~ 종료)은 지원자들이 합격 결과를 확인할 수 있는 공시 기간을 의미합니다. 공정성을 위해 명확히 설정하여 운영하세요.

### Q8. 계약서 작성은 어디서 이루어지나요?

계약서는 **총무인사팀에서 작성**됩니다. 채용부서는 인사부서가 안내한 일시에 총무인사팀을 방문하여 계약서를 작성합니다.
