# 채용 내부결재

## 📍 개요

- **목적**: 채용 진행에 대한 내부 승인(결재) 확보
- **범위**: 신규 직원 채용계획(안)에 대한 부서장급 이상 결재 문서

---

## ⚠️ 핵심 주의사항

**내부결재 완료 후 공고 등록을 진행하세요.**

**결재 문서 확정 전에 공고 내용을 작성하지 마세요.** (변경 리스크 발생)

---

## 📋 결재 전 준비사항

결재 문서 작성 전 다음 항목들을 미리 정리하세요.

- [ ] 채용 사유/배경 및 필요성 정리
- [ ] 직무/담당업무 및 자격요건(필수/우대) 정리
- [ ] 전형 단계/방법(서류, 면접 등) 및 합격자 선발 기준 초안
- [ ] 채용 일정(공고, 접수, 심사, 발표, 임용) 초안
- [ ] 예산/보수/계약기간(해당 시) 검토
- [ ] **부서별 TO(정원) 사전 확인**

---

## 📝 내부결재 문서 작성

### 기본 구성

결재 문서는 다음과 같은 구조로 작성합니다.

#### 1단계: 기본 정보 입력
- **제목**: `[부서명] [직종] 채용(안)` 형식
  - 예: "총무처 총무인사팀 발간실 계약직원 채용(안)"

#### 2단계: 배경 및 사유 작성

채용의 필요성을 명확히 기술합니다.

```
[부서] [직무]운영을 위하여 다음과 같이 [직종]을 채용하고자 합니다.
```

#### 3단계: 퇴직 예정 인원 명시 (해당 시)

| 항목 | 내용 |
|------|------|
| 소속 | 부서 정확한 명칭 |
| 성명 | 담당자 이름 |
| 퇴직예정일 | 연.월.일(요일) 형식 |

#### 4단계: 신규채용 기본사항

| 항목 | 작성 예시 |
|------|----------|
| 직종 | 단기계약직, 기간제직원 등 |
| 근무부서 | 배치 예정 부서(팀) |
| 채용인원 | 명 |
| 채용방법 | 공개채용, 수시채용 등 |
| 임용예정일 | YYYY.MM.DD 형식 |

#### 5단계: 상세 정보 기술

- **담당 업무**: 구체적인 업무 내용
- **채용 절차**: 서류접수 → 서류심사 → 면접심사 → 최종합격
- **지원 자격 및 급여 기준**: 본교 해당 직급 수준 명시
- **급여 재원**: 교비, 특별회계 등

### 첨부 자료 준비

**내부결재 첨부자료로는 다음을 포함하세요:**

- 간단한 채용 사유
- 해당 직무의 필요성
- 직무 설명서(Job Description) 또는 담당 업무 내용
