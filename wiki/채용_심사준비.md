# 채용 심사 준비 (심사위원 등록 ~ 접수 관리)

> **적용 구분:** 신규임용
> **해당 단계:** STEP 5 ~ STEP 8
> **관련 메뉴:** 1.8 / 1.12 / 1.13 / 1.14 / 1.19 및 UWIN 직원채용관리

---

## 개요

채용공고 게시 전후로 심사위원 등록, 자기소개서 항목 설정, 필수 제출파일 지정, 지원자 접수 관리까지의 절차를 다룹니다. 각 단계는 독립적으로 진행되나, **심사위원 등록(STEP 5)은 심사단계 저장 이후에만 진행 가능**하므로 순서에 유의하세요.

---

## STEP 5 — 심사위원 등록

### 진입 경로

`1.3 모집부서 관리 → 심사단계 [선택] → 심사과정 관리 팝업(1.7) → 심사단계별 [관리] 버튼 클릭`

또는

`일반행정 > 채용관리 > 직원 > 1.12 심사사정 관리`

---

### 심사위원 등록 절차

1. `1.7 심사과정 관리 팝업` 또는 `1.12 심사사정 관리` 화면에서 해당 심사단계를 저장합니다.
2. 심사단계 저장 후 **[관리]** 버튼이 활성화되면 클릭하여 심사위원 관리 팝업(1.8)을 엽니다.
3. **[추가]** 버튼을 클릭하여 신규 행을 생성합니다.
4. 구분(`내부` / `외부`)을 선택합니다.
5. 구분에 따라 아래 방법으로 정보를 입력합니다.

| 구분 | 등록 방법 | 입력 항목 |
|------|-----------|-----------|
| **내부** | 🔍 돋보기 클릭 → 교직원 검색 팝업에서 선택 | 성명·부서·직급 자동 입력 |
| **외부** | [추가] 후 직접 입력 | 성명 / 부서(소속) / 직급 |

6. **[저장]** 버튼을 클릭하여 심사위원 등록을 완료합니다.

---

### 심사사정 관리 화면(1.12) — 통합 확인

`1.12 심사사정 관리` 화면에서는 모집연도·모집명·모집부서·모집분야별로 심사단계와 심사항목을 통합 조회·관리할 수 있습니다. `1.7 팝업`과 동일한 기능을 제공합니다.

| 기능 버튼 | 설명 |
|-----------|------|
| [추가] | 심사단계 또는 심사항목 추가 |
| [저장] | 추가·수정한 내용 저장 |
| [삭제] | 선택한 단계 또는 항목 삭제 |
| [심사위원] 관리 | 단계별 심사위원 관리 팝업 열기 |
| [심사항목설정] | 해당 심사단계 항목 조회 |

---

### ⚠️ 유의사항

- **[관리] 버튼은 심사단계가 저장된 후에만 표시됩니다.** 단계를 먼저 저장하지 않으면 버튼이 나타나지 않습니다.
- **이미 심사평가가 진행 중인 심사위원은 삭제할 수 없습니다.**
- 심사단계에 심사위원 또는 심사항목이 이미 등록된 경우, 해당 심사단계를 삭제할 수 없습니다.
- 심사항목은 해당 모집의 지원자 평가가 존재하는 경우 삭제할 수 없습니다.
- **심사항목의 배점 합계가 반드시 100점이어야 저장이 가능합니다.**

---

## STEP 6 — 자기소개서 항목 관리

### 진입 경로

`(UWIN) 직원채용관리 > 자기소개서 항목 관리`

---

### 처리 절차

1. 모집부서 및 모집분야를 선택하여 해당 공고의 자기소개서 항목 화면으로 진입합니다.
2. **[기본설정 가져오기]** 버튼을 클릭하면 공통코드(E95)에 등록된 기본 항목으로 자동 세팅됩니다.
3. 항목을 추가하려면 **[추가]** 버튼을 클릭하여 신규 항목을 입력합니다.
4. 항목 수정 후 **[저장]** 버튼을 클릭합니다.
5. 불필요한 항목은 선택 후 **[삭제]** 버튼으로 제거합니다.

---

### ⚠️ 유의사항

- **이미 게시된 채용공고에서는 자기소개서 항목을 추가·수정·삭제할 수 없습니다.** 항목 변경이 필요한 경우 공고 게시 전에 완료하세요.

---

## STEP 7 — 지원 시 필수 접수 파일 지정

### 진입 경로

`(UWIN) 직원채용관리 > 지원 시 필수 접수 파일 지정`

---

### 처리 절차

1. 모집부서 및 모집분야를 선택하여 해당 공고의 제출서류 관리 화면으로 진입합니다.
2. **[기본설정 가져오기]** 버튼을 클릭하면 공통코드(E88)에 등록된 기본 서류 항목으로 자동 세팅됩니다.
3. 항목을 추가하려면 **[추가]** 버튼을 클릭하여 신규 제출서류(첨부파일) 항목을 입력합니다.
4. 항목 입력·수정 후 **[저장]** 버튼을 클릭합니다.
5. 불필요한 항목은 선택 후 **[삭제]** 버튼으로 제거합니다.

---

### ⚠️ 유의사항

- **이미 게시된 채용공고에서는 필수 접수 파일 항목을 추가·수정·삭제할 수 없습니다.** 공고 게시 전에 반드시 설정을 완료하세요.

---

## STEP 8 — 지원자 접수 관리

### 진입 경로

`일반행정 > 채용관리 > 직원 > 채용지원자 관리 (메뉴 1.13)`

---

### 지원자 현황 조회 및 관리 (1.13)

모집연도·모집명·모집부서·모집분야·지원자번호/성명으로 지원자를 검색하고 상세 정보를 관리합니다.

**주요 기능 버튼**

| 버튼 | 기능 |
|------|------|
| [조회] | 조건별 지원자 검색 |
| [추가] | 관리자가 지원자 직접 등록 (1.14 팝업 사용) |
| [삭제] | 체크박스 선택 지원자 삭제 (심사점수 입력 전만 가능) |
| [지원서 출력] | 지원자 정보 리포트 출력 |
| [자기소개서 출력] | 자기소개서 리포트 출력 |
| [공고별 제출서류 일괄다운로드] | 전체 지원자 첨부파일 압축 다운로드 |

**탭별 관리 항목**

| 탭 | 관리 내용 |
|----|-----------|
| 기본 | 성명·생년월일·성별·연락처·이메일·국적·학가구분 등 |
| 학력사항 | 학력구분·학교명·학과명·학위·입학일·졸업일·취득국가 |
| 경력사항 | 기관명·직위명·재직기간·근무년수·담당직무·임상여부 |
| 자격,면허,상,벌 | 자격증·면허·수상·징계 이력 |
| 자기소개서 | 항목별 자소서 내용 (설정된 항목만 표시) |
| 제출서류 | 첨부파일 업로드·다운로드·일괄다운로드 |

---

### 지원자 직접 등록 (1.14 Popup)

오프라인 접수 지원자 등록 또는 시스템 오류 발생 시 관리자가 직접 등록하는 기능입니다.

1. 1.13 화면에서 **[추가]** 버튼을 클릭하여 1.14 팝업을 엽니다.
2. 아래 항목을 입력합니다.

| 입력 항목 | 입력 방법 |
|-----------|-----------|
| 모집명 | 드롭다운 선택 |
| 모집부서 / 모집분야 | 드롭다운 선택 |
| 한글 성명 / 영문 성명 / 한자 성명 | 직접 입력 |
| 생년월일 | 드롭다운 선택 |
| 성별 | 남 / 여 선택 |

3. **[저장]** 클릭 시 시스템이 지원번호를 자동 생성합니다.

---

### 지원불가자 예외 처리 (1.19)

지원 희망자가 시스템상 지원이 불가한 경우, **직접 처리하지 말고 총무인사팀에 문의**하여 예외자 처리를 요청합니다.

---

### ⚠️ 유의사항

- **이미 심사점수가 입력된 지원자는 삭제할 수 없습니다.**
- **진행상태 변경(임시저장 → 제출)은 부득이한 경우에만 사용합니다.** 변경 후 반드시 해당 지원자에게 재작성 가능 상태임을 안내하세요.

---

## FAQ

**Q. 지원자가 입사지원서 제출 후 수정을 요청하면 어떻게 처리하나요?**

`1.13 채용지원자 관리` 화면에서 해당 지원자의 진행상태를 `임시저장` 상태로 변경 후 저장하면 지원자가 재작성할 수 있는 상태가 됩니다. **변경 후 반드시 대상자에게 수정 가능 여부를 안내**하세요.
