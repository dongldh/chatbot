# 임용 및 계약 처리

> **채용 프로세스 STEP 18 ~ 21** | 구분: 신규임용 | 담당: 총무인사팀

---

## 📌 개요

최종합격자가 확정된 이후, 실제 임용이 이루어지기까지의 행정 절차를 다룹니다.  
직무·급여정보 입력 → 채용결과 통보 → 계약서 작성 → 임용(온보딩)의 4단계로 구성됩니다.

| STEP | 단계명 | 담당 |
|------|--------|------|
| STEP 18 | 직무 및 급여정보 세부정보 입력 | 채용부서 |
| STEP 19 | 채용결과 통보 및 계약의뢰 | 채용부서 → 총무인사팀 |
| STEP 20 | 계약서 작성 | 총무인사팀 |
| STEP 21 | 임용 | 채용부서 + 총무인사팀 |

---

## STEP 18. 직무 및 급여정보 세부정보 입력

### 목적
최종합격자의 업무 및 급여(보수) 정보를 시스템에 입력합니다.  
**이 단계에서 입력된 내용은 계약서 작성 시 직접 활용되므로 정확하게 입력해야 합니다.**

### UWIN 시스템 경로
`일반행정 > 직원인사 > 임용관리 > 계약직원신규계약`

### 입력 절차

1. 위 메뉴 경로 접근 후 `조회` 클릭
2. 작성 상태를 확인하고, **임시 사번(숫자 6자리)** 클릭
3. 아래 항목을 순서대로 입력

| 순서 | 항목 | 예시 | 비고 |
|------|------|------|------|
| 1 | 계약기간 | 2026.04.01. ~ 2027.03.31. | |
| 2 | 근무시간 | 09:00 ~ 18:00 | |
| 3 | 담당업무 | 1. XXXXXXX / 2. AAAAAAA | |
| 4 | 담당직무 | 학사코디네이터 등 | |
| 5 | 급여 정보 | 보수 세부 내역 | **유의하여 입력** |
| 6 | 급여 계정 정보 | 급여 지급 계정 | **오입력 방지를 위해 철저 확인 필요** |

4. 입력 완료 후 `저장` 클릭 (임시 저장)
5. 저장 완료 시 생성되는 `신청` 버튼 클릭 → 인사부서 승인 대기 상태로 전환
6. `인쇄` 버튼으로 **자체직 근무(변동) 신고서 PDF 다운로드**
   - 해당 PDF는 STEP 19의 채용결과 통보 공문에 첨부 (별도 수기 결재 불필요)

### 유의사항
- 소속 부서의 **기존 입력 사례를 참고**하여 일관되게 입력
- 인건비 재원이 사업비 등으로 별도 편성된 경우, **반드시 확인 후 계정 입력**

## STEP 19. 채용결과 통보 및 계약의뢰

### 목적
채용 결과(최종합격자)를 인사부서(총무인사팀)에 공식 통보하고 계약서 작성을 의뢰합니다.

### 처리 기한
**임용(채용일) 5일 전까지** 공문 송부 완료 (채용부서 → 총무인사팀)

### 처리 절차

1. **(채용부서)** 최종합격자 확정 자료 정리 후 아래 서류 일체를 포함한 공문 작성
   - 최종합격자 명단
   - 심사 자료
   - 성범죄 조회 결과
   - **자체직 근무(변동) 신고서 (STEP 18에서 출력한 PDF) ← 필수 첨부**
2. **(채용부서 → 총무인사팀)** 공문 발송
3. **(총무인사팀)** 공문 접수 후 계약서 작성 진행
4. **(총무인사팀 → 채용부서)** 계약서 작성 일시 안내
5. **(채용부서)** 안내된 일시에 총무인사팀 방문

### 공문 작성 양식

```
제목: 총무인사팀 계약직원 채용결과 통보 및 계약 요청

본문

1. 관련: (최종 내부결재 문서 번호 및 제목 기입)
2. 관련 문서에 의거하여 계약직원 채용 결과를 통보드립니다.
3. 해당 인원의 계약서 작성을 의뢰하오니 협조하여 주시기 바랍니다.

   가. 성   명:
   나. 담당업무:
       -
       -
   다. 생년월일:
   라. 임용일자:
   마. 급   여:
   바. 급여재원:

붙임 1. 관련 공문 각 1부.
     2. 자체직 근무(변동) 신고서 1부.  끝.
```

## STEP 20. 계약서 작성

### 목적
최종합격자와의 근로계약(또는 임용계약)을 공식 체결합니다.

### 처리 주체
**총무인사팀에서 일괄 진행**

### 절차

1. 총무인사팀이 STEP 19 공문 접수 후 계약서 초안 작성
2. 채용부서 담당자가 총무인사팀 방문하여 계약 내용 최종 확인
3. 계약서 서명·날인 후 체결 완료

### 유의사항
- **계약 조건(계약기간 / 보수 / 담당업무)이 내부결재 문서 및 STEP 18에서 입력한 급여정보와 일치하는지 최종 점검 필수**

## STEP 21. 임용

### 목적
신규 임용자에게 업무를 부여하고 초기 적응(온보딩)을 지원합니다.

### 산출물
- 업무 분장 / 인수인계 기록
- 계정·출입 등 초기 세팅 완료 확인

### 온보딩 체크리스트

#### 1) 업무 부여 (첫 주 내 권장)
- [ ] 담당업무 및 역할 설명 (업무범위, 우선순위)
- [ ] 인수인계 자료 제공 및 담당자 지정
- [ ] 필수 일정 및 보고 라인 공유

#### 2) 권한·계정 세팅
- [ ] `UWIN` 등 업무 시스템 계정 및 권한 확인
- [ ] 사무공간·출입·장비 지급 (해당하는 경우)
- [ ] 메일·메신저·공용폴더 접근 권한 부여

## 🔁 단계별 흐름 요약

```
[STEP 18] 채용부서
  └─ UWIN에서 직무·급여정보 입력 → 자체직 근무(변동) 신고서 PDF 출력
        ↓
[STEP 19] 채용부서 → 총무인사팀
  └─ 임용 5일 전까지 채용결과 통보 + 계약의뢰 공문 발송 (신고서 PDF 필수 첨부)
        ↓
[STEP 20] 총무인사팀
  └─ 계약서 작성 일시 안내 → 채용부서 방문 → 계약 체결
        ↓
[STEP 21] 채용부서 + 총무인사팀
  └─ 업무 부여 + 시스템 계정·권한 세팅 → 온보딩 완료
```
