## 1. 채용공고 등록

**STEP 4** | 메뉴 경로: `일반행정 > 채용관리 > 직원 > 채용공고 관리`

> ⚠️ **잘못 게시된 공고는 접수기간 내 수정 불가합니다.**
> 오류 발견 시 진행상태를 **N**으로 변경한 후 신규 공고를 재등록하세요.
> 모집부서 등록 전에는 삭제 가능합니다.

---

### 1-1. 공고문 템플릿 관리

메뉴 경로: `일반행정 > 채용관리 > 직원 > 채용공고문 템플릿 관리`

공고문 양식을 미리 만들어두는 단계입니다. 템플릿을 먼저 작성해 두면 이후 공고 등록 시 불러와 활용할 수 있습니다.

1. **[추가]** 클릭 → 템플릿명 입력 칸 및 에디터 초기화
2. 템플릿명 입력 (예: `2025년 직원채용 기본양식`)
3. 에디터에 공고문 내용 작성 (지원자격, 제출서류, 전형일정 등)
4. **[저장]** 클릭 → 템플릿 저장 완료

- **수정:** 목록에서 템플릿명 클릭 → 수정 후 **[저장]**
- **삭제:** 체크박스 선택 후 **[삭제]**

> ⚠️ 템플릿을 나중에 수정하더라도 **이미 등록된 채용공고문에는 반영되지 않습니다.**

---

### 1-2. 채용공고 등록

메뉴 경로: `일반행정 > 채용관리 > 직원 > 채용공고 관리`

1. **[추가]** 클릭 → 채용공고 관리 팝업 열림
2. 아래 입력항목 표를 참조하여 기본 정보 입력
3. **[저장]** 클릭 → 공고 목록에 추가됨
4. 채용공고문 **[선택]** 버튼 클릭 → 템플릿 조회 팝업 열림
5. 원하는 템플릿 선택 → **[공고문 템플릿 적용]** → 공고문 관리 팝업으로 이동
6. 공고문 내용 편집 후 **[저장]** → 등록 완료

#### 입력항목 상세

| 항목 | 필수 | 입력 방법 | 예시 / 설명 |
|---|---|---|---|
| 모집명 | ✅ | 직접 입력 | `2025학년도 울산대학교 직원채용 공고` |
| 구분 | ✅ | 드롭다운 | `직원` / `계약직원` |
| 담당부서 | ✅ | 직접 입력 | `정보화개발팀` |
| 학가구분 | ✅ | 드롭다운 | 미선택 시 전체 대상 |
| 모집시작일/시간 | ✅ | 날짜+시간 | `2025.03.05 / 09:00` |
| 모집종료일/시간 | ✅ | 날짜+시간 | `2025.05.05 / 17:00` |
| 진행상태 | ✅ | Y / N | **Y = 채용사이트에 게시됨** |
| 중복지원 가능여부 | — | 체크박스 | 체크 시 같은 공고 내 다른 부서 복수지원 가능 |
| 타부서열람여부 | — | 체크박스 | 체크 시 타부서 담당자 열람 가능 |

#### 진행상태별 채용사이트 표시

| 진행상태 | 날짜 조건 | 사이트 표시 |
|---|---|---|
| Y | 시작 전 | `오픈예정` |
| Y | 접수 중 | `접수 중` |
| Y | 종료 후 | `마감` |
| N | 무관 | 미게시 |

### 1-3. 공고문 내용 작성 (팝업)

공고문 관리 팝업에서 최종 공고문을 완성합니다.

| 기능 | 설명 |
|---|---|
| 에디터 | 서식 있는 텍스트 편집 (HTML 직접 편집 탭 제공) |
| 첨부파일 | 지원서 양식 등 파일 업로드 |
| **[저장]** | 작성한 공고문 저장 |

> 💡 **채용공고문 생성기 활용 권장**
> 외부 채용공고문 생성기([채용공고문 생성기 바로가기](https://recruitment-generator.vercel.app/))에서 공고문을 보기 좋게 작성한 뒤, 생성된 HTML을 UWIN 공고문 편집기의 **HTML 모드**에 붙여넣어 등록합니다.
> 기본 흐름: 공고 생성 → HTML 생성 → 편집기에 붙여넣기(HTML 모드)

## 2. 직무·급여정보 입력

**STEP 18** | 메뉴 경로: `일반행정 > 직원인사 > 임용관리 > 계약직원신규계약`

> ⭐ **이 단계에서 입력한 정보는 계약서 작성 시 직접 활용됩니다. 정확하게 입력하세요.**

최종합격자의 업무 및 급여(보수) 정보를 입력하는 단계입니다.

---

### 입력 절차

1. 메뉴 접근 후 **[조회]** 클릭
2. 목록에서 작성 상태를 확인하고 **임시 사번(숫자 6자리)** 클릭
3. 아래 항목 표를 참조하여 세부 정보 입력
4. **[저장]** 클릭 → 임시 저장
5. 저장 완료 후 생성된 **[신청]** 버튼 클릭 → 인사부서 승인 대기 상태로 전환
6. **[인쇄]** 클릭 → 자체직 근무(변동) 신고서 PDF 다운로드 → `<채용결과 통보 및 계약의뢰 공문>`에 첨부

#### 입력항목 상세

| 순서 | 항목 | 예시 / 설명 | 비고 |
|---|---|---|---|
| 1 | 계약기간 | `2026.04.01. ~ 2027.03.31.` | |
| 2 | 근무시간 | `09:00 ~ 18:00` | |
| 3 | 담당업무 | 1. XXXXXXX / 2. AAAAAAA / 3. BBBBBBBB | |
| 4 | 담당직무 | `학사코디네이터` 등 | |
| 5 | 급여 정보 | 급여 정보 유의하여 입력 | |
| 6 | 급여 계정 정보 | 급여 지급 계정 입력 | **오입력 방지를 위해 철저히 확인** |

> ⚠️ **유의사항**
> - 소속 부서의 **기존 입력 사례를 참고**하여 일관되게 입력하세요.
> - **인건비 재원이 사업비 등으로 별도 편성된 경우**, 반드시 확인 후 계정 정보를 입력하세요.
> - **[인쇄]** 로 출력한 PDF 파일만 공문에 첨부하면 되며, 별도 수기 결재는 불필요합니다.

## 3. 접수파일 지정

**STEP 7** | 메뉴 경로: `(UWIN) 직원채용관리 > 지원 시 필수 접수 파일 지정`

채용공고의 **모집부서·모집분야별**로 지원자가 반드시 제출해야 하는 첨부파일 항목을 관리합니다.

---

### 처리 절차

1. 메뉴 진입 후 대상 채용공고 및 모집부서·분야 선택
2. **[기본설정 가져오기]** 클릭 → 공통코드(E88) 기준 항목 자동 세팅
3. 항목 추가가 필요한 경우 **[추가]** 클릭 → 제출서류(첨부파일) 항목 입력
4. 불필요한 항목은 선택 후 **[삭제]**
5. **[저장]** 클릭 → 입력 또는 수정 내용 저장 완료

#### 주요 기능 요약

| 기능 | 설명 |
|---|---|
| 기본설정 가져오기 | 공통코드(E88) 기준 항목을 자동으로 불러와 세팅 |
| 추가 | 신규 첨부파일 항목 추가 |
| 저장 | 입력·수정 내용 저장 |
| 삭제 | 선택한 항목 삭제 |

> ⚠️ **이미 게시된 채용공고에서는 항목 추가·수정·삭제가 불가합니다.**
> 게시 전에 반드시 확인하고 저장하세요.

## 4. 자기소개서 항목 관리

**STEP 6** | 메뉴 경로: `(UWIN) 직원채용관리 > 자기소개서 항목 관리`

채용공고의 **모집부서·모집분야별**로 지원자에게 요구할 자기소개서 항목을 관리합니다.

---

### 처리 절차

1. 메뉴 진입 후 대상 채용공고 및 모집부서·분야 선택
2. **[기본설정 가져오기]** 클릭 → 공통코드(E95) 기준 항목 자동 세팅
3. 항목 추가가 필요한 경우 **[추가]** 클릭 → 자기소개서 항목 내용 입력
4. 불필요한 항목은 선택 후 **[삭제]**
5. **[저장]** 클릭 → 입력 또는 수정 내용 저장 완료

#### 주요 기능 요약

| 기능 | 설명 |
|---|---|
| 기본설정 가져오기 | 공통코드(E95) 기준 항목을 자동으로 불러와 세팅 |
| 추가 | 신규 자기소개서 항목 추가 |
| 저장 | 입력·수정 내용 저장 |
| 삭제 | 선택한 항목 삭제 |

> ⚠️ **이미 게시된 채용공고에서는 자기소개서 항목 추가·수정·삭제가 불가합니다.**
> 게시 전에 반드시 최종 확인하고 저장하세요.
