## STEP 15 — 성범죄 사실 조회

### 개요

최종합격자를 대상으로 **임용일 이전**에 성범죄 경력 조회를 완료해야 합니다.

- **의뢰기관**: 울산남부경찰서
- **산출물**: 조회 요청 공문 및 대상자 명단, 동의서(원본 보관), 회신 결과

### 처리 절차

1. 최종합격자에게 조회 목적을 안내하고 **동의서를 수신(원본 보관)**한다.
2. 대상자 명단을 첨부하여 울산남부경찰서로 **외부발신 공문을 발송**한다.
3. 결과 회신을 수신한다. (통상 **2~3일 내** 회신)
4. 결격 여부를 확인하여 총무인사팀에 통보한다.

> **⚠️ 성범죄 조회는 반드시 임용일 이전에 완료해야 합니다.** 결과 수신 전 임용이 진행되지 않도록 일정을 관리하세요.

## STEP 16 — 최종합격자 확정 및 UWIN 이관

### 시스템 경로

`일반행정 > 채용관리 > 직원 > 최종합격자 관리` (메뉴 1.22)

### 처리 절차

1. **[조회]** 버튼을 클릭하여 이전 단계(1.21)에서 합격 처리된 지원자를 불러온다.
2. 각 지원자의 결과를 `합격` / `불합격` / `예비순위` 중 하나로 선택하여 최종 확정한다.
3. **[저장]** 버튼을 클릭하여 임시 저장한다.
4. **[확정]** 버튼을 클릭하여 확정일자를 등록한다.
   - **확정 이후에는 합격 여부 수정이 불가합니다.**
   - 수정이 필요한 경우 **[확정취소]** 버튼으로 확정을 해제한 후 재처리하세요.
5. 합격 처리된 지원자의 체크박스를 선택한다. (합격자만 체크박스 활성화)
6. **[합격자이관]** 버튼을 클릭하여 **교직원정보 시스템으로 데이터를 이관**한다.
7. 필요 시 **[모집현황 출력]** 버튼으로 모집현황 리포트를 출력한다.

> **⚠️ [합격자이관] 실행 전, 교직원정보에 입력된 데이터(성명, 생년월일, 부서 등)의 정확성을 반드시 재확인하세요.** 이관 후 오류 정정은 별도 수정 절차가 필요합니다.

## STEP 17 — 합격자 발표 및 통보

### 시스템 경로

`일반행정 > 채용관리 > 직원 > 합격자발표 관리` (메뉴 1.23)

### 발송 전 체크리스트

아래 항목을 모두 확인한 후 실제 발송을 진행하세요.

- [ ] 모집연도·모집명 조회 완료
- [ ] 확정일자가 등록되어 있음 (STEP 16에서 확정 완료)
- [ ] 합격·불합격 메시지 문구 입력 완료
- [ ] 테스트 발송으로 문구 최종 확인
- [ ] **[실제발송]** 클릭

### 처리 절차

1. 모집연도·모집명을 선택한 후 **[조회]** 버튼을 클릭한다.
2. 확정일자 및 합격자발표 기간(시작~종료 일시)을 입력한 후 **[저장]** 버튼을 클릭한다.
3. 합격 메시지의 **[메일]** 버튼을 클릭하여 에디터에서 합격자 메일 문구를 작성한다.
4. 불합격 메시지의 **[메일]** 버튼을 클릭하여 불합격자 메일 문구를 작성한다.
5. **[테스트 — 메일]** 버튼을 클릭하여 로그인 사용자 본인에게 테스트 메일을 발송하고 문구를 최종 확인한다.
6. 문구 확인 완료 후 **[실제발송 — 메일]** 버튼을 클릭하여 합격자·불합격자에게 실제 발송한다.

> **⚠️ SMS 발송은 현재 발신번호 제약으로 실제 사용이 불가합니다. 반드시 메일로만 발송하세요.**

> 📌 합격·불합격 메시지 문구가 입력되면 해당 체크박스는 자동으로 체크된 후 비활성화 처리됩니다.

## 메일 문구 예시

### 합격 안내 메일

- **제목**: `[울산대학교] 직원 채용 최종합격 안내(임용 예정)`
- **본문**:

```
안녕하세요. 울산대학교 XXXXX팀입니다.
[채용명] 최종합격을 안내드립니다.

임용(출근) 관련 주요 안내사항은 아래 안내자료에서 확인해 주세요.
- 기본 안내자료: 울산대학교 신규임용 직원 안내자료 (링크 삽입)
- 추가 안내사항: (부서별 특정 내용 기입)

세부 안내(계약서 작성 일정, 제출서류 등)는 확정되는 대로 별도 안내드리겠습니다.
감사합니다.
```

### 불합격 안내 메일

- **제목**: `[울산대학교] 직원 채용 결과 안내`
- **본문**:

```
안녕하세요. 울산대학교 XXXXX팀입니다.
[채용명] 전형 결과를 안내드립니다.

이번 전형에서는 안타깝게도 선발되지 않으셨습니다.
지원해 주셔서 감사합니다.
```

## FAQ

**Q1. 실수로 불합격 처리 후 확정 버튼을 눌렀어요.**
→ **[확정취소]** 버튼을 클릭하면 확정이 해제됩니다. 결과를 수정한 후 다시 **[확정]** 버튼을 클릭하세요.

**Q2. 합격 통보 메일이 발송되지 않아요.**
→ 확정일자가 등록되어 있어야 발송이 가능합니다. 메뉴 1.22(`최종합격자 관리`)에서 확정일자를 먼저 등록하세요.

**Q3. SMS 합격 통보가 되지 않아요.**
→ 현재 발신번호 제약으로 SMS 실사용이 불가합니다. **메일로만 발송**하세요.

**Q4. 공고를 게시한 후 모집부서를 추가하고 싶어요.**
→ 게시 후에는 추가·수정·삭제가 불가합니다. 공고의 시작일/시간을 미래로 변경하거나, 진행상태를 `N`으로 설정하고 신규 공고를 등록하세요.

**Q5. 지원자가 임시저장 상태예요.**
→ `1.13 채용지원자 관리`에서 해당 지원자의 진행상태를 `제출`로 변경할 수 있습니다. **부득이한 경우에만 사용**하세요.

**Q6. 계약직 교직원이 지원을 시도할 경우 어떻게 되나요?**
→ 계약직 조건에 따라 지원이 제한됩니다. 예외 허용이 필요한 경우 `1.19 지원불가자예외 관리`에서 해당 교직원을 등록하세요.

**Q7. 엑셀로 심사점수를 업로드했는데 오류가 납니다.**
→ 반드시 시스템에서 **[엑셀 다운로드]로 받은 서식 파일**을 사용해야 합니다. 자체 제작한 파일은 업로드가 불가합니다.
