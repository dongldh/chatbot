## 1. 업무 개요

### 근태란

출근과 퇴근에 관련된 모든 사항을 말하며, 기본적으로 **UWIN 신청 후 부서장 결재**를 통해 이뤄진다. 근태 업무는 근로기준법, 단체협약, 취업규칙에 따라 대학 근태제도를 기획·운영·관리하는 역할을 한다.

### 근태 접수 처리 원칙

근태 접수 시 아래 세 가지 원칙을 반드시 확인한다.

**1. 전결 기준 준수 여부**

| 대상 | 결재 기준 |
|------|-----------|
| 팀(실)원 근태 | 팀(실)장 전결 → 총무인사팀 제출 |
| 팀(실)장 근태 | 부서장 전결 → 총무인사팀 제출 |
| 총무처 소속 팀장 근태 | 총무인사팀장 전결로 종료 (중복 방지) |

**2. 증빙 자료 첨부 여부**
- 자연발생적 휴가(연가, 하계휴가 등): 증빙 불필요
- 사유가 필요한 휴가(병가, 출산휴가 등): 증빙 필수

**3. 특수 근태 보고 여부**
- 병가, 청가 등 특수한 근태는 **총무처장 결재 필요** → 팀장에게 먼저 보고 후 결재 상신

> **참고:** 연가는 소속 부서에서 결재 완료 후 총무인사팀으로 별도 제출하지 않음. 위임전결규정에 따라 결재권이 부서에 위임되어 있기 때문임.

### 근태 접수 방법 (UWIN 경로)

- 경로: `UWIN 교직원포탈 > 전자결재 > 부서수신함`

1. 휴가원 더블 클릭
2. (휴가원 팝업) 접수자 전결 클릭 / 결재자 지정
3. (기록물철 선택 팝업) 근태관계철 더블클릭 후 확인 클릭
4. (UWIN 비밀번호 입력 팝업) 비밀번호 입력 후 확인 클릭
5. 완료

## 2. 시기별 정기 처리 업무

### 매일 해야 하는 업무

1. **일반 근태 접수 및 결재** (`UWIN 교직원포탈 > 전자결재 > 부서수신함`)
2. **선택적 근로시간제 연장근로 문서 접수** (`그룹웨어 > 전자결재 > 부서수신함`)
   - 확인 필요 사항: 시작·종료시간, KT 출퇴근시간, 연장근로 사유, 부서장 결재 여부
   - **사유가 불명확하거나 팀장 전결로 제출된 경우 반송 처리**

### 매월 1일에 해야 하는 업무

1. **인사발령 처리:** 1일자 발령 건 UWIN 등록 여부 확인 (인사담당자)

2. **호봉 승급 처리:**
   - 메뉴: `일반행정 > 직원인사 > 임용관리 > 호봉및근속년수처리[CAD05U]`

3. **시점별 현황 생성:**
   - 메뉴: `일반행정 > 직원인사 > 통계관리 > 시점별현황생성[CAH20U]`
   - 해당 월 1일자로 생성 버튼 클릭
   - **이 작업을 하지 않으면 기본근로시간 등록, 근태시간 확인 불가**

4. **기본근로시간 생성:**
   - 메뉴: `일반행정 > 직원인사 > 근로관리 > 기본근로시간관리(부서별)[CAK05U]`
   - **전체 선택 후 의과대학 3개 팀 선택 해제** (의대는 출퇴근 시각이 달라 직접 입력)
   - 아래 기준에 따라 요일별로 수정 후 저장

   **[학기 중]**

   | 요일 | 근무시간 | 휴게시간1 | 단축시간 | 휴게시간2 |
   |------|----------|-----------|----------|-----------|
   | 월/화/수/목 | 09:00~18:00 | 12:00~13:00 | 00:00~00:00 | 18:00~18:30 |
   | 금 | 09:00~18:00 | 12:00~13:00 | 17:00~18:00 | 18:00~18:30 |

   **[방학 중]**

   | 요일 | 근무시간 | 휴게시간1 | 단축시간 | 휴게시간2 |
   |------|----------|-----------|----------|-----------|
   | 월/화/수/목/금 | 09:00~18:00 | 12:00~13:00 | 15:00~18:00 | 18:00~18:30 |

### 매월 15일 경 해야 하는 업무 (선근제)

1. 각 부서에 **월단위 보상휴가 시행 사항 보고 문서 제출 요청**
2. 접수 경로: `그룹웨어 > 전자결재 > 부서수신함`
3. 확인 메뉴: `일반행정 > 직원인사 > 근로관리 > 부서근태관리(월간)[CAK08U]`

### 매년 해야 하는 업무

- 연가 촉진
- 연가 보상
- 전년도 잔여 연차휴가 적립 처리

## 3. 선택적 근로시간제

### 개요

대학의 근로시간 제도의 근간이 되는 제도로, 연장근로에 대한 보상을 노사 간 합의를 통해 보상휴가 방식으로 운영한다.

### 선근제 부서 등록 절차

1. 부서 요청 접수
2. 공문 접수
3. 협약서 조정
4. 전산 등록

**전산 등록 방법:**
- 메뉴: `일반행정 > 직원인사 > 기준정보관리 > 선택적근무부서등록[CAB13U]`
  1. 부서 조회
  2. 시작일 입력
  3. 종료일 입력 (`99991231`)
  - **종료일을 입력하지 않으면 현재 부서로 확인되지 않으므로 반드시 입력**

### 주요 UWIN 메뉴 설명

| 메뉴명 | 메뉴 코드 | 설명 |
|--------|-----------|------|
| 기본근로시간관리(부서별) | `[CAK05U]` | 월별 전체 부서 근로시간 관리 |
| 기본근로시간관리(개인별) | — | 개인별 근로시간 조정 (로직으로 해결 안 되는 경우) |
| 개인별근태관리 | `[CAK02U]` | 개인 근로 현황 캘린더 확인, 연장근로 신청 |
| 부서근태관리(일간) | `[CAK07U]` | 팀(실)장이 일별 연장근로·대체휴무 승인 |
| 부서근태관리(월간) | `[CAK08U]` | 월 1회 부서별 근로시간 집계 및 총무인사팀 제출 |
| 근로결재관리 | `[CAK09U]` | 전자결재 취소 처리 (추가 야간근로 인원 발생 등) |
| 보상휴가현황 | `[CAK10U]` | 기간별 직원 보상휴가 발생 현황 확인 |
| 근태현황 | `[CAK11P]` | 선근제 통계자료 및 엑셀 다운로드 |

### 방학 중 대체휴무 사용 확인 방법

방학 중 단축근무로 인해 대체휴무 사용이 가능한 경우가 드물게 발생한다.

- 메뉴: `일반행정 > 직원인사 > 근로관리 > 개인별근태관리[CAK02U]`
- 계산 방식: 연장근로(A) - 단축근무(B) = 사용 가능한 대체휴무 시간

> **예시:** 연장근로 10시간 - 단축근무 5시간 = 대체휴무 5시간 사용 가능

### 방학 중 근태 신청 방침 (문의별 답변 기준)

| 문의 내용 | 가능 여부 | 사유 |
|-----------|-----------|------|
| 반일연가 + 외출 + 단축근무 조합으로 연가 효과 | **불가** | 외출은 근무지 복귀를 전제로 하므로, 근무지 미복귀 시 신청 불가 |
| 지각 + 조퇴 + 단축근무 조합으로 연가 효과 | **불가** | 지각은 출근을 전제로 사용하는 근태 |
| 조퇴 + 단축근무 조합으로 반일연가 효과 | **불가** | 조퇴 종료 시각은 근무 종료 시각인 18:00로 신청해야 함 |
| 외출(13~15시) + 단축근무 조합 | **일부 가능** | 15:00 복귀 후 퇴근 기록 시 인정 가능 |

### 52시간 초과자 처리 원칙

- 월 총 허용 근로시간: `=월 총 일수 / 7 * 52시간` (예: 31일 기준 약 230시간)
- 초과 발생 시 전월로 이전 처리 가능하나, **출입 기록 등 대조 시 문제가 없는 범위** 내에서 처리
- **대승적 차원에서 일부는 버림 처리**하는 것이 추후 감사 등에 안전

## 4. 연장근로 및 보상휴가

### 보상휴가 기본 원칙

- 대학은 연장근로에 대해 보상휴가로 수당 지급을 갈음할 수 있음
- **보상휴가는 유효기간(발생 후 6~11개월) 내에 전부 사용하거나 수당으로 보상해야 함**
- 수당으로 보상한 경우 반드시 기록을 남겨야 함

### 보상 정산 처리 절차

1. **보상휴가 잔여시간 확인**
   - 전체 발생일수: `일반행정 > 직원인사 > 근로관리 > 보상휴가현황[CAK10U]`
   - 전체 사용일수: `일반행정 > 직원인사 > 복무관리 > 휴가신청조회[CAC10U]`

2. **미사용자 보상금액 산정**
   - 메뉴: `일반행정 > 직원인사 > 근로관리 > 근태현황[CAK11P]` → `6. 연장근로수당발생현황` → 시작월 선택 후 엑셀 버튼 클릭

3. **연장근로수당 신청서 접수**

4. **노동조합 사전 통보**

5. **보상 후 기록 남기기**
   - 메뉴: `일반행정 > 직원인사 > 복무관리 > 휴가신청[CAC09U]`
   - 휴가구분: '보상휴가 수당지급건' 선택 → 차감 필요시간 입력 → 비고에 '연장근로수당 지급 완
