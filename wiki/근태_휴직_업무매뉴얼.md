## 1. 업무 개요

### 근태란

출근과 퇴근에 관련된 모든 사항을 말한다. 기본적으로 **UWIN에서 신청 후 부서장 결재**를 통해 처리된다. 총무인사팀은 근로기준법, 단체협약, 취업규칙에 따라 대학의 근태 제도를 기획·운영·관리한다.

### 근태 처리 시 공통 원칙

1. **전결 기준 준수 여부 확인**
   - 팀(실)원 근태: 팀(실)장 전결 후 총무인사팀 제출
   - 팀(실)장 근태: 부서장 전결 후 총무인사팀 제출
   - 총무처 소속 팀장 근태: 총무인사팀장 전결로 종료 (중복 방지)
2. **증빙 자료 첨부 여부 확인**
   - 자연발생적 휴가(연가, 하계휴가 등): 증빙 불필요
   - 사유 필요 휴가(병가, 출산휴가 등): 증빙 서류 필수
3. **특수 근태 보고 여부 판단**
   - 병가, 청가 등 특수 근태의 경우 팀장에게 보고 후 총무처장 결재 상신
4. **연가 결재 경로 참고**
   - 연가는 위임전결규정에 따라 소속 부서에서 결재 완료 후 총무인사팀으로 전달되지 않음

## 2. 시기별 정기 처리 업무

### 매일 해야 하는 업무

#### 근태 문서 접수

경로: `UWIN 교직원포털 > 전자결재 > 부서수신함`

1. 부서수신함에서 휴가원 더블 클릭
2. (휴가원 팝업) 접수자 전결 클릭 / 결재자 지정
3. (기록물철 선택 팝업) **근태관계철** 더블클릭 후 확인 클릭
4. (UWIN 비밀번호 입력 팝업) 비밀번호 입력 후 확인 클릭
5. 완료

---

### 매월 1일에 해야 하는 업무

**1. 인사발령 처리 확인**
- 1일자 발령 처리 건이 UWIN에 정상 등록되었는지 확인

**2. 호봉 승급 처리**
- 메뉴: `일반행정 > 직원인사 > 임용관리 > 호봉및근속년수처리[CAD05U]`

**3. 시점별 현황 생성**
- 메뉴: `일반행정 > 직원인사 > 통계관리 > 시점별현황생성[CAH20U]`
- 해당 월 1일자 선택 후 생성 버튼 클릭
- **이 작업을 하지 않으면 기본근로시간 등록, 근태시간 등이 정상 확인되지 않음**

**4. 기본근로시간 생성**
- 메뉴: `일반행정 > 직원인사 > 근로관리 > 기본근로시간관리(부서별)[CAK05U]`
- **전체 선택 후 의과대학 3개 팀은 선택 해제** (의대는 출퇴근 시각이 달라 별도 직접 입력)

| 기간 | 요일 | 근무시간 | 휴게시간1 | 단축시간 | 휴게시간2 |
|------|------|----------|-----------|----------|-----------|
| 학기 중 | 월~목 | 09:00~18:00 | 12:00~13:00 | 00:00~00:00 | 18:00~18:30 |
| 학기 중 | 금 | 09:00~18:00 | 12:00~13:00 | 17:00~18:00 | 18:00~18:30 |
| 방학 중 | 월~금 | 09:00~18:00 | 12:00~13:00 | 15:00~18:00 | 18:00~18:30 |

## 3. 선택적 근로시간제

### 개요

대학의 근로시간 제도의 근간이 되는 제도로, 연장근로에 대한 보상 방식을 노사 간 합의를 통해 운영한다.

### 선근제 부서 등록 절차

1. 부서 요청 접수
2. 공문 접수
3. 협약서 내용 조정
4. 전산 등록

**전산 등록 방법:**
- 메뉴: `일반행정 > 직원인사 > 기준정보관리 > 선택적근무부서등록[CAB13U]`
  1. 부서 조회
  2. 시작일 입력
  3. **종료일 입력 (99991231) — 종료일 미입력 시 현재 부서로 확인되지 않음**
  4. 저장

### 선근제 관련 매일 업무

**선택적 근로시간제 문서 접수:**

경로: `그룹웨어 > 전자결재 > 부서수신함`
사용자 활용 메뉴: `일반행정 > 직원인사 > 근로관리 > 부서근태관리(일간)[CAK07U]`

**연장근로 확인 필수 사항:**

1. **시작·종료시간**: 연장근로로 인정받을 시간
2. **KT 출퇴근 시간**: 출퇴근 단말기에 기록된 시간
3. **내용(사유)**: 사유가 불명확하거나 미기재 시 반송 처리
4. **부서장 결재 여부**: 팀장 전결로 접수된 경우 부서장 전결로 재요청 (지침상 부서장 결재 필수)

> 대체휴무 문서: 특별히 확인할 사항 없음

### UWIN 주요 메뉴 설명

| 메뉴명 | 경로 | 용도 |
|--------|------|------|
| 기본근로시간관리(부서별) | `[CAK05U]` | 월별 전체 부서 근로시간 관리 |
| 기본근로시간관리(개인별) | — | 개인별 근로시간 별도 조정 |
| 개인별근태관리 | `[CAK02U]` | 개인 근로현황 캘린더 확인, 연장근로 신청, 미기록자 입력 |
| 부서근태관리(일간) | `[CAK07U]` | 팀(실)장의 일별 연장근로·대체휴무 승인 |
| 부서근태관리(월간) | `[CAK08U]` | 월 1회 부서별 근로시간 집계 및 총무인사팀 통보 |
| 근로결재관리 | `[CAK09U]` | 부서 요청에 따른 근로결재 반려 및 월별 근로시간 재생성 |
| 보상휴가현황 | `[CAK10U]` | 보상휴가 발생 현황 엑셀 다운로드 (수당 지급 시 검증 자료) |
| 근태현황 | `[CAK11P]` | 선근제 관련 통계 및 엑셀 다운로드 |

## 4. 근태 종류별 상세 처리

### 연가

#### 발생 체계

- **일반직원 / 무기계약직**: 회계연도 기준 발생
- **계약직원**: 입사일자 기준 발생

#### 연차휴가 시행 관리

메뉴: `일반행정 > 직원인사 > 복무관리 > 휴가신청조회[CAC10U]`

1. 조회 일자 입력
2. 조회 후 내용 드래그 복사 → 엑셀에 값만 붙여넣기

#### 휴가 취소 요청 처리

**전자결재가 완료된 건은 임의로 취소 불가**

1. 전화 문의 시 취소 희망 일자를 **이메일로 먼저 받을 것** (근거 확보)
2. 메뉴: `일반행정 > 직원인사 > 복무관리 > 휴가신청[CAC09U]`
   1. 휴가 시작일 클릭
   2. 이름 또는 사번으로 조회
   3. 해당 건 조회
   4. **반려(결재완료된전자결재취소)** 버튼 클릭
   5. 삭제 버튼 클릭 → 완료

#### 연가 적립

**정규직원 / 무기계약직원**
- 단체협약에 따라 미사용 연차휴가 중 연간 최대 2일 적립, 유효기간 3년
- 메뉴: `일반행정 > 직원인사 > 복무관리 > 적립연가일괄생성[CAC26U]`

**계약직원**
- 1년차 계약 종료 후 재계약 시 잔여 연가 자동 적립 (시스템 개발 완료)
- **적립 생성 전, 연가 신청·결재진행 중 상태인 건을 먼저 최종 처리할 것**

#### 학년도 말 근태 점검

- 연간 '작성' 상태로 방치된 미결재 휴가 신청 건을 정리해야 연가 보상 일수 계산 오류 방지
- 메뉴: `일반행정 > 직원인사 > 복무관리 > 휴가신청조회[CAC10U]`
- 조회 기간: 해당 학년도 전체 (예: 2023.03.01.~2024.02.29.)
- 사용 여부를 확인하여 미사용 건은 반려 또는 삭제 처리

#### 연가촉진

- **대상**: 일반직원, 무기계약직원
- **근거**: 근로기준법 제61조

**진행 절차:**

1. **연차 사용 종료 6개월 전** (통상 9월 초): 잔여 휴가 사용 촉구 통지문 서면 발송
   - 엑셀로 잔여 연가 현황 다운로드 후 한글 메일머지 기능으로 개인별 출력
   - **통지문은 반드시 서면(종이)으로 발송**해야 법적 요건 충족
   - 부서별 봉투에 담아 배포 (이름만 보이게 접어 스테이플러 고정)
2. **통지 후 10일 이내**: 직원이 사용 시기를 확정하여 서면으로 통보
3. **미통보 시 연차 사용 종료 2개월 전**: 사용자가 사용 시기를 지정하여 서면 통보
   - 최근 수년간 일자 강제 지정은 시행하지 않음 (단체협약상 10일 보상·2일 적립으로 처리)
   - **단, 원칙적으로 12일 초과 잔여 연가에 대해서는 일자 지정 및 노무수령
