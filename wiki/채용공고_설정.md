## STEP 4. 채용공고 등록

### 진입 경로

`일반행정 > 채용관리 > 직원 > 채용공고 관리`

> ⚠️ **잘못 게시된 공고는 접수기간 내 수정 불가합니다.** 오류 발생 시 진행상태를 **N**으로 변경한 후 신규 공고를 등록하세요.

---

### 4-1. 채용공고문 템플릿 관리 (메뉴 1.1)

공고문 양식을 미리 작성해 두는 단계입니다.

**진입 경로:** `일반행정 > 채용관리 > 직원 > 채용공고문 템플릿 관리`

1. **[추가]** 클릭 → 템플릿명 입력 칸 및 에디터 초기화
2. 템플릿명 입력 (예: `2025년 직원채용 기본양식`)
3. 에디터에 공고문 내용 작성 (지원자격, 제출서류, 전형일정 등)
4. **[저장]** 클릭 → 템플릿 저장 완료

- **수정:** 목록에서 템플릿명 클릭 → 내용 수정 후 [저장]
- **삭제:** 체크박스 선택 후 [삭제]

> ⚠️ **템플릿을 나중에 수정해도 이미 등록된 채용공고문에는 반영되지 않습니다.**

---

### 4-2. 채용공고 등록 (메뉴 1.2)

실제 채용공고를 생성하는 단계입니다.

1. **[추가]** 클릭 → 채용공고 관리 팝업(1.6) 열림
2. 아래 입력항목 표를 참고하여 기본 정보 입력
3. **[저장]** 클릭 → 공고 목록에 추가됨
4. 채용공고문 **[선택]** 버튼 클릭 → 템플릿 조회 팝업(1.4) 열림
5. 템플릿 선택 → **[공고문 템플릿 적용]** → 공고문 관리 팝업(1.5)으로 이동
6. 공고문 내용 최종 편집 후 **[저장]**

**입력항목 상세**

| 항목 | 필수 | 입력 방법 | 예시/설명 |
|------|------|-----------|-----------|
| 모집명 | ✅ | 직접 입력 | `2025학년도 울산대학교 직원채용 공고` |
| 구분 | ✅ | 드롭다운 | `직원` / `계약직원` |
| 담당부서 | ✅ | 직접 입력 | `정보화개발팀` |
| 학가구분 | ✅ | 드롭다운 | 미선택 시 전체 대상 |
| 모집시작일/시간 | ✅ | 날짜+시간 | `2025.03.05 / 09:00` |
| 모집종료일/시간 | ✅ | 날짜+시간 | `2025.05.05 / 17:00` |
| 진행상태 | ✅ | Y/N | **Y = 채용사이트 게시됨** |
| 중복지원 가능여부 | - | 체크박스 | 체크 시 동일 공고 내 다른 부서 복수지원 허용 |
| 타부서열람여부 | - | 체크박스 | 체크 시 타부서 담당자 열람 허용 |

**진행상태별 채용사이트 표시**

| 진행상태 | 날짜 조건 | 사이트 표시 |
|----------|-----------|-------------|
| Y | 시작 전 | `오픈예정` |
| Y | 접수 중 | `접수 중` |
| Y | 종료 후 | `마감` |
| N | 무관 | 미게시 |

### 4-3. 채용공고문 작성 (메뉴 1.5 Popup)

공고문 내용을 최종 완성하는 단계입니다.

| 기능 | 설명 |
|------|------|
| 에디터 | 서식 있는 텍스트 편집 (HTML 직접 편집 탭 제공) |
| 첨부파일 | 지원서 양식 등 첨부파일 업로드 |
| [저장] | 작성한 공고문 저장 |

> 💡 **채용공고문 생성기 활용 안내**
> 외부 채용공고문 생성기([https://recruitment-generator.vercel.app/](https://recruitment-generator.vercel.app/))에서 공고문을 작성한 뒤 생성된 HTML을 UWIN 공고문 편집기의 **HTML 모드**에 붙여넣어 등록할 수 있습니다.
> 기본 흐름: **공고 생성 → HTML 생성 → 공고문 편집기에 붙여넣기**

## STEP 5. 심사위원 등록

### 진입 경로

`1.3 모집부서 관리 → 심사단계 [선택] → 심사과정 관리 팝업(1.7) → 심사단계별 [관리] 버튼 클릭`

> ⚠️ **[관리] 버튼은 심사단계가 저장된 후에만 표시됩니다.** 심사단계를 먼저 저장해야 합니다.

---

### 5-1. 심사위원 등록 방법 (메뉴 1.8 Popup)

**내부 vs 외부 심사위원 등록 방법**

| 구분 | 등록 방법 | 입력 항목 |
|------|-----------|-----------|
| **내부** | 🔍 돋보기 → 교직원 검색 팝업 → 선택 | 자동 입력 |
| **외부** | [추가] → 직접 입력 | 성명 / 부서(소속) / 직급 |

**처리 순서**

1. **[추가]** 클릭 → 신규 행 생성
2. 구분 선택: `내부` 또는 `외부`
3. 내부 위원: 🔍 클릭 → 교직원 검색 후 선택 (성명·부서·직급 자동 입력)
4. 외부 위원: 성명·부서(소속)·직급 직접 입력
5. **[저장]** 클릭 → 심사위원 등록 완료

> ⚠️ **이미 심사평가가 진행 중인 심사위원은 삭제할 수 없습니다.**

---

### 5-2. 심사 관련 주요 제약사항

1. 심사위원 [관리] 버튼은 **심사단계가 저장된 후**에만 표시됩니다.
2. 심사단계는 **심사위원 또는 심사항목이 이미 등록된 경우 삭제 불가**합니다.
3. 심사항목은 **해당 모집의 지원자 평가가 존재하는 경우 삭제 불가**합니다.
4. **심사항목의 배점 합계가 총 100점이 되어야** 저장할 수 있습니다.

---

### 5-3. 심사사정 관리 (메뉴 1.12) — 단계별 통합 확인

모집연도/모집명/모집부서/모집분야별로 심사단계와 심사항목을 통합 확인하고 관리하는 화면입니다. `1.7 팝업`과 동일한 기능을 제공합니다.

| 기능 | 설명 |
|------|------|
| [추가] | 심사단계 또는 심사항목 추가 |
| [저장] | 추가·수정한 내용 저장 |
| [삭제] | 선택한 단계 또는 항목 삭제 |
| [심사위원] 관리 | 단계별 심사위원 관리 팝업 |
| [심사항목설정] 선택 | 해당 심사단계 항목 조회 |

## STEP 6. 자기소개서 항목 관리

### 진입 경로

`(UWIN) 직원채용관리 > 자기소개서 항목 관리`

---

### 화면 개요

관리자가 채용공고의 **모집부서, 모집분야**별로 자기소개서 항목을 설정합니다.

### 처리 순서

1. **기본설정 가져오기**: 공통코드(E95)에서 가져온 기본 항목으로 초기 세팅
2. **[추가]** 클릭: 신규 자기소개서 항목 행 생성 후 내용 입력
3. **[저장]** 클릭: 신규 입력 또는 수정한 항목 저장
4. **[삭제]** 클릭: 선택한 항목 삭제

> ⚠️ **이미 게시된 채용공고에서는 자기소개서 항목을 추가/수정/삭제할 수 없습니다.** 항목 변경이 필요한 경우 게시 전에 완료하세요.

---

## STEP 7. 지원 시 필수 접수 파일 지정

### 진입 경로

`(UWIN) 직원채용관리 > 지원 시 필수 접수 파일 지정`

---

### 화면 개요

관리자가 채용공고의 **모집부서, 모집분야**별로 지원 시 필수 첨부파일 항목을 설정합니다.

### 처리 순서

1. **기본설정 가져오기**: 공통코드(E88)에서 가져온 기본 항목으로 초기 세팅
2. **[추가]** 클릭: 신규 제출서류(첨부파일) 항목 행 생성 후 내용 입력
3. **[저장]** 클릭: 신규 입력 또는 수정한 첨부파일 항목 저장
4. **[삭제]** 클릭: 선택한 제출서류 항목 삭제

> ⚠️ **이미 게시된 채용공고에서는 필수 접수 파일 항목을 추가/수정/삭제할 수 없습니다.** 항목 변경이 필요한 경우 게시 전에 완료하세요.
