# 지원자 심사 관리

## 📍 개요

지원자 접수부터 서류심사, 면접심사까지 전 과정을 통합 관리하는 가이드입니다.

- **목적**: 공정하고 투명한 심사 절차 운영
- **주요 산출물**: 지원자 명단, 서류심사 평가표, 면접평가표, 최종합격자(안)

---

## 1단계: 자기소개서 항목 설정

### 관리 메뉴
`(UWIN) 직원채용관리 > 자기소개서 항목 관리`

### 처리 절차

1. **기본설정 가져오기**: 공통코드(E95)에서 기본 항목을 세팅합니다.
2. **항목 추가**: 필요 시 신규 자기소개서 항목을 추가합니다.
3. **저장**: 신규 입력 또는 수정한 항목을 저장합니다.
4. **삭제**: 선택한 항목을 삭제합니다(필요시).

### ⚠️ 주의사항
**이미 게시된 채용공고에서는 자기소개서 항목을 추가/수정/삭제할 수 없습니다.** 채용공고 게시 전에 항목 설정을 완료하세요.

---

## 2단계: 지원자 접수 관리

### 관리 메뉴
`일반행정 > 채용관리 > 직원 > 채용지원자 관리`

### 지원자 현황 조회 및 관리

**조회 기능**
- 모집연도, 모집명, 모집부서, 분야, 지원자번호/명으로 검색 가능

**주요 기능 버튼**

| 버튼 | 기능 |
|------|------|
| **[조회]** | 다양한 조건으로 지원자 검색 |
| **[추가]** | 관리자가 지원자 직접 등록 |
| **[삭제]** | 체크박스 선택 지원자 삭제 |
| **[지원서 출력]** | 지원자 정보 리포트 출력 |
| **[자기소개서 출력]** | 자기소개서 리포트 출력 |
| **[공고별 제출서류 일괄다운로드]** | 전체 지원자 첨부파일 압축 다운로드 |

### 지원자 정보 관리 탭

| 탭 | 관리 항목 |
|----|---------|
| **기본** | 성명, 생년월일, 성별, 연락처, 이메일, 국적, 학가구분 등 |
| **학력사항** | 학력구분, 학교명, 학과명, 학위, 입학/졸업일, 취득국가 |
| **경력사항** | 기관명, 직위명, 재직기간, 근무년수, 담당직무, 임상여부 |
| **자격,면허,상,벌** | 자격증, 면허, 수상, 징계 이력 |
| **자기소개서** | 설정된 항목별 자소서 내용 |
| **제출서류** | 첨부파일 업로드, 다운로드, 일괄다운로드 |

### 지원자 직접 등록

오프라인 접수 또는 시스템 오류 발생 시 관리자가 직접 등록합니다.

**입력항목**

| 항목 | 입력 방식 |
|------|---------|
| 모집명 | 드롭다운 선택 |
| 모집부서 / 모집분야 | 드롭다운 선택 |
| 한글 성명 / 영문 성명 / 한자 성명 | 직접 입력 |
| 생년월일 | 드롭다운 선택 |
| 성별 | 남/여 선택 |

**지원번호는 저장 시 시스템이 자동 생성됩니다.**

### ⚠️ 주의사항

- **심사점수가 입력된 지원자는 삭제 불가**
- **진행상태 변경(임시저장→제출)은 부득이한 경우에만 사용**
- 지원 희망자가 지원이 안 된다면 총무인사팀으로 예외처리 문의

**지원서 수정 안내**: 지원자가 제출 후 수정을 희망하면 상태 정보를 변경하여 재작성 가능하게 한 후, 대상자에게 안내하세요.

## 3단계: 서류심사 진행

### 목적
1차 서류심사를 수행하여 합격자, 예비자, 불합격자를 결정합니다.

### 주요 산출물
- 서류심사 평가표(서명본)
- 합격자·예비자·불합격자 명단

### 진행 순서

1. **심사위원 확정 및 서약서 수령**
   - 심사위원 3명 이상 권장(내부/외부 혼합)
   - 서약서, 회피신고서 등 필수 서류 확보

2. **지원서·자기소개서·증빙자료 배부**
   - 심사위원에게 평가 관련 자료 제공 또는 열람 안내

3. **평가 기준 사전 공유**
   - 평가 항목, 배점, 가중치 사전 안내
   - 평가의 객관성·공정성 확보

4. **개별 평가**
   - 각 심사위원이 평가표를 작성하고 날인

5. **점수 취합 및 순위 산정**
   - 각 위원의 평가표 취합
   - 점수의 오류·누락·서명 여부 검증
   - 합계/평균 산출 및 순위 결정

6. **결과 확정**
   - 내부결재 또는 합의체 확정 방식으로 결과 확정
   - 결과는 내부결재 첨부자료 또는 회의록으로 남기기

### 체크리스트

- [ ] 심사위원 명단 확정(3명 이상 권장)
- [ ] 서약서/회피신고서 등 필수 서류 확보
- [ ] 평가 기준(항목/배점/가중치) 사전 공유
- [ ] 점수 취합(오류·누락·서명 여부) 검증
- [ ] 합격/예비/불합격 기준 및 배수(예: 3배수) 확정

### ⚠️ 유의사항

- **개인정보(주민등록번호 등) 포함 자료는 최소 열람·보관 원칙 준수**
- **평가표는 원본(서명본)을 차후 분쟁/감사 대비용으로 보관**

## 4단계: 면접심사 진행

### 목적
면접심사를 운영하여 면접 점수를 확정하고 최종합격자를 결정합니다.

### 주요 산출물
- 면접평가표(서명본)
- 면접점수집계표
- 최종합격자(안)

### 사전 준비

- [ ] 면접 일정/장소 확정 및 대상자 안내(유선/문자/메일 등)
- [ ] 면접관(심사위원) 확정 및 서약서 수령
- [ ] 면접 질문지/평가 기준 공유(질문 편향 방지)
- [ ] 준비물 확인: 필기도구, 출력물(지원서/자소서), 명찰, 대기공간 안내문

### 운영 기준(권장)

| 항목 | 내용 |
|------|------|
| **면접 방식** | 일대다 면접 원칙 |
| **면접 시간** | 1인 10분 이상(지원자 설명 기회 보장) |
| **복장/분위기** | 부서 자율(과도한 위압/불필요한 사생활 질문 금지) |

### 점수 취합 및 결과 확정

1. 심사위원별 평가표 취합
2. 합계/평균 산출
3. 순위 산정
4. 결과는 내부결재 첨부자료(또는 회의록)로 남기기

### ⚠️ 금지 질문(유의사항)

**다음 항목의 질문은 지양하세요:**
- 연애/결혼 계획
- 임신 계획
- 종교
- 출신지역
- 가족관계 등 사생활
- 차별 소지가 있는 질문

**법령·가이드라인을 준수하여 공정한 면접을 진행하세요.**

## 5단계: 성범죄 사실 조회

### 목적
최종합격자를 임용하기 전 성범죄 경력을 조회하여 결격 여부를 확인합니다.

### ⚠️ 중요: 임용일 이전 완료 필수

### 의뢰기관
울산남부경찰서

### 의뢰 절차

1. **대상자 안내 및 동의**
   - 최종합격자에게 성범죄 조회 안내
   - 동의서 수신(원본 보관)

2. **공문 발송**
   - 대상자 명단을 첨부하여 외부발신 공문 발송

3. **결과 회신 수신**
   - 조회 결과 회신(통상 2~3일 내)

4. **결과 통보**
   - 결격 여부를 총무인사팀에 통보

### 산출물
- 조회 요청 공문/명단
- 동의서(원본 보관)
- 회신 결과

## 전체 프로세스 요약

```
자기소개서 항목 설정
        ↓
지원자 접수 관리
        ↓
서류심사 진행 (1차)
        ↓
면접심사 진행 (2차)
        ↓
성범죄 사실 조회
        ↓
최종합격 확정
```

---

## 주요 확인사항

| 단계 | 확인 항목 |
|------|---------|
| **지원자 접수** | 심사점수 입력 전까지만 삭제 가능 |
| **서류심사** | 평가표 원본 보관, 개인정보 최소 취급 원칙 준수 |
| **면접심사** | 사생활 관련 질문 금지, 10분 이상 면접 시간 확보 |
| **성범죄 조회** | 임용일 이전 반드시 완료 |
