## 1. 근로계약서 작성

- 작성 일정은 별도 안내 예정
- 총무인사팀에서 **2부 작성** → 1부는 본인 보관, 1부는 총무인사팀 보관
- **1년 단위** 계약 (필요 시 변경근로계약서 별도 작성 가능)
- 📍 장소: 행정본관 1층 총무인사팀
- 👤 담당: 김대현 / ☎ 2037

---

## 2. 프로필 사진 촬영 *(선택)*

- 계약서 작성 후 **전산 등록용 사진** 촬영
- 보유 사진이 있으면 이메일로 파일 전달 가능
- **사진 등록 완료 후 UWIN에서 신분증(직원증) 발급 가능**
- 📍 장소: 행정본관 4층 대외홍보팀
- 👤 담당: 남수찬 / ☎ 2067

## 3. UWIN 계정 발급 및 사용

- 🌐 접속: [https://gw.ulsan.ac.kr/](https://gw.ulsan.ac.kr/)

## 4. 직원증 발급

직원증은 **신분 확인 + 사무실 출입 + 체크카드** 기능을 겸합니다.

**사전 확인사항**

1. `UWIN → 원스탑 → 개인정보조회` 에서 사번(6자리) 확인
2. 본인 사진 등록 여부 확인
   - 사진 미등록 시 → **총무인사팀 문의 후 사진 등록 선행**

**발급 절차**

1. 학사관리팀 방문 → 신분증 카드 발급신청서 작성
2. 신분증 카드 발급 수령
3. 교내 **경남은행 울산대학교 지점** 방문하여 카드 활성화
   - 운영 시간: 09:00 ~ 16:00
   - **이 때 급여계좌 개설 및 퇴직연금(DC) 계좌 발급도 함께 진행**

- 📍 발급 장소: 학생회관 1층 학사관리팀
- 🪪 지참물: 신분증 (주민등록증 또는 운전면허증)
- 💰 발급 비용: **무료**

---

## 5. 급여계좌 등록

- 등록 메뉴: `UWIN → 원스탑 → 급여/회계 → 급여(개인)계좌 신청`
- **경남은행 계좌만 등록 가능** (직원증 발급 시 교내 경남은행에서 함께 개설)
- **매달 10일 이전에 등록해야 당월 급여 수령 가능**
- 💰 급여일: 매달 **15일** (15일이 휴일인 경우 사전 지급)
- 👤 담당: 회계팀 이정혁 / ☎ 2054

---

## 6. 출입권한 신청

- 신청 메뉴: `UWIN → 원스탑 → 총무업무 → 출입권한신청`
- 결재 완료 후 → **행정본관 지하 1층 종합상황실** 방문하여 카드 등록 필요
- 부서 기존 직원에게 신청을 요청하거나, 직접 신청 시 담당자에게 문의
- 👤 담당: 총무인사팀 김남균 / ☎ 2039

## 8. 주차 등록 *(차량 보유자)*

- 🚗 주차요금: 월 **1만원** (급여에서 공제)
- 교직원 신분증 + 차량등록증 지참하여 주차관리실 방문
  - 가족 차량의 경우: **가족관계증명서 추가 제출**
- 📍 제출처: 주차관리실 (북카페 뒷쪽 출입문 → 복도에서 확인)

## 9. 휴가 사용 안내

- 신청 메뉴: `UWIN → 원스탑 → 인사정보 → 근태 신청`

### 휴가 종류

| 휴가 종류 | 내용 |
|-----------|------|
| **연가** | 매월 만근 시 1일 부여 / 첫 해 총 11일 |
| **병가** | 연 60일 이내 (질병·부상 등으로 집무 불가 시) |
| **보건휴가** | 월 1회, 무급 처리 |
| **특별휴가** | 하·동계 방학 중 각 5일 / **6개월 이상 근속 시 사용 가능** (방학 시작 전 별도 안내) |

> **외출·조퇴·지각**은 1시간 단위 사용하며, 합산 8시간 초과 시 연가 1일 차감

### 경조사 휴가 (청가)

| 사유 | 일수 |
|------|------|
| 본인 결혼 | 5일 |
| 자녀 결혼 | 1일 |
| 형제자매 결혼 | 1일 |
| 본인 출산 | 90일 |
| 배우자 출산 | 10일 |
| 입양 | 20일 |
| 배우자 사망 | 5일 |
| 본인·배우자 부모 사망 | 5일 |
| 조부모·외조부모 사망 | 2일 |
| 자녀 및 자녀 배우자 사망 | 2일 |
| 본인·배우자 형제자매 사망 | 1일 |

**담당자**
- 직원 문의: 총무인사팀 김현아 / ☎ 2034
- 조교 문의: 교무팀 오현호 / ☎ 1007

| 상황 | 방법 |
|------|------|
| 학내 → 학내 | 끝 4자리만 누름 (예: 259-2033 → **2033**) |
