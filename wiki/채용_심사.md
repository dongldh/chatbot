> **적용 구분:** 신규임용
> **관련 시스템 메뉴:** `1.3 모집부서 관리` / `1.7 심사과정 관리` / `1.8 심사위원 관리` / `1.12 심사사정 관리` / `1.20 전형별 심사 입력`

① 서류심사 진행  →  ② 심사위원 등록  →  ③ 면접심사 진행  →  ④ 전형별 심사점수 입력
 (STEP 9)              (STEP 5)              (STEP 12)              (STEP 10 / 13)

## ① 서류심사 진행

### 개요

| 항목 | 내용 |
|------|------|
| 목적 | 서류심사(1차) 수행 및 결과 확정 |
| 산출물 | 서류심사 평가표(서명본) / 합격자·예비·불합격자 명단 |

### 진행 순서

1. 심사위원(내부/외부) 확정 및 서약서 수령
2. 지원서·자기소개서·증빙자료 배부(또는 열람 안내)
3. 심사위원별 개별 평가(평가표 작성 및 날인)
4. 점수 취합 및 순위 산정
5. 결과 확정(내부결재 또는 합의체 확정 방식)

### 사전 체크리스트

- [ ] 심사위원 명단 확정 (**3명 이상 권장**)
- [ ] 서약서·회피신고서 등 필수 서류 확보
- [ ] 평가 기준(항목·배점·가중치) 사전 공유
- [ ] 점수 취합 시 오류·누락·서명 여부 검증
- [ ] 합격/예비/불합격 기준 및 배수(예: 3배수) 확정

### ⚠️ 유의사항

- **개인정보(주민등록번호 등) 포함 자료 취급 시 최소 열람·보관 원칙 준수**
- **평가표 원본(서명본)은 차후 분쟁·감사 대비를 위해 반드시 보관**

## ② 심사위원 등록

### 개요 및 진입 경로

`1.3 모집부서 관리 → 심사단계 [선택] → 심사과정 관리 팝업(1.7) → 심사단계별 [관리] 버튼 클릭`

> **⚠️ [관리] 버튼은 심사단계가 저장된 후에만 표시됩니다. 반드시 단계를 먼저 저장하세요.**

### 심사위원 등록 방법 (메뉴 `1.8 심사위원 관리`)

#### 내부 vs 외부 심사위원 등록 구분

| 구분 | 등록 방법 | 입력 항목 |
|------|-----------|-----------|
| **내부** | 🔍 돋보기 → 교직원 검색 팝업 → 선택 | 성명·부서·직급 자동 입력 |
| **외부** | [추가] → 직접 입력 | 성명 / 부서(소속) / 직급 |

#### 처리 순서

1. **[추가]** 클릭 → 신규 행 생성
2. 구분 선택: `내부` 또는 `외부`
3. **내부** 심사위원: 🔍 클릭 → 교직원 검색 후 선택 (성명·부서·직급 자동 입력)
4. **외부** 심사위원: 성명·부서(소속)·직급 직접 입력
5. **[저장]** 클릭 → 심사위원 등록 완료

### 심사사정 통합 관리 (메뉴 `1.12 심사사정 관리`)

모집연도·모집명·모집부서·모집분야별로 심사단계와 심사항목을 통합 확인·관리하는 화면으로, `1.7 팝업`과 동일한 기능을 제공합니다.

| 기능 버튼 | 설명 |
|-----------|------|
| [추가] | 심사단계 또는 심사항목 추가 |
| [저장] | 추가·수정한 내용 저장 |
| [삭제] | 선택한 단계 또는 항목 삭제 |
| [심사위원] 관리 | 단계별 심사위원 관리 팝업 열기 |
| [심사항목설정] | 해당 심사단계 항목 조회 |

### ⚠️ 유의사항

- **[관리] 버튼은 심사단계 저장 후에만 표시됨** (단계를 먼저 저장할 것)
- **이미 심사평가가 진행 중인 심사위원은 삭제 불가**
- 심사단계는 심사위원 또는 심사항목이 이미 등록되어 있는 경우 삭제 불가
- 심사항목은 해당 모집의 지원자 평가가 존재하는 경우 삭제 불가
- **심사항목의 배점 합계가 반드시 100점이 되어야 저장 가능**

## ③ 면접심사 진행

### 개요

| 항목 | 내용 |
|------|------|
| 목적 | 면접심사 운영 및 면접 점수 확정 |
| 산출물 | 면접평가표(서명본) / 면접점수집계표 / 최종합격자(안) |

### 사전 준비 체크리스트

- [ ] 면접 일정·장소 확정 및 대상자 안내 (유선·문자·메일 등)
- [ ] 면접관(심사위원) 확정 및 서약서 수령
- [ ] 면접 질문지·평가 기준 공유 (질문 편향 방지)
- [ ] 준비물 확인: 필기도구, 출력물(지원서·자소서), 명찰, 대기공간 안내문

### 면접 운영 기준 (권장)

| 항목 | 기준 |
|------|------|
| 면접 방식 | 일대다 면접 원칙 |
| 면접 시간 | 1인 10분 이상 권장 (지원자 설명 기회 보장) |
| 복장·분위기 | 부서 자율 (단, 과도한 위압·불필요한 사생활 질문 금지) |

### ⚠️ 금지 질문 유의사항

**연애·결혼·임신 계획, 종교, 출신지역, 가족관계 등 사생활·차별 소지가 있는 질문은 지양** (관련 법령·가이드라인 준수)

### 점수 취합 방법

1. 심사위원별 평가표 취합
2. 합계·평균 산출 및 순위 산정
3. 결과를 내부결재 첨부자료(또는 회의록)로 보관

## ④ 전형별 심사점수 입력

### 개요 및 진입 경로

`일반행정 > 채용관리 > 직원 > 전형별 심사 입력` (메뉴 `1.20`)

> 서류심사(STEP 10)와 면접심사(STEP 13) 모두 동일한 화면에서 입력합니다.
> 면접 전형의 경우 **심사구분을 '면접'으로 선택**하여 동일하게 진행합니다.

### 조회 전 필수 선택 항목

| 항목 | 필수 여부 |
|------|-----------|
| 모집연도 | ✅ 필수 |
| 모집명 | ✅ 필수 |
| 모집부서 | ✅ 필수 |
| 모집분야 | ✅ 필수 |
| 심사구분 | ✅ 필수 |
| 심사위원 | 🔍 돋보기로 검색·선택 (**심사위원 미등록 시 먼저 등록**) |

### 방법 1 — 화면 직접 입력 (소수 지원자)

1. 조회 조건 선택 후 **[조회]** 클릭
2. 왼쪽 목록에서 지원자 클릭
3. 오른쪽 심사점수 입력 영역에 항목별 점수 입력
4. **[저장]** 클릭

### 방법 2 — 엑셀 업로드 (다수 지원자, 권장)

1. **[엑셀 다운로드]** 클릭 → 시스템 서식 파일 저장
2. 엑셀 파일에 심사위원·심사항목·점수·등급 입력
3. **[엑셀 업로드]** 클릭 → 파일 선택
4. 오류내역 없음 확인 후 **[저장]** 클릭

> **⚠️ 반드시 시스템에서 [엑셀 다운로드]로 받은 서식 파일을 사용하세요. 자체 제작 파일은 업로드 불가합니다.**

> 📌 등급은 필수값이 아니며, 점수와 별개로 필요한 경우에만 사용합니다.

### ⚠️ 핵심 주의사항

- **엑셀 업로드 시 반드시 시스템에서 제공하는 [엑셀 다운로드] 서식 사용**
- **심사위원이 등록되어 있지 않으면 점수 입력 불가 → 심사위원 먼저 등록**
- 심사구분(서류/면접 등)을 정확히 선택한 후 조회할 것
