# 선택적 근로시간제 사용 안내

> 이 페이지는 울산대학교 선택적 근로시간제의 운영 방법을 **팀원**과 **팀장** 관점으로 구분하여 안내합니다.
> 문의: 총무인사팀 (메일 또는 메신저)

---

## 1. 제도 개요

### 선택적 근로시간제란?

유연근무제의 일종으로, **1개월 정산 기간 내 1주 평균 52시간(기본 40시간 + 연장근로 12시간)을 초과하지 않는 범위**에서 근로자가 근무시간을 자유롭게 조정할 수 있는 제도입니다.

- **의무근로시간대**: 11:00~16:00
- **일반 근무시간**: 09:00~18:00 (일부 부서 08:00~17:00)
- 근무시간 조정은 **부서 내 협의**를 통해 결정합니다.

### 제도의 이점

- 특정일에 더 많이 일하고 다른 날에 적게 일하는 **유연한 시간 배분**이 가능합니다.
  - 예) 월요일 10시간 근무 → 화요일 6시간 근무 가능
- 한 달 근로시간 합산 후 **기준근로시간(근무일수 × 8시간)을 초과한 경우**, 다음 달에 보상휴가를 부여받습니다.
- **받은 보상휴가는 해당 월 내에 사용하는 것이 원칙**입니다.

---

## 2. 대체휴무 vs 보상휴가 차이

| 구분 | 대체휴무 | 보상휴가 |
|------|----------|----------|
| 사용 시기 | 연장근로가 발생한 **해당 월** | 해당 월에 미사용 시 **다음 달 이후** |
| 사용 조건 | 해당 월 소정근로시간 초과가 예상되는 경우에만 사용 가능 | 해당 월에 보상휴무를 사용하지 못한 경우 |
| 휴무 시간 | 연장근로시간과 **동일한 시간** | 연장근로시간의 **1.5배** |

## 3. 팀원 가이드

### 🙋 1단계. 연장근로 사전 보고 및 UWIN 입력

**연장근로는 반드시 사전 보고 후 승인을 받아야 인정됩니다.**

1. 팀장에게 연장근로 필요성을 보고하고 구두 승인을 받습니다.
2. UWIN에 연장근로 정보를 입력합니다.
   - 메뉴 경로: `원스탑 근로관리 > 근로관리 > 내근로현황` → 해당 일자 클릭 → 팝업 입력
   1. 연장근로 예정 시간을 입력합니다. (10분 단위)
   2. 휴게시간을 반영하여 자동 산정된 근로시간을 확인합니다.
   3. 연장근로 사유를 **구체적으로** 입력합니다.
   4. [저장]을 누르고 저장 내용을 최종 확인합니다.
3. 팀장이 해당 내용을 부서장 전결로 전자결재하여 총무인사팀으로 송부합니다. *(팀장 처리 사항)*

---

### ✅ 2단계. 퇴근 기록 남기기

1. 연장근로 후 퇴근 시 **반드시 KT 단말기에 퇴근 기록을 남깁니다.**
2. **퇴근 기록이 있는 경우에만 연장근로로 인정됩니다.**
3. **실수로 기록하지 않은 경우에도 소급 인정되지 않으므로** 반드시 기록하시기 바랍니다.

---

### 🏄 3단계. 대체휴무 또는 보상휴가 신청하기

#### 대체휴무 신청

- 메뉴 경로: `원스탑 근로관리 > 근로관리 > 내근로현황` → 해당 일자 클릭 → 팝업 입력

1. [대체휴무]를 선택하고 사용할 시간을 입력합니다. (10분 단위)
2. 입력된 휴무 시간을 확인합니다.
3. 사유란에 `대체휴가 사용`이라고 입력합니다.
4. [저장]을 누르고 저장 내용을 최종 확인합니다.

#### 보상휴가 신청

- 메뉴 경로: `원스탑 근로관리 > 근로관리 > 내근로현황` → 해당 일자 클릭 → 팝업 입력

1. [보상휴가]를 선택하고 사용할 시간을 입력합니다. (10분 단위)
2. 입력된 보상휴가 시간을 확인합니다.
3. 사유란에 `보상휴가 사용`이라고 입력합니다.
4. [저장]을 누르고 저장 내용을 최종 확인합니다.

---

## 4. 팀장 가이드

### 📋 일별 연장근로 관리 절차

#### 연장근로 당일 — 사전 협의 및 결재

1. 팀원으로부터 연장근로 보고를 받고 필요성을 검토하여 승인합니다.
2. **연장근로는 사전 승인된 경우에만 인정됩니다.**
3. 팀원이 UWIN에 연장근로 정보를 입력하도록 안내합니다.
4. 일과 종료 전, 당일 예정된 연장근로 건을 **부서장 전결로 전자결재**하여 총무인사팀으로 송부합니다.

#### 연장근로 다음날 — 출퇴근 기록 확인

1. 팀원이 KT 단말기에 남긴 퇴근 기록을 확인합니다.
2. **퇴근 기록이 있는 경우에만 연장근로로 인정됩니다.**
3. 퇴근 기록이 전날 UWIN에 입력한 시간과 다른 경우, 총무인사팀에 **조정 요청**합니다.
   - 예) UWIN 입력: 22:00 종료 예정 → 실제 퇴근 기록: 23:00 → 조정 요청 필요

---

### 📅 월별 근로시간 확정 및 보상휴가 부여

1. 해당 월 근로시간이 **최종 확정된 후** 보상휴가 부여 전자결재를 진행합니다.
2. UWIN에서 소속 단위의 **[초과시간]이 존재하는 경우** 전자결재 버튼이 활성화됩니다.
3. 전자결재 결재라인은 **팀(실)장 → 부서장(처장)** 단까지 지정하며, 총무인사팀을 수신 부서로 설정합니다.
4. **⚠️ 전자결재 버튼을 누르는 순간 해당 연월의 근로 데이터가 확정(고정)되며, 이후 수정이 불가합니다.** 반드시 최종 근로시간을 확인한 후 전자결재를 올리시기 바랍니다.

---

### 🔄 연장근로 보상 처리 기준

| 연장근로 발생 시기 | 보상 사용 시기 | 처리 방식 |
|------------------|--------------|----------|
| 3월 발생 | 3월 사용 | 대체휴무 |
| 3월 발생 | 4월 이후 사용 | 보상휴가 (연장시간 × 1.5배) |

- 연장근로에 따른 **휴게시간 보장** 여부를 반드시 확인합니다.
- 보상휴가는 **부여된 달에 모두 사용**할 수 있도록 팀원에게 안내합니다.
