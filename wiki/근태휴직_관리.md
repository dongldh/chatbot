# 근태 및 휴직 관리

## 개요

근태는 출근과 퇴근에 관련된 모든 사항을 말하며, 근로기준법, 단체협약, 취업규칙에 따라 울산대학교의 근태제도를 기획하고 운영하는 업무입니다. 근태제도는 업무로 소진된 에너지를 재충전하는 생활에 필수적인 제도입니다.

**관련규정**: 울산대학교 일반직원 복무 규정

---

## 근태 접수 및 처리

### 매일 해야 하는 일

근태는 매일 신청서가 접수되고 결재되어야 하는 업무입니다.

#### 근태 접수 방법

**경로**: `UWIN 교직원포탈 > 전자결재 > 부서수신함`

1. 휴가원 더블 클릭
2. (휴가원 팝업) 접수자 전결 클릭 / 결재자 지정
3. (기록물철 선택 팝업) 근태관계철 더블클릭 후 확인 클릭
4. (UWIN비밀번호 입력 팝업) 비밀번호 입력 후 확인 클릭
5. 완료

#### 처리 시 원칙

**1. 전결 기준 확인**
- 팀(실)원 근태: 팀(실)장 전결 → 총무인사팀 제출 → 총무인사팀도 팀장 전결로 종료
- 팀(실)장 근태: 부서장 전결 → 총무인사팀 제출 → 총무인사팀도 부서장(처장) 전결로 종료
- 총무처 소속 팀장 근태: 총무인사팀장 전결로 종료 (중복 방지)

**2. 증빙 자료 확인**
- **자연발생적 휴가** (연가, 하계휴가 등): 증빙 불필요
- **사유가 필요한 휴가** (병가, 출산휴가 등): 증빙 필요

**3. 특수성에 따른 보고**
- 병가, 청가 등 특수한 근태: 총무처장 결재까지 올림 (팀장에게 사전 보고)

#### 참고사항

- **연가는 부서에서 결재가 종료되며 총무인사팀으로 결재문서가 오지 않음** (위임전결규정에 따름)

## 매월 해야 하는 일

### 1일자 업무

**선택적 근로시간제 시행에 따른 월초 필수 업무**

1. **시점별 현황 생성**
   - 메뉴: `일반행정 > 직원인사 > 통계관리 > 시점별현황생성[CAH20U]`
   - 해당 월 1일자로 [생성] 버튼 클릭
   - 기본근로시간 등록, 근태시간 등이 정상 확인되기 위한 기초 작업

2. **기본근로시간 생성**
   - 메뉴: `일반행정 > 직원인사 > 근로관리 > 기본근로시간관리(부서별)[CAK05U]`
   - **의과대학 3개 팀 제외** (별도 출퇴근 시간 사용)
   - 학기중/방학 구분하여 입력

**학기중 기본근로시간** (월~목)
- 근무시간: 09:00~18:00
- 휴게시간1: 12:00~13:00
- 단축시간: 00:00~00:00
- 휴게시간2: 18:00~18:30

**학기중 기본근로시간** (금)
- 근무시간: 09:00~18:00
- 휴게시간1: 12:00~13:00
- 단축시간: 17:00~18:00
- 휴게시간2: 18:00~18:30

**방학중 기본근로시간** (월~금)
- 근무시간: 09:00~18:00
- 휴게시간1: 12:00~13:00
- 단축시간: 15:00~18:00 (단체협약 단축시간)
- 휴게시간2: 18:00~18:30

## 선택적 근로시간제

### 기본 개념

대학의 근로시간 제도의 근간으로, 1개월 정산 기간 내 **1주일 평균 52시간** (기본 40시간 + 연장근로 12시간)을 초과하지 않는 범위에서 근로자가 근무시간을 자유롭게 조정하는 제도입니다.

**의의**: 연장근로에 대한 보상을 체계적으로 관리하기 위해 노사간 논의를 거쳐 도입

### 부서 등록

#### 선택적 근로시간제 부서 등록 절차

1. 부서 요청
2. 공문 접수
3. 협약서 조정
4. 전산 등록

#### 전산 등록 방법

**메뉴**: `일반행정 > 직원인사 > 기준정보관리 > 선택적근무부서등록[CAB13U]`

- 부서조회
- 시작일 입력
- **종료일 입력 (99991231)** - 미입력 시 현재부서로 확인되지 않음

### UWIN 메뉴 설명

#### 근로관리 메뉴

**기본근로시간관리(부서별)[CAK05U]**
- 월별 근로시간을 관리하는 메뉴
- 모든 부서의 근로시간을 컨트롤 가능

**기본근로시간관리(개인별)[CAK03U]**
- 개인별 근로시간 조정용
- 산정로직으로 해결되지 않는 경우 사용

**개인별근태관리[CAK02U]**
- 캘린더로 개인 근로 현황 확인
- 연장근로신청 및 근로시간기록 입력

**부서근태관리(일간)[CAK07U]**
- 팀(실)장이 활용하는 메뉴
- 일일별 연장근로, 대체휴무 승인

**부서근태관리(월간)[CAK08U]**
- 월 1회 근무 현황 종합
- 각 부서 팀(실)장이 월 근로시간 집계 후 총무인사팀에 보고

#### 가산시간관리 메뉴

**근로결재관리[CAK09U]**
- 부서에서 근로결재 반려 요청 시 사용
- 기존 전자결재 취소 필요

**보상휴가현황[CAK10U]**
- 보상휴가 발생 현황 다운로드
- 보상휴가 수당 지급 시 검증자료로 활용

**근태현황[CAK11P]**
- 선택적근로시간제 관련 통계자료 및 엑셀 다운로드

## 일별 연장근로 처리

### 연장근로 신청 및 확인

**경로**: `일반행정 > 직원인사 > 근로관리 > 부서근태관리(일간)[CAK07U]`

#### 확인 필요사항

1. **시작-종료시간**: 연장근로로 인정받을 시간
2. **KT출퇴근시간**: 직원이 출퇴근단말기에 기록한 시간
3. **내용**: 연장근로 사유 (명확하지 않으면 반송)
4. **부서장 결재**: 팀장 전결로 오는 경우 부서장 전결로 요청

**특별 확인사항**
- 대체휴무: 특별히 확인할 필요 없음

## 월별 보상휴가 관리

### 월 15일경 업무

1. **월단위 보상휴가 시행 사항 보고 문서 제출 요청**

2. **보상휴가 현황 접수**
   - 경로: `그룹웨어 > 전자결재 > 부서수신함`
   - 메뉴: `일반행정 > 직원인사 > 근로관리 > 부서근태관리(월간)[CAK08U]`

### 보상휴가 정산 후 업무 절차

**기본 원칙**
- 발생한 보상휴가는 반드시 사용하거나 보상하거나 수당으로 지급해야 함
- 수당으로 보상한 경우 기록을 반드시 남김

#### 처리 순서

1. 보상휴가 잔여시간 확인
2. 미사용자 보상금액 산정
3. 연장근로수당 신청서 접수
4. 보상금 지급
5. 노동조합 통보
6. 보상 후 기록 남기기

#### 보상휴가 기록 입력 방법

**메뉴**: `원스탑 인사정보 > 인사관리 > 휴가신청[CAC19G]`

**처리**
- 휴가구분: '보상휴가 수당지급건' 선택
- 휴가기간: 비활성화 (일자 지정 불필요)
- 차감필요시간: 보상한 시간 입력 (예: 73.25)
- 비고: '연장근로수당 지급 완료' 입력
- 총무처장 결재

## 근태 종류별 상세 내용

## 연가(연차휴가)

### 발생 체계

- **일반직원/무기계약직**: 회계연도 기준
- **계약직원**: 입사일자 기준

### 연차휴가 시행 관리

**메뉴**: `일반행정 > 직원인사 > 복무관리 > 휴가신청조회[CAC10U]`

**조회 방법**
1. 일자 입력
2. 조회 후 마우스로 드래그하여 복사
3. 엑셀에 값만 붙여넣기

### 휴가취소 처리

**방법**
1. 취소 요청 시 메일로 해당 일자를 받음 (근거 확보)
2. 메뉴: `일반행정 > 직원인사 > 복무관리 > 휴가신청[CAC09U]` 활용
3. 휴가시작일 일자 클릭 (기간 설정 가능)
4. 이름 조회 (또는 사번)
5. 조회 후 [반려(결재완료된전자결재취소)] 클릭
6. [삭제] 버튼 클릭

### 연가적립

**정규직원/무기계약직원**
- 단체협약에 따라 미사용연차휴가 2일 적립
- 메뉴: `일반행정 > 직원인사 > 복무관리 > 적립연가일괄생성[CAC26U]`
- 연간 최대 2일 적립, 유효기간 3년

**계약직원**
- 현재 시스템으로 자동 적립 처리됨 (재계약 시)
- **주의**: 신청, 결재진행중 상태인 연가를 최종 처리 후 적립생성 진행

### 학년도말 근태 정리

**확인 필요**
- 작성만 되고 결재하지 않은 휴가신청 현황 조회
- 메뉴: `일반행정 > 직원인사 > 복무관리 > 휴가신청조회[CAC10U]`
- 기간: 해당 회계연도 전체 (예: 2023.03.01. ~ 2024.02.29.)

**처리**
- 신청 상태: 본인에게 취소 권고
- 결재진행중: 결재 완료 또는 반려 처리

### 연가촉진

**대상**: 일반직원, 무기계약직원

**법적 근거**: 근로기준법 제61조

**진행 절차**
1. **6개월 전** (기준일: 연차 소멸일의 6개월 전)
   - 연차휴가사용촉진안내문 발송
   - 개인별 통지문 서면 발송

2. **2개월 전**
   - 휴가 사용 기한 강제지정
   - 과거 수년간 미시행 (단체협약상 10일 보상, 2일 적립이므로 12일 초과 연가가 드물 것으로 판단)

**참고**: 연가촉진 관련하여 법령 변경이 있을 수 있으므로 정기적 검토 필요

### 연가보상

**기본사항**
- 단체협약에 따라 10일까지 보상
- 대상: 일반직원, 무기계약직원, 계약직원 (체육감독/코치 등 특정 직급)
- 계산 기간: 3월 1일 ~ 다음해 2월 마지막 날
- 보상기준: 통상임금 기준
