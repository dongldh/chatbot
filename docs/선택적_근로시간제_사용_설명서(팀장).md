2. 연장근로 확인하기(연장근로 시행일 +1)
가. 출퇴근 기록 확인

 - 출퇴근 기록이 있는 경우만 연장근로로 인정받을 수 있으며 기록이 전날 UWIN에 입력한 시간과 다르다면 총무인사팀에 조정 요청을 합니다.(22시 종료 예정, 퇴근기록 23시인 경우 조정요청)
