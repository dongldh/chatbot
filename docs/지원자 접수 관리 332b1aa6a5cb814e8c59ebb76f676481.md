- 📘 4-1. 지원자 현황 조회 및 관리 (메뉴 1.13)
    
    > 
    > 
    > 
    > ![ PDF 16페이지](%EC%A7%80%EC%9B%90%EC%9E%90%20%EC%A0%91%EC%88%98%20%EA%B4%80%EB%A6%AC/image.png)
    > 
    >  PDF 16페이지

    **주요 기능 버튼**
    
    | 버튼 | 기능 |
    | --- | --- |
    | **[조회]** | 모집연도·모집명·모집부서·분야·지원자번호/명으로 검색 |
    | **[추가]** | 관리자가 지원자 직접 등록 (1.14 팝업) |
    | **[삭제]** | 체크박스 선택 지원자 삭제 (심사점수 입력 전만 가능) |
    | **[지원서 출력]** | 지원자 정보 리포트 출력 |
    | **[자기소개서 출력]** | 자기소개서 리포트 출력 |
    | **[공고별 제출서류 일괄다운로드]** | 전체 지원자 첨부파일 압축 다운로드 |
    
    **탭별 관리 내용**
    
    | 탭 | 관리 항목 |
    | --- | --- |
    | **기본** | 성명·생년월일·성별·연락처·이메일·국적·학가구분 등 |
    | **학력사항** | 학력구분·학교명·학과명·학위·입학일·졸업일·취득국가 |
    | **경력사항** | 기관명·직위명·재직기간·근무년수·담당직무·임상여부 |
    | **자격,면허,상,벌** | 자격증·면허·수상·징계 이력 |
    | **자기소개서** | 항목별 자소서 내용 (설정된 항목만 표시) |
    | **제출서류** | 첨부파일 업로드·다운로드·일괄다운로드 |
    
    > ⚠️ 이미 심사점수가 입력된 지원자는 삭제 불가
    >

FAQ

- 입사지원서를 제출하고 본인이 수정을 희망할 때 어떻게 해야 하나요?
    
    상태 정보를 변경하고 저장하면 재작성할 수 있는 상태가 됩니다. 변경 후에 대상자에게 안내를 하시기 바랍니다.
