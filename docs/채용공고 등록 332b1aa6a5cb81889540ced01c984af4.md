## 📍 화면 진입 경로

`일반행정 > 채용관리 > 직원 > 채용공고 관리`

---

> ⚠️ **잘못 게시된 공고 수정 불가** — 접수기간 내에는 내용 수정 불가. 진행상태를 **N**으로 변경 후 신규 공고 등록.
> 

---

- 📘 1-1. 템플릿 관리 (메뉴 1.1) — 공고문 양식을 미리 만들어 두는 곳
    
    **화면 위치:** `일반행정 > 채용관리 > 직원 > 채용공고문 템플릿 관리`
    
    > 
    > 
    > 
    > ![📷 [PDF 4페이지]](%EC%B1%84%EC%9A%A9%EA%B3%B5%EA%B3%A0%20%EB%93%B1%EB%A1%9D/image.png)
    > 
    > 📷 [PDF 4페이지]
    > 
    
    | 순서 | 버튼/동작 | 설명 |
    | --- | --- | --- |
    | 1 | **[추가]** 클릭 | 템플릿명 입력 칸 + 에디터 초기화 |
    | 2 | 템플릿명 입력 | 예: `2025년 직원채용 기본양식` |
    | 3 | 에디터에 공고문 작성 | 지원자격·제출서류·전형일정 등 |
    | 4 | **[저장]** 클릭 | 템플릿 저장 완료 |
    
    **수정/삭제:** 목록에서 템플릿명 클릭 → 수정 후 [저장] / 체크박스 선택 후 [삭제]
    
    > ⚠️ 템플릿을 나중에 수정해도 **이미 등록된 채용공고문은 변경되지 않습니다**
    > 

---

- 📘 1-2. 채용공고 등록 (메뉴 1.2) — 실제 공고를 생성하는 곳
    
    **화면 위치:** `일반행정 > 채용관리 > 직원 > 채용공고 관리`
    
    > 
    > 
    > 
    > ![📷 [PDF 5페이지] ](%EC%B1%84%EC%9A%A9%EA%B3%B5%EA%B3%A0%20%EB%93%B1%EB%A1%9D/image%201.png)
    > 
    > 📷 [PDF 5페이지] 
    > 
    > ![image.png](%EC%B1%84%EC%9A%A9%EA%B3%B5%EA%B3%A0%20%EB%93%B1%EB%A1%9D/image%202.png)
    > 
    
    **등록 순서**
    
    | 순서 | 버튼/동작 | 설명 |
    | --- | --- | --- |
    | 1 | **[추가]** 클릭 | 채용공고 관리 팝업(1.6) 열림 |
    | 2 | 기본 정보 입력 | 아래 입력항목 표 참조 |
    | 3 | **[저장]** 클릭 | 공고 목록에 추가됨 |
    | 4 | 채용공고문 **[선택]** 버튼 클릭 | 템플릿 조회 팝업(1.4) 열림 |
    | 5 | 템플릿 선택 → **[공고문 템플릿 적용]** | 공고문 관리 팝업(1.5)으로 이동 |
    | 6 | 공고문 내용 편집 후 **[저장]** | 완료 |
    
    **입력항목 상세**
    
    | 항목 | 필수 | 입력 방법 | 예시/설명 |
    | --- | --- | --- | --- |
    | 모집명 | ✅ | 직접 입력 | `2025학년도 울산대학교 직원채용 공고` |
    | 구분 | ✅ | 드롭다운 | `직원` / `계약직원` |
    | 담당부서 | ✅ | 직접 입력 | `정보화개발팀` |
    | 학가구분 | ✅ | 드롭다운 | 미선택 시 전체 대상 |
    | 모집시작일/시간 | ✅ | 날짜+시간 | `2025.03.05 / 09:00` |
    | 모집종료일/시간 | ✅ | 날짜+시간 | `2025.05.05 / 17:00` |
    | 진행상태 | ✅ | Y/N | **Y = 사이트 게시됨** |
    | 중복지원 가능여부 | — | 체크박스 | 체크 시 같은 공고 내 다른 부서 복수지원 가능 |
    | 타부서열람여부 | — | 체크박스 | 체크 시 타부서 담당자 열람 가능 |
    
    **진행상태별 채용사이트 표시**
    
    | 진행상태 | 날짜 조건 | 사이트 표시 |
    | --- | --- | --- |
    | Y | 시작 전 | `오픈예정` |
    | Y | 접수 중 | `접수 중` |
    | Y | 종료 후 | `마감` |
    | N | 무관 | 미게시 |
