## 📍 화면 진입 경로

`일반행정 > 채용관리 > 직원 > 전형별 심사 입력`

---

- 📘 5-1. 심사점수 입력 방법 (메뉴 1.20)
    
    > 
    > 
    > 
    > ![📷 [PDF 23페이지]](%EC%A0%84%ED%98%95%EB%B3%84%20%EC%8B%AC%EC%82%AC%EC%A0%90%EC%88%98%20%EC%9E%85%EB%A0%A5/image.png)
    > 
    > 📷 [PDF 23페이지]
    > 
    
    **조회 전 필수 선택 항목**
    
    | 항목 | 필수 여부 |
    | --- | --- |
    | 모집연도 | ✅ |
    | 모집명 | ✅ |
    | 모집부서 | ✅ |
    | 모집분야 | ✅ |
    | 심사구분 | ✅ |
    | 심사위원 | 🔍 돋보기로 검색·선택 (없을 경우 위원등록 먼저) |

---

- 📘 방법 1 — 화면에서 직접 입력 (소수 지원자)
    
    
    | 순서 | 동작 |
    | --- | --- |
    | 1 | 조회 조건 선택 후 **[조회]** |
    | 2 | 왼쪽 목록에서 지원자 클릭 |
    | 3 | 오른쪽 심사점수 입력 영역에 항목별 점수 입력 |
    | 4 | **[저장]** 클릭 |

---

- 📘 방법 2 — 엑셀 업로드 (다수 지원자, 권장)
    
    
    | 순서 | 동작 |
    | --- | --- |
    | 1 | **[엑셀 다운로드]** 클릭 → 서식 파일 저장 |
    | 2 | 엑셀 파일에 심사위원·심사항목·점수·등급 입력 |
    | 3 | **[엑셀 업로드]** 클릭 → 파일 선택 |
    | 4 | 오류내역 없음 확인 후 **[저장]** |
    
    > ⚠️ 반드시 **시스템에서 [엑셀 다운로드]로 받은 서식 파일** 사용. 자체 제작 파일 업로드 불가.
    > 
    
    > 📌 등급은 필수값이 아니며 점수와 별개로 필요 시만 사용합니다.
    >
