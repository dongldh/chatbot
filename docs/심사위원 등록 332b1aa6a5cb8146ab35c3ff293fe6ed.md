## 📍 진입 경로

`1.3 모집부서 관리 → 심사단계 [선택] → 심사과정 관리 팝업(1.7) → 심사단계별 [관리] 버튼 클릭`

1. 심사위원 버튼은 심사단계가 저장이 된 후에 보인다.
2. 심사단계는 심사위원, 심사항목이 이미 등록되어 있는 경우 삭제할 수 없다.
3. 심사항목은 해당 모집(모집연도, 모집명, 모집부서, 모집분야)의 지원자 평가가 있는 경우 삭제할 수 없다.
4. 심사항목의 배점이 총100점이 되어야 저장할 수 있다.

    **내부 vs 외부 심사위원 등록 방법**
    
    | 구분 | 등록 방법 | 입력 항목 |
    | --- | --- | --- |
    | **내부** | 🔍 돋보기 → 교직원 검색 팝업 → 선택 | 자동 입력 |
    | **외부** | [추가] → 직접 입력 | 성명 / 부서(소속) / 직급 |
    
    **처리 순서**
    
    | 순서 | 버튼/동작 | 설명 |
    | --- | --- | --- |
    | 1 | **[추가]** 클릭 | 신규 행 생성 |
    | 2 | 구분 선택 | `내부` 또는 `외부` |
    | 3 | 내부: 🔍 클릭 → 교직원 검색·선택 | 성명·부서·직급 자동 입력 |
    | 4 | 외부: 성명·부서(소속)·직급 직접 입력 | — |
    | 5 | **[저장]** 클릭 | 심사위원 등록 완료 |
    
    ⚠️ **[관리] 버튼은 심사단계가 저장된 후에만 표시됨** (단계를 먼저 저장해야 함)
    
    ⚠️ 이미 심사평가가 진행 중인 심사위원은 삭제 불가
