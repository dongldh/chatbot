- 📘 2-1. 모집부서 등록 (메뉴 1.3 → 1.11 Popup)
    
    > 
    > 
    > 
    > ![image.png](%EB%AA%A8%EC%A7%91%EB%B6%80%EC%84%9C%20%EC%84%A4%EC%A0%95/image.png)
    > 
    
    | 순서 | 버튼/동작 | 설명 |
    | --- | --- | --- |
    | 1 | 모집년도·모집명 선택 후 **[조회]** | 해당 공고의 모집부서 목록 조회 |
    | 2 | **[추가]** 또는 모집분야명 클릭 | 모집부서 관리 팝업(1.11) 열림 |
    | 3 | 팝업에서 항목 입력 후 **[저장]** | 아래 입력항목 표 참조 |
    
    **모집부서 입력항목**
    
    | 항목 | 설명 | 예시 |
    | --- | --- | --- |
    | 모집부서 | 🔍 돋보기로 부서 검색·선택 | `정보화개발팀` |
    | 모집분야 | 채용 직무명 | `전산직` |
    | 모집인원 | 숫자 입력 | `2` |
    | 지원자격 | 텍스트 직접 입력 | `학사학위 이상 소지자` |
    | 우대사항 | 텍스트 직접 입력 | `정보처리기사 자격증 소지자` |
    
    > 📌 이 팝업 내용은 **공고문에 자동 반영되지 않으며** 출력물·내부 관리용입니다.
    > 
    
    ⚠️ 게시 중인 공고의 모집부서는 삭제 불가 / 지원자가 있으면 삭제 불가

- 📘 2-2. 심사과정(전형 단계) 설정 (메뉴 1.7 Popup)
    
    > 
    > 
    > 
    > ![📷 [PDF 10페이지]](%EB%AA%A8%EC%A7%91%EB%B6%80%EC%84%9C%20%EC%84%A4%EC%A0%95/image%201.png)
    > 
    > 📷 [PDF 10페이지]
    > 
    
    | 순서 | 버튼/동작 | 설명 |
    | --- | --- | --- |
    | 1 | 심사단계 **[추가]** | 단계 행 생성 |
    | 2 | 심사명·심사비율·최종점 입력 | 예: `서류심사 / 50% / 100점` |
    | 3 | 심사단계 **[저장]** | 저장 후 [심사위원] 버튼 활성화됨 |
    | 4 | 심사항목설정 **[선택]** | 오른쪽 심사항목 영역에 표시 |
    | 5 | **[기본 설정 가져오기]** 또는 직접 추가 | 공통코드(E93) 기본 항목 불러오기 |
    | 6 | 심사항목별 배점 입력 | ⚠️ **합계 반드시 100점** |
    | 7 | 심사항목 **[저장]** | 완료 |
    
    **심사단계 구성 예시**
    
    | 단계 | 심사명 | 비율(%) |
    | --- | --- | --- |
    | 1단계 | 서류심사 | 50 |
    | 2단계 | 전공심사 | 20 |
    | 3단계 | 면접심사 | 30 |
    
    ⚠️ [심사위원] 버튼은 심사단계 저장 후에만 표시 / 배점 합계 ≠ 100점이면 저장 불가

- 📘 2-3. 자기소개서 항목 설정 (메뉴 1.9 Popup)
    
    > 
    > 
    > 
    > ![📷 [PDF 12페이지] ](%EB%AA%A8%EC%A7%91%EB%B6%80%EC%84%9C%20%EC%84%A4%EC%A0%95/image%202.png)
    > 
    > 📷 [PDF 12페이지] 
    > 
    
    | 순서 | 동작 | 설명 |
    | --- | --- | --- |
    | 1 | **[기본설정 가져오기]** | 공통코드(E95) 표준 항목 자동 세팅 |
    | 2 | 항목 추가 필요 시 **[추가]** | 신규 항목명 입력 |
    | 3 | 글자수 제한 설정 | 기본값 1000자 |
    | 4 | **[저장]** | 완료 |
    
    기본 항목: 지원동기 / 성장과정 / 생활신조 / 연구역량 및 연구계획 / 강의계획
