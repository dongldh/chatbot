- 📘 6-1. 전형별 심사 결과 및 합격 처리 (메뉴 1.21)

    > 
    > 
    > ![📷 [PDF 25페이지]](%EC%8B%AC%EC%82%AC%20%EA%B2%B0%EA%B3%BC%20%EC%B2%98%EB%A6%AC/image.png)
    > 
    > 📷 [PDF 25페이지]
    > 
    
    **처리 순서**
    
    | 순서 | 버튼/동작 | 설명 |
    | --- | --- | --- |
    | 1 | 조회 조건 선택 후 **[조회]** | 지원자 목록 + 심사위원별 점수 확인 |
    | 2 | 사정일자 설정 후 **[심사사정]** 클릭 | 총점 기준 자동 순위 산정 |
    | 3 | 가산점 입력 (해당자) | 총점·평균점수 자동 반영 |
    | 4 | 결과 선택 | `합격` / `불합격` / `예비순위` |
    | 5 | **[저장]** 클릭 | 임시 저장 (아직 확정 아님) |
    | 6 | **[확정]** 클릭 | ⚠️ 확정일자 등록, 이후 합격여부 수정 불가 |
    
    **합격/불합격/예비 처리 기준**
    
    | 결과값 | 의미 | 다음 단계 처리 |
    | --- | --- | --- |
    | `합격` | 다음 심사단계 진행 대상 | 다음 전형에서 조회 가능 |
    | `불합격` | 탈락 처리 | 다음 전형에서 미표시 |
    | `예비순위` | 추후 합격 전환 가능 | 담당자 관리 |
    
    ⚠️ **확정 후 합격여부 수정 불가** (가산점·비고는 수정 가능)
    
    ⚠️ 확정 취소: **[확정취소]** → 확정일자 공란 변경 → 결과 수정 → 재확정
    
    ⚠️ **합격 결과 지원자만** 다음 심사구분에서 조회 가능
    
    ⚠️ 심사사정은 점수 기준 등수만 산정 → 합격/불합격/예비순위는 **담당자가 직접 선택 후 확정**
